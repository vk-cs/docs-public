Резервная копия ВМ необходима, чтобы восстановить состояние виртуальной машины, сохраненное в определенный момент времени. Платформа VK Cloud предоставляет сервис для создания резервных копий ВМ в [ручном](/ru/storage/backups/service-management/create-backup-copy) или [автоматическом](/ru/storage/backups/service-management/create-backup-plan) режиме.

В ручном режиме создание резервных копий запускается пользователем. При этом создаются полные резервные копии ВМ.

Автоматический режим поддерживает [стратегию GFS](/ru/storage/backups/concepts/retention-policy/gfs-backup) и позволяет настроить расписание резервного копирования для создания полных и инкрементальных резервных копий.
