# {heading(Виртуальные машины)[id=iaas-concepts-vm]}

Виртуальная машина (ВМ) — совокупность ресурсов, которые эмулируют поведение реального компьютера. Это полноценный виртуальный сервер, который подходит для приложений и сервисов, работающих постоянно. Пример: веб-серверы, Telegram-боты.

Для своей работы виртуальная машина использует {ifdef(public)}{linkto(../../../../computing/iaas/concepts/vm/cpu-generation#iaas-cpu-generation)[text=ресурсы физического сервера]}{/ifdef}{ifdef(private,private-pg,private-pdf,private-pg-pdf,private-cert)}ресурсы физического сервера{/ifdef} (хоста). На одном хосте может размещаться несколько ВМ.

При создании виртуальной машины задается ее {linkto(../../../../computing/iaas/concepts/vm/flavor#iaas-flavor)[text=конфигурация]}:

- количество процессоров (CPU);
- объем оперативной памяти (RAM);
- тип и размер диска;
{ifndef(private-cert)}
- зона доступности;
{/ifndef}
- операционная система.

{ifndef(private-cert)}
{note:info}
Виртуальные машины, созданные в {var(cloud)}, используют эмуляцию BIOS. Эмуляция UEFI не поддерживается.
{/note}
{/ifndef}
