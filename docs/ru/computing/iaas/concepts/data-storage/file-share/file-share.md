# {heading(Файловое хранилище)[id=iaas-file-share]}

Файловое хранилище — это виртуальная файловая система, которую можно подключить к нескольким виртуальным машинам и использовать для совместного доступа к данным. При {linkto(../../../../../computing/iaas/instructions/fs-manage/fs-create#iaas-fs-create)[text=создании файлового хранилища]} указывается протокол сетевого доступа к нему (NFS или CIFS).

{note:info}
Доступ к файловым хранилищам осуществляется только из виртуальных машин внутри проекта {var(cloud)}.
{/note}

Размер файлового хранилища задается на этапе создания. При необходимости размер файлового хранилища может быть увеличен, уменьшить файловое хранилище нельзя.

Для файловых хранилищ доступны операции создания снимка и восстановления из снимка.
