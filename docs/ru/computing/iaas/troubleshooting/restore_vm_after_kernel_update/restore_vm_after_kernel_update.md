# {heading(ВМ не загружается после обновления ядра)[id=iaas-restore-vm-after-kernel-update]}

После установки нового ядра ОС виртуальная машина перестала загружаться.

Проблема может быть вызвана различными причинами: несовместимость, ошибки конфигурации, отсутствие необходимых модулей или ошибки в новой версии ядра.

### {heading(Решение)[id=iaas-restore-vm-after-kernel-update-decision]}

При загрузке системы выберите предыдущую версию ядра. Если не получается это сделать, обратитесь в [техническую поддержку](/ru/contacts).
