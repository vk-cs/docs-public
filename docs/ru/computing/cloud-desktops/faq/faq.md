# {heading(Вопросы и ответы)[id=iaas-faq]}

{cut(В работе сервиса возникла ошибка, которую я не могу исправить. Куда обратиться?)}

Обратитесь в [техническую поддержку](/ru/contacts) {var(cloud)}.

{/cut}

{cut(Какую информацию предоставлять при обращении в техническую поддержку по работе сервиса?)}

- Точное время возникновения проблемы.
- {linkto(../../../tools-for-using-services/account/instructions/project-settings/manage#project-pid-view)[text=Идентификатор]} проекта [личного кабинета](https://msk.cloud.vk.com/app), в котором возникла проблема.
- Название пула рабочих столов, если их в проекте несколько.
- Название рабочего стола, если проблема возникла с конкретным рабочим столом.
- Пользователь, от имени которого вы пытаетесь выполнить действия.
- Подробное описание проблемы и способ ее воспроизведения. Рекомендуется приложить скриншоты.

{/cut}

{cut(Можно ли подключиться к рабочему столу Cloud Desktop через браузер?)}

Нет, такая возможность не предусмотрена. Для подключения к виртуальному рабочему столу воспользуйтесь приложением {linkto(../user-instructions#desktops-install-client)[text=Cloud Desktop Client]}.

{/cut}
