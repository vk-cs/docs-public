# {heading(Шаблон конфигурации инстансов БД)[id=dbaas-flavor]}

Шаблон конфигурации ВМ (flavor) определяет количество ресурсов ВМ: количество CPU и объем RAM.

При установке {var(cloud)} для инстансов БД создаются шаблоны конфигурации ВМ Standard со следующими характеристиками:

* vCPU: от 2 до 16.
* RAM: от 8 ГБ до 64 ГБ.

Полный перечень доступных шаблонов конфигурации ВМ можно увидеть, развернув список **Тип виртуальной машины** при {linkto(../../instructions/create#dbaas-create)[text=создании]} базы данных.

{note:warn}
Возможно добавление новых пользовательских шаблонов конфигурации ВМ с произвольными конфигурациями. Чтобы создать другие шаблоны конфигурации ВМ, обратитесь к администратору {var(cloud)}.
{/note}
