## {heading(Redis Metrics Exporter)[id=dbaas-extensions-redis-exporter]}

Название модуля в личном кабинете: `redis_exporter`.

Redis Metrics Exporter позволяет собирать метрики для БД Redis (подробнее — в [официальной документации](https://github.com/oliver006/redis_exporter) на GitHub).

Пример настройки мониторинга на основе Prometheus+Grafana+Redis Exporter приведен в [официальной документации](https://mcs.mail.ru/docs/ru/additionals/cases/cases-monitoring/case-redis-exporter) {var(cloud)}.
