# {heading(Тарификация)[id=captcha-tariffication]}

Сервис VK Капча платный. Тарифицируется каждая проверка пользователя при прохождении капчи. Для расчета стоимости напишите команде VK Капча на почту [captcha@corp.vk.com](mailto:captcha@corp.vk.com).

Чтобы протестировать сервис бесплатно, при вызове {linkto(../../../tools-for-using-services/api/api-spec/captcha-api#api-spec-captcha)[text=метода]} `GET /captchaNotRobot.createSession` передайте заголовок `Captcha-Test` со значением `true`. В режиме тестирования все попытки прохождения капчи считаются успешными, а запросы не тарифицируются.

{note:warn}
Режим тестирования предназначен только для проверки интеграции вашего приложения с сервисом. Не используйте заголовок `Captcha-Test` в production-среде — в этом режиме VK Капча не защищает от автоматизированных атак.
{/note}
