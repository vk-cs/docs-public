## {heading(Создание секрета)[id=sm-manage-create]}

{tabs}

{tab(Личный кабинет)}

1. {ifdef(public)}[Перейдите](https://msk.cloud.vk.com/app/){/ifdef}{ifndef(public)}{linkto(../../../../tools-for-using-services/account/instructions/lk-entry#tools-account-lk-entry)[text=Перейдите]}{/ifndef} в личный кабинет {var(cloud)}.
1. Перейдите в раздел **Сервисы безопасности** → **Менеджер секретов**.
1. Нажмите кнопку **Создать**.
1. Заполните поле **Название**.
1. (Опционально) Заполните поле **Описание**.
1. Задайте **Идентификатор** секрета.

    {note:info}

    Идентификатор секрета должен быть уникальным среди всех секретов в проекте.

    {/note}

1. Выберите тип данных секрета и задайте его содержание:

    {tabs}

    {tab(Генерируемые)}

    Содержит случайную последовательность символов, которая генерируется автоматически и без вмешательства пользователя.

    - **Количество символов** — длина строки с данными.
    - **Использование символов** — из каких символов генерируется последовательность. Можно выбрать несколько типов:

        - `Заглавные буквы латинского алфавита (A-Z)`,
        - `Строчные буквы латинского алфавита (a-z)`,
        - `Цифры от 0 до 9`,
        - `Специальные символы (!@#$%^&*)`.

    - **Ключ** — уникальный идентификатор пары, задается вручную.

    {/tab}

    {tab(Пользовательские)}

    Содержит одну или нескольких пар ключ/значение, которые задаются пользователем. 

    - **Ключ** — уникальный идентификатор пары, задается вручную.
    - **Значение** — чувствительные данные. Может быть строкой или файлом:

        - Чтобы добавить строку, заполните поле **Значение**.
        - Чтобы добавить файл, нажмите **Выбрать файл**.

    Если необходимо добавить несколько пар ключ/значение, нажмите кнопку **Добавить ключ**.

    {/tab}

    {/tabs}

1. Нажмите кнопку **Создать**.

{/tab}

{/tabs}

## {heading(Редактирование метаданных)[id=sm-manage-edit-metadata]}

Отредактировать можно только метаданные секрета. Чтобы обновить содержание секрета, {linkto(../versions#sm-versions-create)[text=создайте]} новую версию.

{tabs}

{tab(Личный кабинет)}

1. {ifdef(public)}[Перейдите](https://msk.cloud.vk.com/app/){/ifdef}{ifndef(public)}{linkto(../../../../tools-for-using-services/account/instructions/lk-entry#tools-account-lk-entry)[text=Перейдите]}{/ifndef} в личный кабинет {var(cloud)}.
1. Перейдите в раздел **Сервисы безопасности → Менеджер секретов**.
1. Откройте окно редактирования одним из способов.

    - Из списка секретов:

        1. В списке секретов нажмите ![ ](../../../../assets/more-icon.svg "inline") для нужного секрета.
        1. Выберите пункт **Редактировать**.

    - На странице секрета:

        1. В списке секретов нажмите на имя нужного секрета.
        1. На странице секрета нажмите кнопку **Редактировать**.

1. Внесите необходимые изменения.
1. Нажмите кнопку **Сохранить**.

{/tab}

{/tabs}
