## {heading(Создание новой версии секрета)[id=create]}

{tabs}

{tab(Личный кабинет)}

1. [Перейдите](https://msk.cloud.vk.com/app/) в личный кабинет VK Cloud.
1. Перейдите в раздел **Сервисы безопасности → Менеджер секретов**.
1. В списке секретов нажмите на имя нужного секрета.
1. Перейдите на вкладку **История версий**.
1. Нажмите кнопку **Создать версию**.
1. Выберите тип данных секрета и задайте его содержание:

    {tabs}

    {tab(Генерируемые)}

    Содержит случайную последовательность символов, которая генерируется автоматически и без вмешательства пользователя.

    - **Количество символов** — длина строки с данными.
    - **Использование символов** — из каких символов генерируется последовательность. Можно выбрать несколько типов:

        - `Заглавные буквы латинского алфавита (A-Z)`,
        - `Строчные буквы латинского алфавита (a-z)`,
        - `Цифры от 0 до 9`,
        - `Специальные символы (!@#$%^&*)`.

    - **Ключ** — уникальный идентификатор пары, задается вручную.

    {/tab}

    {tab(Пользовательские)}

    Содержит одну или нескольких пар ключ/значение, которые задаются пользователем. 

    - **Ключ** — уникальный идентификатор пары, задается вручную.
    - **Значение** — чувствительные данные. Может быть строкой или файлом:

        - Чтобы добавить строку, заполните поле **Значение**.
        - Чтобы добавить файл, нажмите **Выбрать файл**.

    Если необходимо добавить несколько пар ключ/значение, нажмите кнопку **Добавить ключ**.

    {/tab}

    {/tabs}

1. (Опционально) Заполните поле **Описание**.
1. Нажмите кнопку **Создать**.

{/tab}

{/tabs}

## {heading(Просмотр версий секретов)[id=view]}

{tabs}

{tab(Личный кабинет)}

1. [Перейдите](https://msk.cloud.vk.com/app/) в личный кабинет VK Cloud.
1. Перейдите в раздел **Сервисы безопасности → Менеджер секретов**.
1. В списке секретов нажмите на имя нужного секрета.
1. Перейдите во вкладку **История версий**.
1. Нажмите на номер нужной версии. Откроется окно с информацией о версии:

    - **Ключ** — уникальный идентификатор пары.
    - **Значение** — чувствительные данные. Может быть строкой в маскированном виде или файлом.

        Чтобы расшифровать значение, нажмите на значок ![ ](/ru/assets/eye-icon.svg "inline").

{/tab}

{/tabs}
