## {heading(Создание новой версии секрета)[id=sm-versions-create]}

{tabs}

{tab(Личный кабинет)}

1. {ifdef(public)}[Перейдите](https://msk.cloud.vk.com/app/){/ifdef}{ifndef(public)}{linkto(../../../../tools-for-using-services/account/instructions/lk-entry#tools-account-lk-entry)[text=Перейдите]}{/ifndef} в личный кабинет {var(cloud)}.
1. Перейдите в раздел **Сервисы безопасности → Менеджер секретов**.
1. В списке секретов нажмите на имя нужного секрета.
1. Перейдите на вкладку **История версий**.
1. Нажмите кнопку **Создать версию**.
1. Задайте одну или несколько пар ключ/значение для новой версии:

    - **Ключ** — уникальный идентификатор пары, задается вручную.
    - **Значение** — строка с чувствительными данными. Заполните поле вручную или нажмите **Сгенерировать**, чтобы задать настройки автоматического заполнения.

        - **Количество символов** — длина строки с данными.
        - **Использование символов** — из каких символов генерируется последовательность. Можно выбрать несколько типов:

            - `Заглавные буквы латинского алфавита (A-Z)`,
            - `Строчные буквы латинского алфавита (a-z)`,
            - `Цифры от 0 до 9`,
            - `Специальные символы`. Поддерживаются спецсимволы: `!`, `"`, `#`, `$`, `%`, `&`,`'`, `(`, `)`, `*`, `+`, `,`, `-`, `.`,`/`,`:`, `;`, `<`, `=`, `>`, `?`, `@`, `[`,`\`, `]`, `^`, `_`, `` ` ``, `{`,`|`, `}`, `~`.

        После настройкми параметров нажмите кнопку **Сгенерировать**. В поле **Значение** появится случайный набор символов, который сгенерирован автоматически и без вмешательства пользователя.

    Если необходимо добавить несколько пар ключ/значение, нажмите кнопку **Добавить ключ**.

1. (Опционально) Заполните поле **Описание**.
1. Нажмите кнопку **Создать**.

{/tab}

## {heading(Просмотр версий секретов)[id=sm-versions-view]}

{tabs}

{tab(Личный кабинет)}

1. {ifdef(public)}[Перейдите](https://msk.cloud.vk.com/app/){/ifdef}{ifndef(public)}{linkto(../../../../tools-for-using-services/account/instructions/lk-entry#tools-account-lk-entry)[text=Перейдите]}{/ifndef} в личный кабинет {var(cloud)}.
1. Перейдите в раздел **Сервисы безопасности → Менеджер секретов**.
1. В списке секретов нажмите на имя нужного секрета.
1. Перейдите во вкладку **История версий**.
1. Нажмите на номер нужной версии. Откроется окно с информацией о версии:

    - **Ключ** — уникальный идентификатор пары.
    - **Значение** — строка с чувствительными данными в маскированном виде.

        Чтобы просмотреть значение, нажмите на значок ![ ](../../../../assets/eye-icon.svg "inline").
