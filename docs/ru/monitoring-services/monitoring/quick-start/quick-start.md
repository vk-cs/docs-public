Быстрый старт поможет вам начать работу с сервисом Cloud Monitoring и познакомиться с его возможностями на примере виртуальной машины.

Пройдя все шаги быстрого старта, вы:

- Создадите новую ВМ с агентом мониторинга или подключите мониторинг к существующей ВМ, чтобы сервис начал собирать метрики.
- Посмотрите системные графики для этой ВМ.
- Добавите один из системных графиков в свои графики.
- Создадите собственный график с данными мониторинга на примере метрики `cpu_usage_system`.

{note:warn}
Работающая ВМ потребляет вычислительные ресурсы и [тарифицируется](/ru/computing/iaas/tariffication).
{/note}
