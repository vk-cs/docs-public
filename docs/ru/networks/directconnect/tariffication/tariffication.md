# {heading(Тарификация)[id=dc-tariffication]}

Услуга Cloud Direct Connect предоставляется в соответствии с индивидуальным соглашением между пользователем и провайдером.

Тарифная ставка рассчитывается в индивидуальном порядке после подачи пользователем {linkto(../connect#directconnect-connect)[text=заявки на подключение]}.
