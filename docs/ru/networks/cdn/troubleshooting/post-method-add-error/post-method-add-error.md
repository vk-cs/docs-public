# {heading(Ошибка при добавлении POST в список разрешенных методов)[id=vnet-port-unavailability]}

При добавлении POST в список разрешенных HTTP-методов возникает ошибка. Метод POST обычно не подлежит кешированию и часто исключается из списков разрешенных методов для кеширования в CDN. 

### {heading(Решение)[id=vnet-port-unavailability-decision]}

Если вы настраиваете список методов, которые проксируются на источник, метод POST не получится добавить в настройки кеширования.
