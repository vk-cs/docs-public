Если не удается добавить Floating IP-адрес, вероятно, у вас закончились [квоты](https://cloud.vk.com/docs/ru/tools-for-using-services/account/concepts/quotasandlimits#seti_podseti_i_ip_adresa_a84cb9f0) на количество адресов.

### Решение

1. Проверьте квоты проекта на количество адресов.

1. Если лимит исчерпан, [обратитесь в техническую поддержку](/ru/contacts) с запросом на увеличение квот.
