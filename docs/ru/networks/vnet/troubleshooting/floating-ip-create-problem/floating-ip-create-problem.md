# {heading(Не удается добавить Floating IP-адрес)[id=vnet-floating-ip-create-problem]}

Если не удается добавить Floating IP-адрес, вероятно, у вас закончились {linkto(../../../../tools-for-using-services/account/concepts/quotasandlimits#quotasandlimits-nets-subnets-ips)[text=квоты]} на количество адресов.

### {heading(Решение)[id=vnet-floating-ip-create-problem-resolve]}

1. Проверьте квоты проекта на количество адресов.
1. Если лимит исчерпан, обратитесь в [техническую поддержку](/ru/contacts) с запросом на увеличение квот.
