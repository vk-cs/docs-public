# {heading(Ограничена выдача Floating IP-адресов в конкретной подсети)[id=vnet-ip-create-problem]}

Выдача публичных IP-адресов производится из общих пулов платформы. Проблема возникает, если в конкретной подсети закончились свободные адреса.

### {heading(Решение)[id=vnet-ip-create-problem-resolve]}

Попробуйте создать Floating IP-адрес в другой {linkto(../../../../start/concepts/architecture#architecture-az)[text=зоне доступности]} или обратитесь в [техническую поддержку](/ru/contacts) для расширения квоты или пула адресов.
