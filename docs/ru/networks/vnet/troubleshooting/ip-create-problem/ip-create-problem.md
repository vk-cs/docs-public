Выдача публичных IP-адресов производится из общих пулов платформы. Проблема возникает, если в конкретной подсети закончились свободные адреса.

### Решение

Попробуйте создать Floating IP-адрес в другой [зоне доступности](/ru/start/concepts/architecture#az) или [обратитесь в техническую поддержку](https://cloud.vk.com/contacts/) для расширения квоты или пула адресов.
