# {heading(Порт не переименовывается после отвязки группы безопасности)[id=vnet-port-renaming-problem]}

Отвязка группы безопасности от порта не должна влиять на возможность его переименования в интерфейсе.

### {heading(Решение)[id=vnet-port-renaming-problem-resolve]}

Измените имя порта через OpenStack CLI:

```console
openstack port set --name <НОВОЕ_ИМЯ> <ID_ПОРТА>
```

Если проблема сохраняется, обратитесь в [техническую поддержку](/ru/contacts).
