Отвязка группы безопасности от порта не должна влиять на возможность его переименования в интерфейсе.

### Решение

Измените имя порта через OpenStack CLI:
```console
openstack port set --name <НОВОЕ_ИМЯ> <ID_ПОРТА>
```

Если проблема сохраняется, [обратитесь в техническую поддержку](https://cloud.vk.com/contacts/).
