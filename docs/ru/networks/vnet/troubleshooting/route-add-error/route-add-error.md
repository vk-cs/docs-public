Ошибка при добавлении маршрута может быть связана с пересечением диапазонов адресов со служебными сетями платформы. 

### Решение

Проверьте, не пересекается ли подсеть `150.0.0.0/8` с уже существующими маршрутами или внешними сетями.

Если проблема сохраняется, [обратитесь в техническую поддержку](https://cloud.vk.com/contacts/).
