# {heading(Настройка статической маршрутизации)[id=vnet-manage-static-routs]}

## {heading(Добавление статического маршрута)[id=vnet-manage-static-routs-add]}

{tabs}

{tab(Личный кабинет)}

{include(../../../../../_includes/_open_advanced_router.md)}

1. Выберите вкладку **Статические маршруты**.
1. Нажмите кнопку **Добавить статический маршрут**.
1. Задайте сеть назначения и укажите префикс сети.
1. Укажите промежуточный узел (next hop). При настройке статических маршрутов на продвинутом маршрутизаторе в качестве Next Hop следует указывать IP-адрес шлюза, через который доступна целевая подсеть.
1. (Опционально) Укажите метрику, которая определяет приоритет статистических маршрутов относительно друг друга в таблице маршрутизации.
1. (Опционально) Добавьте описание маршрута.
1. Нажмите кнопку **Добавить**.

{/tab}

{/tabs}

## {heading(Удаление статического маршрута)[id=vnet-manage-static-routs-delete]}

{tabs}

{tab(Личный кабинет)}

Это групповая операция: при необходимости можно удалить сразу несколько маршрутов, выбрав их с помощью флажков.

{include(../../../../../_includes/_open_advanced_router.md)}

1. Выберите вкладку **Статические маршруты**.
1. Удалите маршрут одним из способов:

   - Нажмите ![ ](../../../../../assets/more-icon.svg "inline") для маршрута и выберите **Удалить статический маршрут**.
   - Выберите нужный маршрут с помощью флажка, затем нажмите кнопку **Удалить** над таблицей.

1. Подтвердите удаление.

{/tab}

{/tabs}
