# {heading(Вход в личный кабинет IAM Only)[id=s3-instructions-iamo-auth]}

Чтобы получить доступ к личному кабинету, необходимо пройти аутентификацию.

Чтобы использовать личный кабинет:

1. С помощью любого поддерживаемого веб-браузера откройте страницу с адресом:

   ```console
   lk.<ДОМЕННОЕ_ИМЯ>
   ```

   Здесь `<ДОМЕННОЕ_ИМЯ>` — доменное имя, которое было указано при установке сервиса.

1. Введите логин и пароль своей учетной записи.

   {note:warn}
   Для получения учетных данных обратитесь к администратору.
   {/note}

1. Нажмите кнопку **Войти**.
1. В шапке личного кабинета выберите проект для дальнейшей работы.

При работе с личным кабинетом:

- Для каждой учетной записи единовременно может быть запущен только один сеанс доступа.
- После 1 часа бездействия сеанс доступа будет завершен. Пройдите аутентификацию заново.
- Учетная запись может быть заблокирована, если не использовалась более 45 дней.

  {note:warn}
  45 дней — период, заданный по умочанию. Администратор может изменить его.
  {/note}

  При попытке входа в личный кабинет с помощью заблокированной учетной записи появится ошибка «403».

  Блокировка учетной записи владельца проекта не ограничивает работу других пользователей в проекте. Для разблокировки учетной записи обратитесь к администратору.
