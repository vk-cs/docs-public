# {heading(Не создается резервная копия ВМ при использовании средств информационной безопасности)[id=storage-backups-unavailable-with-security-tools]}

Средства информационной безопасности (антивирусы, DLP), установленные внутри ВМ, могут влиять на создание резервных копий, если они блокируют файловую систему или создают высокую нагрузку на диск в момент снятия снимка.

### {heading(Решение)[id=storage-backups-unavailable-with-security-tools-decision]}

1. [Проверьте](/ru/computing/iaas/instructions/vm/vm-manage#iaas-vm-manage-view-logs) в журнале событий ВМ, не блокируют ли средства ИБ доступ к файлам и дискам.
2. Настройте исключения для процессов резервного копирования.
