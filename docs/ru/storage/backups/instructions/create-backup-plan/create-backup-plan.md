# {heading(Создание плана резервного копирования)[id=backup-plan-create]}

План резервного копирования служит для регулярного создания в автоматическом режиме резервных копий виртуальной машины{ifndef(private-cert)} или базы данных{/ifndef}.

План резервного копирования включает в себя:

- тип и имя объекта, для которого будет выполняться резервное копирование;
- настройки стратегии хранения и удаления резервных копий;
- расписание запусков создания резервных копий.

Вы можете создать план резервного копирования:

- автоматически, включив опцию резервного копирования при создании ВМ{ifndef(private-cert)} или инстанса БД{/ifndef};
- вручную в личном кабинете {var(cloud)}{ifdef(public)} или с помощью {linkto(../../../../tools-for-using-services/api/api-spec/backup-api#api-spec-karboii)[text=API сервиса Cloud Backup]}{/ifdef}.

{note:info}
Каждая ВМ{ifndef(private-cert)} или инстанс БД{/ifndef} может иметь только один план резервного копирования.
{/note}

## {heading(Создание плана резервного копирования для виртуальной машины)[id=backup-plan-create-vm]}

{tabs}

{tab(Личный кабинет)}

1. {ifdef(public)}[Перейдите](https://msk.cloud.vk.com/app/){/ifdef}{ifndef(public)}{linkto(../../../../tools-for-using-services/account/instructions/lk-entry#tools-account-lk-entry)[text=Перейдите]}{/ifndef} в личный кабинет {var(cloud)}.
1. Перейдите в раздел **Cloud Backup**.
1. Нажмите кнопку **Создать расписание**.
1. Введите **Название плана** резервного копирования.
1. Выберите тип ресурса: `Виртуальная машина`.
   {ifdef(public)}
1. Выберите нужные ВМ с помощью кнопки **Добавить**.
   {/ifdef}
    Ниже на странице создания плана указаны настройки по умолчанию, которые соответствуют лучшим практикам {var(cloud)}:

   - **Включить инкрементальные бекапы**: опция отключена.
   - **Дни создания полного бэкапа**: выбраны все дни недели, время запуска выбрано в промежутке от 09:00 до 18:00.
   - **Включить стратегию хранения полных копий (GFS)**: опция включена.
   - **Хранить недельные полные копии**: выбран срок `4` недели.
   - **Хранить месячные полные копии**: опция включена, выбран срок `12` месяцев.
   - **Хранить годовые полные копии**: опция включена, выбран срок `3` года.

1. Если эти настройки не подходят для вашего плана, измените их:

   {include(../../../../_includes/_backup_plan_create_change.md)[tags=vm]}

1. Нажмите кнопку **Создать план**.

   Созданный план резервного копирования появится в списке.

{/tab}
