## Реализованные сервисы

В VK Cloud реализованы и функционируют сервисы:

1. VK Cloud BigData — сервис больших данных.
1. VK Cloud Billing — сервис биллинга.
1. VK Cloud CDN — сервис Content delivery network (CDN).
1. VK Cloud DNS — сервис Domain Name System (DNS).
1. VK Cloud AI — сервис распознавания изображений и голоса.
1. VK Cloud Backup — сервис создания и работы с резервными копиями.
1. VK Cloud BlockStorage — сервис блочных хранилищ.
1. VK Cloud FileStorage — сервис файловых хранилищ.
1. VK Cloud Images — сервис управления образами.
1. VK Cloud VirtualMachine — сервис работы с виртуальными машинами.
1. VK Cloud CloudAudit — сервис журнала аудита приложений.
1. VK Cloud CloudLogging — сервис логирования данных.
1. VK Cloud CloudMonitoring — сервис сбора метрик.
1. VK Cloud DBaaS — сервис работы с базами данных.
1. VK Cloud GPU — сервис управления ресурсами GPU.
1. VK Cloud IAM — сервис аутентификации и авторизации.
1. VK Cloud IoT — сервис интернета вещей.
1. VK Cloud KaaS — сервис облачных контейнеров.
1. VK Cloud LoadBalancer — сервис балансировки нагрузки.
1. VK Cloud Network — сервис виртуальных сетей.
1. VK Cloud MachineLearning — сервис машинного обучения.
1. VK Cloud Portal — сервис личного кабинета VK Cloud.
1. VK Cloud Hotbox — сервис объектного хранилища S3.
1. VK Cloud SQS — сервис очередей.
1. VK Cloud 1С — готовое рабочее место 1С; интегрирован через партнера.
1. VK Cloud XaaS — сервис магазина приложений.
