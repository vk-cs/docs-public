1. Перейдите в раздел **Виртуальные сети → Сети**.
1. Удалите сеть одним из способов:

   - Нажмите ![more-icon](../../../../assets/more-icon.svg "inline") для сети и выберите пункт ![trash-icon](../../../../assets/trash-icon.svg "inline") **Удалить**.
   - Выберите нужную сеть с помощью флажка, затем нажмите кнопку ![trash-icon](../../../../assets/trash-icon.svg "inline") **Удалить** над таблицей.

1. Подтвердите удаление.
