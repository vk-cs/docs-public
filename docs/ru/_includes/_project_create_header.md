Вы можете создавать собственные [проекты](/ru/tools-for-using-services/account/concepts/projects) в VK Cloud. Создав проект, вы становитесь его [владельцем](/ru/access/iam/concepts/roles-reference#mcs_owner).

{note:info}
В рамках [базовых квот](/ru/tools-for-using-services/account/concepts/quotasandlimits#quotasandlimits-general) один пользователь может создать не более 5 проектов. Это число можно [увеличить](/ru/tools-for-using-services/account/instructions/project-settings/manage#project-increase-quota) в пределах существующего [лимита](/ru/tools-for-using-services/account/concepts/quotasandlimits#quotasandlimits-tech-limits).
{/note}
