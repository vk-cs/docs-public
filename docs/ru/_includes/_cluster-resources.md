1. Настройте список ресурсов с помощью настроек поиска, группировки и фильтрации:

   - Воспользуйтесь поиском по имени ресурса.
   - В выпадающем списке выберите нужную категорию ресурсов (по умолчанию выбрана категория `Рабочая нагрузка`). Отображаются ресурсы только из указанной категории.
   - В выпадающем списке установите фильтр по пространствам имен (по умолчанию выбраны все пространства).

1. Раскройте нужный блок на странице. Откроется список ресурсов, соответствующий выбранным настройкам.
