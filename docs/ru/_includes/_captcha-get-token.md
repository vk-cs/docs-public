- Список доменов (`domains`), где будет интегрирована VK Капча. Они будут закреплены за вашим сервисным токеном и использоваться в CSP-заголовке при создании CAPTCHA-сессии. Если домены не указаны, в CSP-заголовок будет подставляться значение из параметра `domain` {linkto(../../../../tools-for-using-services/api/api-spec/captcha-api#api-spec-captcha)[text=метода]} `GET /captchaNotRobot.createSession`.

- Список разрешенных IP-адресов или CIDR (`ips`), с которых будут приходить запросы на создание CAPTCHA-сессии. Если они не указаны, перечень адресов для создания CAPTCHA-сессии будет не ограничен.

{note:err}
Не публикуйте токен в открытом доступе и никому не передавайте его, особенно если в запросе на регистрацию вы не зафиксировали список разрешенных адресов и доменов. С его помощью злоумышленники могут получить доступ к сервису за ваш счет.
{/note}
