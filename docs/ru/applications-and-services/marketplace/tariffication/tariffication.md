# {heading(Тарификация)[id=marketplace-tariffication]}

## {heading(О тарификации)[id=marketplace-tariffication-about]}

Стоимость использования инстанса сервиса зависит от выбранного тарифного плана и опций. {ifdef(public)}Она отражается в {linkto(../../../applications-and-services/marketplace/instructions/pr-instance-add#marketplace-pr-instance-add)[text=мастере конфигурации сервиса]} и состоит из двух частей:

- Стоимости использования сервиса, она отражается на вкладке **Тарифные планы**.
- Для image-based сервисов — стоимости инфраструктуры, она отражается в разделе **Детализация**.

Итоговая стоимость будет отображена на последнем шаге подключения сервиса.
{/ifdef}
{ifdef(private,private-pg,private-pdf,private-pg-pdf)}
Она отражается в блоке **Детализация** в мастере конфигурации тарифного плана. Сумма, указанная в поле **Спишется**, спишется со счета проекта при подключении сервиса.

Для image-based приложения дополнительно тарифицируется инфраструктура Платформы. {linkto(../../../tools-for-using-services/account/instructions/project-settings/payment#account-payment)[text=Стоимость инфраструктуры]} рассчитывается отдельно от стоимости тарифного плана и опций по правилам {var(cloud)}.

При редактировании текущего тарифного плана или переходе на новый тарифный план в блоке **Детализация** будет отображаться следующая информация:

- **За месяц** — новая стоимость тарифного плана за месяц (с учетом обновленных значений опций).
- **Вернется бонусами** — сумма денежных средств, возвращаемая бонусами.

  Если новая стоимость тарифного плана с опциями меньше текущей, неизрасходованный остаток денежных средств за оставшиеся дни расчетного периода возвращается на бонусный счет проекта.

- **Спишется** — сумма списания за оставшиеся дни расчетного периода (полный расчетный период равен 1 месяц). Рассчитывается с учетом стоимости текущего тарифного плана и опций.
{/ifdef}

Для тарификации сервисов доступно два типа тарифных планов:

{tabs}

{tab(Предоплатный)}

- Стоимость фиксированная.
- Оплата списывается раз в отчетный период в дату подключения сервиса.

{ifdef(public)}
Если при списании недостаточно средств, происходит изменение статуса инстанса:

- Если для инстанса доступен бесплатный промо-тариф, инстанс принудительно переводят на него.
- Если для инстанса недоступен бесплатный промо-тариф, инстанс блокируется для использования до пополнения средств.
{/ifdef}
{/tab}

{tab(Постоплатный)}

- Стоимость рассчитывается исходя из фактически использованного количества ресурсов сервиса (например, количество используемого объема хранилища).
- Периодичность и день списания зависят от конкретного сервиса.

{ifdef(public)}
Если при списании недостаточно средств, происходит изменение статуса инстанса:

1. Soft Limit. Началом периода считается первая неуспешная попытка списания средств. Максимальная длительность периода — до 10 дней, баланс проверяется ежедневно. Если за этот период не было успешных списаний, начинается период Hard Limit.
1. Hard Limit. Принудительное списание всех долгов по сервису. Инстанс сервиса блокируется до момента полной оплаты задолженности на 10 дней:

   - Если после списания баланс остается положительным, период обнуляется, приходит уведомление о необходимости {linkto(../../../applications-and-services/marketplace/instructions/pr-instance-manage#marketplace-pr-instance-manage-active)[text=ручной активации]} инстанса сервиса.
   - Если средств недостаточно, после истечения периода инстанс сервиса удаляется без возможности восстановления.
{/ifdef}

{/tab}

{/tabs}

{ifdef(public)}
{note:info}
О том, как оплатить использование инстанса сервиса, читайте в разделе {linkto(../../../intro/billing#billing)[text=Биллинг]}.
{/note}

## {heading(Тарифицируется)[id=marketplace-tariffication-yes]}

Для SaaS и image-based сервисов:

- Применение платных тарифных опций плана. Тарифная опция — конкретный параметр тарифного плана.
- Виртуальные ресурсы проекта, используемые для резервного копирования или восстановления.
- Другие услуги, предоставляемые в рамках сервиса (конечный набор зависит от сервиса вендора).

Для image-based сервисов тарифицируются виртуальные ресурсы {var(cloud)}, развернутые для поддержания инфраструктуры сервиса: ВМ, диски, сети, маршрутизаторы. Полный список необходимых ресурсов доступен при {linkto(../../../applications-and-services/marketplace/instructions/pr-instance-add#marketplace-pr-instance-add)[text=подключении]} инстанса сервиса.

## {heading(Не тарифицируется)[id=marketplace-tariffication-no]}

- Использование личного кабинета любого из сервиса вендора.
- Выполнение базовых операций с инстансом сервиса: перезапуск, удаление.
- Сетевой трафик для поддержания инфраструктуры.
- Использование сервиса при выборе промо-тарифа.
{/ifdef}
