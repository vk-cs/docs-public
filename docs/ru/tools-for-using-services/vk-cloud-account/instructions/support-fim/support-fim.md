Вы можете настроить единый вход для федеративных пользователей на [портал технической поддержки](https://support.mcs.mail.ru) VK Cloud:

1. [Перейдите](https://cloud.vk.com/account) в аккаунт VK Cloud.
1. Перейдите в раздел **Федерация удостоверений** на вкладку **Федерации**.
1. Скопируйте значение параметра **ID федерации**.
1. Создайте заявку одним из способов:

    - Войдите на [портал технической поддержки](/ru/contacts) с помощью логина и пароля и нажмите кнопку **Создать заявку**.
    - Напишите на почту [support@mcs.mail.ru](mailto:support@mcs.mail.ru).
1. В заявке передайте полученный ранее ID федерации и список доверенных доменов вашей организации.

{note:info}
Если вы пересоздали федерацию удостоверений, заявку на создание единого входа на портал технической поддержки нужно отправить еще раз, указав новый ID федерации.
{/note}
