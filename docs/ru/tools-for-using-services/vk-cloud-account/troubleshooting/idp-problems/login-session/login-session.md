При попытке входа через федерацию удостоверений отображается сообщение:

```plaintext
Forbidden: probably your login session is expired: InResponseTo does not match any of the possible request IDs (expected [])
```

Ошибка возникает, если с момента загрузки страницы входа до ввода логина и пароля прошло более пяти минут.

## Решение

Обновите страницу для входа федеративных пользователей и повторите попытку. Если проблема сохраняется, обратитесь в [техническую поддержку](/ru/contacts).
