Ошибка при создании проекта может возникать, если в аккаунте создано слишком много проектов. В рамках [базовых квот](/ru/tools-for-using-services/account/concepts/quotasandlimits#quotasandlimits-general) один пользователь может создать не более 5 проектов. 

### Решение

1. [Перейдите](https://cloud.vk.com/app/) в личный кабинет VK Cloud.
1. Проверьте список всех проектов.
1. Если достигнут лимит, [удалите](/ru/tools-for-using-services/account/instructions/project-settings/manage#project-delete) проект.

Если лимит не превышен, [обратитесь в техническую поддержку](/ru/contacts) со скриншотом ошибки.
