# {heading(Баланс проекта)[id=account-balance]}

{include(../../../../_includes/_balance.md)}

При наличии бонусов, средства сначала списываются с бонусного счета, затем с основного.

Тарификация производится по принципу «pay as you go»: плата взимается только за потребляемые ресурсы. Денежные средства списываются поминутно с работающих объектов виртуализации. Если объекты виртуализации остановлены, списания продолжаются только за арендованное дисковое пространство и хранение имеющихся резервных копий.

{note:warn}
Итоговая стоимость округляется до **двух знаков после запятой**.
{/note}
