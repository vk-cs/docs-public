# {heading(Приглашение участника в проект)[id=tools-account-project-invite]}

Приглашать участника и настраивать ему доступ могут пользователи со следующими {linkto(../../../../../access/iam/concepts/roles-reference#iam-roles-reference)[text=ролями]}: владелец, суперадминистратор и администратор пользователей (IAM). Просмотр участников проекта доступен также администратору проекта и наблюдателю.
