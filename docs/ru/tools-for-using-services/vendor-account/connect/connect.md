Чтобы иметь возможность добавлять приложения в [Marketplace](https://cloud.vk.com/docs/ru/applications-and-services/marketplace):

1. [Зарегистрируйте](/ru/intro/onboarding/account) в {var(cloud)} пользователя, который будет загружать ваше приложение в Marketplace, если это еще не сделано.

   При регистрации для всех новых пользователей автоматически создается проект. Вы можете использовать этот проект для тестирования сервисных пакетов и загрузки их в Marketplace.

1. Отправьте письмо на [marketplace@cloud.vk.com](mailto:marketplace@cloud.vk.com) со следующей информацией:

   - Имя компании.
   - Идентификатор проекта ([Project ID](/ru/tools-for-using-services/api/rest-api/endpoints#rest-api-endpoints-get-project-id)), в котором будет выполняться тестирование манифестов Terraform для вашего приложения.

     Значение отображается в личном кабинете VK Cloud на странице настроек проекта на вкладке **Terraform**.

   - Электронная почта пользователя, которому необходим доступ в тестовые пространства имен Marketplace, чтобы тестировать сервис до публикации.
   - Электронная почта пользователя, который будет назначен владельцем сервисного ключа.
   - Электронная почта пользователя, которому будут приходить уведомления об ошибках при создании инстансов приложения.

   В ответном письме будет выслан сервисный ключ. По умолчанию срок его действия — 1 год. Сервисный ключ содержит тестовые и открытые пространства имен Marketplace, в которых будет доступно приложение.

   Идентификатор вашего проекта (Project ID) будет занесен в список поставщиков. На бонусный счет вашего проекта в VK Cloud будут зачислены бонусные средства для тестирования вашего приложения в Marketplace.
