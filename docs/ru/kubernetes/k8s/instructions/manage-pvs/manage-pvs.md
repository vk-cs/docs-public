# {heading(Управление постоянными томами (PV))[id=k8s-manage-pvs]}

В сервисе Cloud Containers вы можете перемещать или удалять {linkto(../../reference/pvs-and-pvcs#k8s-pvs-and-pvcs)[text=постоянные тома (PV)]}, созданные для кластеров Kubernetes. 

Переместить или удалить PV можно, только если он не подключен к группе узлов. Группа узлов, к которой подключен PV, отображается для него в столбце **Принадлежит** в разделе **Кластеры Kubernetes** → **Диски PV**.

{note:info}
Управление созданными PV через личный кабинет {var(cloud)} доступно только для кластеров {linkto(../../concepts/cluster-generations#k8s-cluster-generations)[text=второго поколения]}.
{/note}

## {heading(Перемещение PV между проектами)[id=k8s-manage-pvs-move-between-projects]}

{note:warn}
После переноса вернуть PV из вашего проекта в {linkto(../../concepts/cluster-generations#k8s-cluster-generations-service-projects)[text=сервисный]} нельзя.
{/note}

1. [Перейдите](https://msk.cloud.vk.com/app/) в личный кабинет {var(cloud)}.
1. Выберите проект, где находится нужный кластер второго поколения.
1. Перейдите в раздел **Кластеры Kubernetes** → **Диски PV**.
1. Переместите PV в свой проект одним из способов: 

   - Нажмите ![](../../../../assets/more-icon.svg "inline") для нужного PV и выберите пункт **Переместить диск в проект**.
   - Выберите нужный PV с помощью флажка и нажмите кнопку **Переместить диск**.

1. Подтвердите перемещение. 

После переноса PV он больше не будет отображаться в разделе **Контейнеры** → **Диски PV** и появится в разделе **Облачные вычисления → Диски**, где вы сможете {linkto(../../../../computing/iaas/instructions/volumes#iaas-volumes)[text=управлять]} им так же, как и любым другим диском.

## {heading(Удаление PV)[id=k8s-manage-pvs-delete]}

1. [Перейдите](https://msk.cloud.vk.com/app/) в личный кабинет {var(cloud)}.
1. Выберите проект, где находится нужный кластер второго поколения.
1. Перейдите в раздел **Кластеры Kubernetes** → **Диски PV**.
1. Выполните одно из действий:
   
   - Нажмите ![](../../../..//assets/more-icon.svg "inline") для нужного PV и выберите пункт **Удалить диск**.
   - Выберите нужный PV с помощью флажка и нажмите кнопку **Удалить диск**.

1. Подтвердите удаление.
