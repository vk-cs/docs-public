В сервисе Cloud Containers вы можете перемещать или удалять [постоянные тома (PV)](/ru/kubernetes/k8s/reference/pvs-and-pvcs), созданные для кластеров Kubernetes. 

Переместить или удалить PV можно, только если он не подключен к группе узлов. Группа узлов, к которой подключен PV, отображается для него в столбце **Принадлежит** в разделе **Кластеры Kubernetes** → **Диски PV**.

{note:info}
Управление созданными PV через личный кабинет VK Cloud доступно только для кластеров [второго поколения](/ru/kubernetes/k8s/concepts/cluster-generations).
{/note}

## Перемещение PV между проектами

{note:warn}
После переноса вернуть PV из вашего проекта в [сервисный](/ru/kubernetes/k8s/concepts/cluster-generations#service-projects) нельзя.
{/note}

1. [Перейдите](https://msk.cloud.vk.com/app/) в личный кабинет VK Cloud.
1. Выберите проект, где находится нужный кластер второго поколения.
1. Перейдите в раздел **Кластеры Kubernetes** → **Диски PV**.
1. Переместите PV в свой проект одним из способов: 

   - Нажмите ![ ](/ru/assets/more-icon.svg "inline") для нужного PV и выберите пункт **Переместить диск в проект**.
   - Выберите нужный PV с помощью флажка и нажмите кнопку **Переместить диск**.

1. Подтвердите перемещение. 

После переноса PV он больше не будет отображаться в разделе **Контейнеры** → **Диски PV** и появится в разделе **Облачные вычисления → Диски**, где вы сможете [управлять](/ru/computing/iaas/instructions/volumes) им так же, как и любым другим диском.

## Удаление PV

1. [Перейдите](https://msk.cloud.vk.com/app/) в личный кабинет VK Cloud.
1. Выберите проект, где находится нужный кластер второго поколения.
1. Перейдите в раздел **Кластеры Kubernetes** → **Диски PV**.
1. Выполните одно из действий:
   
   - Нажмите ![ ](/ru/assets/more-icon.svg "inline") для нужного PV и выберите пункт **Удалить диск**.
   - Выберите нужный PV с помощью флажка и нажмите кнопку **Удалить диск**.

1. Подтвердите удаление.
