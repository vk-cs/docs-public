# {heading(Разрыв соединения в балансировщике нагрузки)[id=k8s-lb-timeout]}

{linkto(../../concepts/network#k8s-network-loadbalancer-integration)[text=Сетевой балансировщик нагрузки]}, который создается через сервис Cloud Containers, использует собственные настройки тайм-аутов. Если ваше приложение работает с длительными сессиями, стандартный тайм-аут может преждевременно разрывать соединение.

### {heading(Решение)[id=k8s-lb-timeout-solution]}

Чтобы изменить тайм-ауты, [обратитесь в техническую поддержку](/ru/contacts).
