# {heading(О сервисе)[id=jupyterhub_info]}

JupyterHub — инструмент для создания многопользовательской среды на основе Jupyter Notebook. Он позволяет запускать для разных пользователей через веб-интерфейс множество отдельных сессий Jupyter Notebook и управлять ими. Подробнее о JupyterHub cм. в [официальной документации](https://jupyterhub.readthedocs.io/en/stable/).

Сервис JupyterHub предоставляет среду для проведения экспериментов с данными. Он содержит набор популярных библиотек и преднастроенное окружение для GPU. Поддерживает одновременную работу множества пользователей.

Основные возможности JupyterHub:

- Гибкая конфигурация:

    - поддержка различных вычислительных бэкендов;
    - возможность использования Docker-контейнеров;
    - настройка пользовательских образов;
    - интеграция с системами хранения данных.

- Образовательные возможности:

    - создание виртуальных классов;
    - управление заданиями и проектами;
    - отслеживание прогресса обучения;
    - совместная работа над проектами.

- Расширяемость:

    - поддержка плагинов и расширений;
    - возможность кастомизации интерфейса;
    - интеграция с CI/CD-инструментами;
    - поддержка различных языков программирования.

Экземпляр JupyterHub можно создать только в конфигурации {linkto(../../instructions/create#jupyterhub_create)[text=Кластер]} (состоит из нескольких узлов). Решение подходит для обработки больших объемов данных и обеспечивает высокую производительность и отказоустойчивость: сбой одного узла не приведет к остановке работы экземпляра.
