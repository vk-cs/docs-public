# {heading(О сервисе)[id=mlflow_info]}

MLflow — платформа с открытым исходным кодом для управления полным жизненным циклом моделей машинного обучения (ML). Позволяет организовать централизованный трекинг и хранение: моделей, параметров экспериментов, артефактов работы дата-специалистов. Подробнее о MLflow см. в [официальной документации](https://mlflow.org/docs/latest/).

Ключевые возможности MLflow:

- Отслеживание экспериментов через компонент MLflow Tracking:

    - логирование метрик и параметров экспериментов;
    - сохранение артефактов: модели, изображения, файлы;
    - визуализация результатов в UI.

- Управление версиями кода через компонент MLflow Projects:

    - стандартизированный формат для упаковки кода;
    - воспроизводимость экспериментов;
    - поддержка различных сред выполнения.

- Упаковка окружений через компонент MLflow Models:

    - создание переносимых окружений для моделей;
    - поддержка множества фреймворков: TensorFlow, PyTorch, Scikit-learn;
    - автоматическое создание REST API;
    - упаковка моделей в Docker-образ для последующего использования в Kubernetes.

- Регистрация моделей через компонент MLflow Registry:

    - централизованное хранилище моделей;
    - управление версиями;
    - контроль доступа и аннотации.

Экземпляр MLflow можно {linkto(../../instructions/create#mlflow_create)[text=создать]} только в конфигурации Кластер (состоит из нескольких узлов). Решение подходит для обработки больших объемов данных и обеспечивает высокую производительность и отказоустойчивость: сбой одного узла не приведет к остановке работы экземпляра.
