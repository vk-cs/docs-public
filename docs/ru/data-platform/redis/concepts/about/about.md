# {heading(О сервисе)[id=redis_info]}

Redis — высокоэффективная и высокоскоростная СУБД NoSQL-типа с открытым исходным кодом, которая хранит данные в оперативной памяти в виде пар ключ/значение. Подробнее о Redis — в [официальной документации](https://redis.io/).

Redis обеспечивает микросекундную задержку при обработке запросов, может обрабатывать миллионы операций в секунду и подходит для задач, в которых приоритетна скорость: кеширование данных, хранение пользовательских сессий, реализация чатов, очереди, брокеров сообщений, аналитики в реальном времени и других.

Области применения сервиса:

- веб-приложения;
- системы реального времени;
- геймификация;
- аналитические системы;
- микросервисная архитектура;
- распределенные приложения.
