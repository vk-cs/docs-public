# {heading(Подключение к экземпляру сервиса)[id=redis_connect]}

Подключиться к экземпляру сервиса Redis можно с помощью любого поддерживаемого клиента.

{include(../../../_includes/_connect.md)[tags=connect-secure]}

## {heading(Подключение через Redis CLI)[id=redis_connect-redis-cli]}

1. Перейдите в раздел **Data Platform** → **Экземпляры сервисов**.
1. Нажмите на название экземпляра сервиса.
1. На странице экземпляра перейдите на вкладку **Общая информация**.
1. Скопируйте строку подключения, указанную в поле **Redis connection string**.
1. Откройте терминал.
1. Выполните команду для подключения к Redis:

   ```console
   redis-cli -u redis://<ЛОГИН>:<ПАРОЛЬ>@<ХОСТ_И_ПОРТ>
   ```

   Здесь:
   
   - `<ЛОГИН>` — логин учетной записи Redis;
   - `<ПАРОЛЬ>` — пароль учетной записи Redis;
   - `<ХОСТ_И_ПОРТ>` — хост и порт из строки подключения.
   

1. Если в процессе выполнения команды возникла ошибка валидации сертификата, добавьте к команде параметры `--tls --insecure`:

   ```console
   redis-cli --tls --insecure -u redis://<user>:<password>@10.101.0.111:2000
   ```
