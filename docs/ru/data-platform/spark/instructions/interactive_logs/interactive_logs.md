# {heading(Просмотр интерактивных логов)[id=spark_interactive_logs]}

В Cloud Spark доступен просмотр интерактивных логов с подробной информацией о выполнении Spark-задач в текстовом или JSON-формате.

{note:warn}
Логи отображается только для завершенных Spark-задач, в том числе с ошибкой, если в их коде не используется функция удаления задачи `job.delete`.
{/note}

Чтобы посмотреть интерактивные логи:

1. Перейдите в раздел **Data Platform** → **Экземпляры сервисов**.
1. Нажмите на название экземпляра сервиса Cloud Spark.
1. На вкладке **Общая информация** скопируйте URL-адрес из пункта **Spark Service**.
1. Перейдите по скопированному URL-адресу. Откроется History Server — веб-интерфейс для визуализации событий завершенных Spark-задач.
1. Найдите интересующую задачу по значению `App Name` и нажмите на ее App ID.
1. Перейдите на вкладку **Executors**.
1. В таблице Executors справа в столбце `logs` нажмите **log**.

   Откроется страница с интерактивными логами выполнения Spark-задачи в текстовом или JSON-формате (зависит от настроек).

   Если логов нет и отображается сообщение `failed to get pod logs`, убедитесь, что в коде Spark-задачи не используется функция `job.delete`. Если функция есть, удалите ее и перезапустите задачу.
