1. Выберите источник:

   - `PostgreSQL` — подключение к базе данных PostgreSQL.
   - `Greenplum` — подключение к базе данных Greenplum.
   - `Clickhouse` — подключение к базе данных ClickHouse.
   - `MySQL` — подключение к базе данных MySQL.
   - `MariaDB` — подключение к базе данных MariaDB.
   {ifndef(public)}
   - `MS SQL` — подключение к базе данных MS SQL.
   - `Oracle DB` — подключение к базе данных Oracle.
   {/ifndef}
   - `Redis` — подключение к базе данных Redis.
   - `Iceberg c хранением данных на S3 VK Cloud` — подключение к {var(s3)} в том же проекте.
   - `Iceberg c хранением данных на внешнем S3` — подключение к внешнему объектному хранилищу S3.
   - `Hive Metastore c хранением данных на HDFS` — подключение к серверу Hive Metastore с хранилищем HDFS.
   - `Hive Metastore c хранением данных на внешнем S3` — подключение к серверу Hive Metastore с хранилищем S3.
   - `Hive Metastore c хранением данных на VK Cloud S3` — подключение к серверу Hive Metastore с хранилищем {ifdef(public)} [VK Object Storage](/ru/storage/s3/concepts/about){/ifdef}{ifndef(public)} {var(s3)}{/ifndef}.
