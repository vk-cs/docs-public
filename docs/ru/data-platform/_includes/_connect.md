<!-- Инфа о TLS-соединении -->

{includetag(connect-secure)}

{note:info}
Соединение с экземпляром сервиса защищено с помощью самоподписанного TLS-сертификата.
{/note}

{/includetag}

<!-- Предупреждение браузера -->

{includetag(browser-warning)}

{note:info}
При переходе по ссылке в браузере отобразится предупреждение, что подключение не защищено. Это связано с тем, что самоподписанный TLS-сертификат не может быть проверен браузером. Соединение безопасно и зашифровано.
{/note}

{/includetag}


<!-- Переходы на небезопасный сайт в разных браузерах -->

{includetag(browsers-action)}

- Google Chrome: нажмите **Дополнительные настройки** → **Перейти на сайт**.
- Mozilla Firefox: нажмите **Дополнительно** → **Перейти на <URL_АДРЕС>**.
- Яндекс Браузер: нажмите **Подробнее** → **Сделать исключение для этого сайта**.
- Safari: нажмите **Подробнее** → **посетить этот сайт**.

{/includetag}

<!-- Предупреждение браузера -->

{includetag(curl-insecure)}

{note:info}
Опция `--insecure` позволяет игнорировать предупреждение о самоподписанном сертификате. Соединение при этом остается зашифрованным и защищенным.
{/note}

{/includetag}
