# {heading(Тарификация)[id=dlh-tariffication]}

Общей тарификации на весь сервис Data Lakehouse не предусмотрено. Вместо этого каждый отдельный сервис, входящий в состав Data Lakehouse, тарифицируется по своим правилам. Для получения точной информации о стоимости конкретного сервиса, используемого в рамках Data Lakehouse, ознакомьтесь с разделом **Тарификация** соответствующего сервиса.
