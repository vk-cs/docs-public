# {heading(О сервисе)[id=nifi_info]}

NiFi — платформа для автоматизации и управления потоками данных. Сервис разработан на основе Apache NiFi. Подробнее в [официальной документации](https://nifi.apache.org).

Сервис предназначен для автоматизации процессов сбора, обработки и передачи информации между различными системами и позволяет создавать сложные конвейеры данных с возможностью мониторинга и управления в реальном времени.

Возможности сервиса:

- Сбор данных из множества источников.
- Преобразование и очистка данных.
- Перенос данных между системами.
- Создание отказоустойчивых решений для обработки данных.
- Визуализация и мониторинг потоков данных.
