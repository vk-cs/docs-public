# {heading(Введение)[id=dp_index]}

## {heading(Общая информация)[id=dp_index_inform]}

Полное наименование системы: {var(data-p)}.

{var(data-p)} — платформа для работы с проектами обработки данных в ЦОДе заказчика. Предоставляет удобный доступ к сервисам хранения, обработки и управления данными.

## {heading(Возможности {var(data-p)})[id=dp_main_functionalities]}

- Поиск и отображение проектов обработки данных.
- Централизованное управление проектами, включая:

   - Создание, удаление, изменение проектов.
   - Выделение проекту необходимых ресурсов и инструментов в связанных подсистемах {var(data-p)}, в том числе на основе шаблонов конфигурации.

- Отображение и навигация всех подключенных к проекту инструментов.
- Отображение окружений, доступных проектам.
- Создание и удаление инфраструктуры для окружения проектов.
- Создание и удаление специальных вычислительных узлов для работы конвейеров обработки и анализа данных.
- Отображение статистики используемых инструментов и ресурсов в подсистемах.
