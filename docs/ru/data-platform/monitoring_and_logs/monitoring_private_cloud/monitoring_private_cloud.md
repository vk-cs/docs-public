# {heading(Мониторинг Data Platform в составе Private Cloud)[id=monitoring-private_cloud]}

Часть сервисов {var(data-p)}, развернутой в составе Private Cloud, интегрированы с системой мониторинга Cloud Monitoring. Мониторинг собранных метрик позволяет контролировать состояние сервисов {var(data-p)}, своевременно выявлять неполадки и сбои в работе сервисов. Подробнее о работе Cloud Monitoring — в документе **Руководство администратора** VK Private Cloud в разделе **Компоненты** → [Cloud Monitoring](https://cloud.vk.com/docs/on-premises/private-cloud/ru/4_3/admin-guide/components/cloud_monitoring).

Список сервисов, интегрированных с Cloud Monitoring:

- Cloud Kafka;
- Cloud Airflow;
- Cloud Trino;
- Redis.
