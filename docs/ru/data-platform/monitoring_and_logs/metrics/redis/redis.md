# {heading(Cloud Redis)[id=redis_metrics]}

В сервисе Cloud Redis метрики собираются в формате Prometheus. Для каждой метрики автоматически создается панель, которую можно отобразить на дашборде. Список метрик перечислен в {linkto(#redis_tab_metrics)[text=таблице %number]}.

{caption(Таблица {counter(table)[id=numb_redis_tab_metrics]} — Метрики Redis)[align=right;position=above;id=redis_tab_metrics;number={const(numb_redis_tab_metrics)}]}
[cols="1,1", options="header"]
|===
|Имя метрики
|Описание

|`% CPU`
|Загрузка CPU в процентах относительно установленного лимита за выбранный интервал времени

|`% Memory`
|Использование памяти в процентах относительно установленного лимита за выбранный интервал времени

|`Read IOPS`
|Общее количество операций чтения с диска за выбранный интервал времени

|`Write IOPS`
|Общее количество операций записи на диск за выбранный интервал времени

|`Read MB`
|Объем данных, прочитанных с диска за выбранный интервал времени, в МБ

|`Write, Mbytes`
|Объем данных, записанных на диск за выбранный интервал времени, в МБ

|`Network received, Mbytes`
|Объем входящего сетевого трафика за выбранный интервал времени, в МБ

|`Network sent, Mbytes`
|Объем исходящего сетевого трафика за выбранный интервал времени, в МБ

|`Connected clients`
|Текущее количество активных клиентских подключений

|`Total commands per second`
|Общее количество выполненных команд в секунду за выбранный интервал времени

|`Blocked clients`
|Текущее количество заблокированных клиентских подключений

|`Memory fragmentation ratio`
|Текущее значение коэффициента фрагментации памяти

|`Keyspace hits per second`
|Количество попаданий запросов к данным в кеш за секунду на выбранном интервале времени

|`Keyspace misses per second`
|Количество пропусков запросов к данным в кеш за секунду на выбранном интервале времени

|`Hit ratio`
|Доля попаданий запросов к данным в кеш за выбранный интервал времени

|`Expired keys (per sec)`
|Количество истекших ключей в секунду за выбранный интервал времени

|`Evicted keys per sec`
|Количество удаленных ключей из-за нехватки памяти в секунду за выбранный интервал времени

|`Connected slaves`
|Текущее количество подключенных реплик

|`Replication offsets for master`
|Текущее значение смещения репликации на мастер-узле в байтах

|`Replication offsets for slave`
|Текущее значение смещения репликации на узле реплики в байтах

|===
