# {heading(Cloud ClickHouse)[id=clickhouse_metrics]}

В сервисе Cloud ClickHouse метрики собираются в формате Prometheus. Для каждой метрики автоматически создается панель, которую можно отобразить на дашборде.

## {heading(Метрики узлов ClickHouse)[id=clickhouse_metrics_instance]}

Список метрик узлов ClickHouse перечислен в {linkto(#tab_clickhouse_metrics_instance)[text=таблице %number]}.

{caption(Таблица {counter(table)[id=numb_tab_clickhouse_metrics_instance]} — Метрики узлов ClickHouse)[align=right;position=above;id=tab_clickhouse_metrics_instance;number={const(numb_tab_clickhouse_metrics_instance)}]}
[cols="1,2", options="header"]
|===
|Имя метрики
|Описание

|`Использование CPU`
|Процент использования CPU относительно установленного лимита за выбранный интервал времени

|`Использование RAM`
|Процент использования RAM относительно установленного лимита за выбранный интервал времени

|`Чтение с диска`
|Количество операций чтения с диска за выбранный интервал времени

|`Запись на диск`
|Количество операций записи на диск за выбранный интервал времени

|`Скорость чтения с диска`
|Скорость чтения данных с диска за выбранный интервал времени, в мебибайтах в секунду

|`Скорость записи на диск`
|Скорость записи данных на диск за выбранный интервал времени, в мебибайтах в секунду

|`Входящий сетевой трафик`
|Скорость входящего сетевого трафика за выбранный интервал времени, в мебибайтах в секунду

|`Исходящий сетевой трафик`
|Скорость исходящего сетевого трафика за выбранный интервал времени, в мебибайтах в секунду

|`Использование диска`
|Процент использования дискового пространства за выбранный интервал времени

|`HTTP-соединения`
|Текущее количество активных HTTP-соединений

|`TCP-соединения`
|Текущее количество активных TCP-соединений

|`Количество выполняемых запросов`
|Текущее количество выполняемых запросов

|`Запросы в S3`
|Текущее количество активных запросов в S3

|`Время выполнения запросов`
|Среднее количество микросекунд CPU, затраченных на запросы за секунду за выбранный интервал времени

|`Скорость SELECT-запросов`
|Количество запросов `SELECT` в секунду за выбранный интервал времени

|`Скорость INSERT-запросов`
|Количество запросов `INSERT` в секунду за выбранный интервал времени

|`Ошибочные SELECT-запросы`
|Количество ошибочных запросов `SELECT` в секунду за выбранный интервал времени

|`Ошибочные INSERT-запросы`
|Количество ошибочных запросов `INSERT` в секунду за выбранный интервал времени

|`Скорость вставки данных`
|Скорость вставки данных за выбранный интервал времени, в байтах в секунду

|===
