# {heading(О сервисе)[id=opensearch_info]}

OpenSearch — программный комплекс для создания поисковых систем и аналитических решений. Сервис содержит поисковую платформу OpenSearch и веб-интерфейс OpenSearch Dashboards для визуализации и анализа данных. Подробнее см. в [официальной документации](https://docs.opensearch.org/docs/latest/).

Области применения сервиса:

- поисковые системы;
- аналитика логов;
- мониторинг приложений;
- анализ пользовательского поведения;
- создание бизнес-аналитических решений.

Экземпляр OpenSearch можно {linkto(../../instructions/create#opensearch_create)[text=создать]} только в конфигурации **Кластер** (состоит из нескольких узлов). Решение подходит для обработки больших объемов данных и обеспечивает высокую производительность и отказоустойчивость: сбой одного узла не приведет к остановке работы экземпляра.
