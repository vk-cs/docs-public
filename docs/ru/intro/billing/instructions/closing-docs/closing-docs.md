# {heading(Скачивание закрывающих документов)[id=billing-closing-docs]}

Если вы работаете по {linkto(/ru/intro/billing/concepts/physical-corporate#billing-physical-corporate-postpay)[text=постоплате]}, вы можете скачать закрывающие документы — счета на оплату и универсальные передаточные документы (УПД) — в личном кабинете {var(cloud)}.

1. [Перейдите](https://msk.cloud.vk.com/app/) в личный кабинет {var(cloud)}.

{include(/ru/_includes/_balance_page.md)}

1. Перейдите на вкладку **Закрывающие документы**.
1. (Опционально) Установите фильтры на отображаемые записи.
1. Нажмите на значок ![Скачать](assets/download_icon.svg "inline") для нужного документа.

   Если вы хотите скачать несколько документов сразу, выберите их с помощью флажка и нажмите на значок ![Скачать](assets/download_icon.svg "inline").

Документы будут загружены на ваш компьютер.
