# {heading(Отчетные документы)[id=billing-report]}

Пользователи получают от {var(cloud)} финансовые документы по всем совершенным платежам. Состав таких документов различается для физических и юридических лиц.

## {heading(Физические лица)[id=billing-report-individuals]}

При пополнении лицевого счета физические лица получают чек об оплате в соответствии с [ФЗ-54](https://base.garant.ru/12130951/) «О применении контрольно-кассовой техники». Чек отправляется на электронную почту владельца проекта {var(cloud)}. Оригинал чека об оплате не предоставляется.

## {heading(Юридические лица и ИП)[id=billing-report-legal-entities]}

В конце каждого отчетного периода {var(cloud)} предоставляет контрагенту пакет бухгалтерских {linkto(#billing-report-documents-composition)[text=отчетных документов]}. Отчетный период равен одному календарному месяцу, документы подписываются последней датой этого месяца.

Документы отправляются получателям тем {linkto(#billing-report-receive-ways)[text=способом]}, который был выбран в начале сотрудничества, в следующие сроки:

- Если требуется согласование сумм — около 15 числа месяца, следующего за отчетным периодом.
- Если согласования не требуется — 7–10 числа месяца, следующего за отчетным периодом.

{note:info}
Если у организации или ИП несколько {linkto(../../../../tools-for-using-services/account/concepts/projects#tools-account-concepts-projects)[text=проектов]}, формируется один пакет документов на все проекты. По запросу могут предоставляться отдельные пакеты документов на каждый проект.
{/note}

Как начать получать от {var(cloud)} отчетные документы, читайте в статье {linkto(../../../../intro/billing/instructions/corporate#billing-corporate-registration-complete)[text=Донастройка проекта для ЮЛ]}. По всем вопросам, связанным с отчетными документами, пишите в отдел документооборота {var(cloud)} на почту [docs_vktech@vk.company](mailto:docs_vktech@vk.company).

{note:warn}
В отчетных документах отражаются только расходы, проведенные как банковские переводы от юридического лица или ИП.
{/note}

### {heading(Состав отчетных документов)[id=billing-report-documents-composition]}

{tabs}

{tab(Работа по договору публичной оферты)}

Имеется в виду работа по [договорам оферты сервисов](/ru/start/legal). Эта схема по умолчанию используется во всех проектах, для физических и юридических лиц.

Юридическим лицам и ИП, работающим по такому договору, предоставляется только один документ — универсальный передаточный документ (УПД). Форма УПД содержит все реквизиты счет-фактуры, установленные п.п. 5, 5.1, 5.2 ст. 169 НК РФ.

Счет-фактура и Акт выполненных работ не предоставляются, так как УПД заменяет собой оба эти документа.

{/tab}

{tab(Работа по индивидуальному договору)}

Состав отчетных документов зависит от конкретных условий договора. В частности, есть различия для организаций, работающих {linkto(../physical-corporate#billing-physical-corporate-payment-scheme)[text=по предоплате и по постоплате]}.

{tabs}

{tab(Предоплата)}

- Основные документы: УПД.
- Дополнительные документы, если требуются по договору:

  - счет-фактура;
  - акт выполненных работ.

   {note:info}
   Если предоставляется cчет-фактура, УПД не предоставляется. Акт выполненных работ может предоставляться вместе с УПД.
   {/note}

Счет на оплату не предоставляется. Организации, работающие по предоплате, {linkto(../../instructions/bill-generation#billing-bill-generation)[text=формируют]} его самостоятельно в личном кабинете.

{/tab}

{tab(Постоплата)}

- Основные документы:

  - УПД;
  - счет на оплату, который предоставляет {var(cloud)}.

- Дополнительные документы, если требуются по договору:

  - счет-фактура;
  - акт выполненных работ.

  {note:info}
   Если предоставляется cчет-фактура, УПД не предоставляется. Акт выполненных работ может предоставляться вместе с УПД.
  {/note}

{/tab}

{/tabs}

{/tab}

{/tabs}

Кроме документов, предусмотренных договором, в личном кабинете доступен отчет по биллингу за выбранный период. Он содержит детализацию по потреблению ресурсов и используется для подтверждения сумм в отчетных документах. Информация о скачивании доступна в разделе {linkto(../../instructions/detail#includes-detail-report-download)[text=Скачивание отчета]}.

Акт сверки с детализацией взаиморасчетов предоставляется по запросу через [техническую поддержку](/ru/contacts).

### {heading(Способы получения отчетных документов)[id=billing-report-receive-ways]}

Документы отправляются через систему электронного документооборота (ЭДО).

Если электронный документооборот с организацией невозможен, {var(cloud)} отправляет бумажные оригиналы документов на адрес организации.

{note:info}
{var(cloud)} никогда не отправляет документы на юридический адрес без явной договоренности с организацией.
{/note}

#### {heading(Доставка оригиналов документов)[id=billing-report-originals-delivering]}

Чтобы настроить бумажный документооборот, обратитесь в отдел документооборота {var(cloud)}:

1. Напишите на почту [docs_vktech@vk.company](mailto:docs_vktech@vk.company).
1. Укажите:

   - Желаемый способ доставки документов: по почте или курьерской службой.
   - (Если выбрана доставка курьерской службой) Контактный телефон получателя.
   - ФИО получателя.
   - Адрес получателя. Если адрес совпадает с юридическим адресом организации, укажите это явно в запросе.

#### {heading(Скан-копии документов)[id=billing-report-scans]}

{var(cloud)} может предоставлять скан-копии отчетных документов в формате PDF по электронной почте — каждый месяц или разово, по запросу. Для получения скан-копий напишите в отдел документооборота {var(cloud)} на почту [docs_vktech@vk.company](mailto:docs_vktech@vk.company).

## {heading(Учет налогов)[id=billing-report-tax-accounting]}

Для налоговых резидентов РФ в платежных и отчетных документах указывается сумма НДС.
