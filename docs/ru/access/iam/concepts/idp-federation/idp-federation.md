# {heading(Федерация удостоверений)[id=iam-idp-federation]}

*Федерация удостоверений* — это система доверительных отношений между независимыми доменами безопасности, при котором каждый домен сохраняет собственную систему управления доступом, но принимает результаты аутентификации, выполненные другим доменом, входящим в федерацию. При федеративном входе пользователь, прошедший аутентификацию в одном домене, получает доступ к ресурсам другого. Например, пользователь может заходить в VK Cloud без ввода логина и пароля, используя учетные данные для входа в свою корпоративную сеть.

За аутентификацию и хранение учетных данных в режиме федерации отвечает отдельный сервис — *поставщик удостоверений* (Identity Provider, IdP).

Вы можете подключить к VK Cloud вашего поставщика удостоверений. Доступна настройка федерации удостоверений через:

- [Active Directory](/ru/access/iam/how-to-guides/federations/add-fim-for-ad),
- [Keycloak](/ru/access/iam/how-to-guides/federations/keycloak-to-fim),
- [Microsoft Entra](/ru/access/iam/how-to-guides/federations/ms-entra-fim).

Поставщик удостоверений должен поддерживать стандарт SAML 2.0.
