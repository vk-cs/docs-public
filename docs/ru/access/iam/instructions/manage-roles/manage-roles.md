# {heading(Настройка прав доступа)[id=iam-manage-roles]}

Операции управления доступом доступны следующим ролям личного кабинета: владельцу, суперадминистратору и администратору пользователей (IAM). Просмотр участников проекта доступен также администратору проекта и наблюдателю.
