Жедел жадтың өнімділігі секундына орындалатын оқу және жазу операцияларының санымен, сондай-ақ секундына берілетін деректер көлемімен анықталады.

Жедел жадтың өнімділігін тестілеу үшін бұдан әрі [sysbench](https://github.com/akopytov/sysbench) утилитасы пайдаланылады.

## Дайындық қадамдары

{include(/kz/_includes/_testing_preparatory.md)}

## 1. Жедел жадты тестілеуді орындаңыз

1. [Қосылыңыз](/kz/computing/iaas/instructions/vm/vm-connect/vm-connect-nix) ВМ-ге.
1. ВМ-ге sysbench утилитасын орнатыңыз:

    ```console
    sudo apt install sysbench
    ```

1. Тізбекті оқу және жазу операцияларын орындауды іске қосыңыз:

    ```console
    sysbench --test=memory run
    ```

## 2. Тестілеу нәтижелерін тіркеңіз

Команда шығысын файлға сақтаңыз. Ең көрсеткішті параметрлер:

* `Operations per second` — секундына орындалатын оқу және жазу операцияларының саны;
* `MiB transferred` — секундына берілген деректер көлемі.

Енді сіз бұл деректерді басқа бұлттық платформалардың өнімділігімен салыстыра аласыз.

{note:warn}

Өнімділікті салыстыру объективті болуы үшін, сізді қызықтыратын барлық платформаларда бірдей тестілеу әдістемелерін пайдаланыңыз.

{/note}

## Пайдаланылмайтын ресурстарды жойыңыз

Жасалған ресурстар тарификацияланады. Егер олар енді қажет болмаса, оларды жойыңыз:

1. [Жойыңыз](/kz/computing/iaas/instructions/vm/vm-manage#iaas-vm-delete) тестілік ВМ-ді.
1. ВМ орналастырылған [ішкі желіні](/kz/networks/vnet/instructions/net#udalenie_podseti) және [желіні](/kz/networks/vnet/instructions/net#vnet-net-delete) жойыңыз.
