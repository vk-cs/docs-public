Процессорлардың өнімділік параметрлерін көбірек алу үшін тестілеуге арналған бірнеше утилитаны бір уақытта пайдаланыңыз:

* [sysbench](https://github.com/akopytov/sysbench);
* [Geekbench 6](https://www.geekbench.com/);
* [7-Zip Benchmark](https://www.7-cpu.com/);
* [UnixBench](https://github.com/ProTechEx/unixbench).

## Дайындық қадамдары

{include(/kz/_includes/_testing_preparatory.md)}

## 1. sysbench утилитасының көмегімен тестілеуді орындаңыз

1. ВМ-ге sysbench утилитасын орнатыңыз:

    ```console
    sudo apt install sysbench
    ```

1. CPU өнімділігін бір ағынмен тестілеуді қатарынан 5 рет іске қосыңыз:

    ```console
    sysbench cpu --threads=1 --time=60 run
    ```

1. CPU өнімділігін 16 ағынмен тестілеуді қатарынан 5 рет іске қосыңыз:

    ```console
    sysbench cpu --threads=16 --time=60 run
    ```

1. Нәтижелерді [кестеге](../assets/cpu_testing.xlsx "download") енгізіңіз. Ең көрсеткішті мән бес іске қосу қорытындысы бойынша `CPU Speed (events/s)` параметрінің орташа мәні болады.

## 2. Geekbench 6 утилитасының көмегімен тестілеуді орындаңыз

1. ВМ-ге Geekbench 6 утилитасын орнатыңыз:

    ```console
    wget https://cdn.geekbench.com/Geekbench-6.2.1-Linux.tar.gz
    tar xf Geekbench-6.2.1-Linux.tar.gz
    cd Geekbench-6.2.1-Linux
    ```

1. CPU өнімділігін тестілеуді қатарынан 5 рет іске қосыңыз:

    ```console
    ./geekbench6
    ```

1. Тестілеу нәтижелеріне алынған сілтемелерді, сондай-ақ `Single-Core Score` және `Multi-Core Score` параметрлері үшін 5 іске қосу бойынша орташа мәнді сақтаңыз.

## 3. 7-Zip Benchmark утилитасының көмегімен тестілеуді орындаңыз

1. ВМ-ге 7-Zip Benchmark утилитасын орнатыңыз:

    ```console
    sudo apt install p7zip-full
    ```

1. CPU өнімділігін бір ағынмен тестілеуді қатарынан 5 рет іске қосыңыз және команда шығысын файлға сақтаңыз:

    ```console
    7z b -md26 -mmt1 5
    ```

1. CPU өнімділігін 16 ағынмен тестілеуді қатарынан 5 рет іске қосыңыз және команда шығысын файлға сақтаңыз:

    ```console
    7z b -md26 -mmt16 5
    ```

1. Бір ағынмен және 16 ағынмен іске қосу командасы үшін бес іске қосу қорытындысы бойынша `MIPS` параметрінің орташа мәндерін бөлек есептеңіз.

## 4. UnixBench утилитасының көмегімен тестілеуді орындаңыз

1. ВМ-ге UnixBench утилитасын орнатыңыз:

    ```console
    sudo snap install unixbench
    ```

1. CPU өнімділігін бір ағынмен тестілеуді қатарынан 5 рет іске қосыңыз және команда шығысын файлға сақтаңыз:

    ```console
    unixbench -c 1
    ```

1. CPU өнімділігін 16 ағынмен тестілеуді қатарынан 5 рет іске қосыңыз және команда шығысын файлға сақтаңыз:

     ```console
    unixbench -c 16
    ```

1. `Benchmark Index Score` параметрінің мәнін бір ағынмен және 16 ағынмен іске қосу командасы үшін бөлек сақтаңыз.

## 5. Тестілеу нәтижелерін тіркеңіз

CPU өнімділігін тестілеудің алдыңғы қадамдарында алынған барлық нәтижелерді жинаңыз. Енді сіз бұл деректерді басқа бұлттық платформалардың өнімділігімен салыстыра аласыз.

{note:warn}

Өнімділікті салыстыру объективті болуы үшін, сізді қызықтыратын барлық платформаларда бірдей тестілеу әдістемелерін пайдаланыңыз.

{/note}
