VK Cloud 152-ФЗ «Дербес деректер туралы» заңына сәйкес дербес деректерді (ПДн) өңдеу үшін пайдаланылуы мүмкін. VK Cloud өңделетін дербес деректер үшін бірінші қорғалу деңгейі (УЗ-1) қамтамасыз етілетінін растайтын Ресей ФСТЭК аттестатына ие.

Дербес деректерді өңдеу [152-ФЗ бұлты](https://cloud.vk.com/solutions/152-fz/) сервисінің көмегімен қамтамасыз етіледі. Бұл сервис бұлтта келесі қорғалу деңгейлеріндегі дербес деректерді өңдеуге жарамды қорғалған және аттестатталған серверлерде дербес деректерді орналастырады:

- УЗ-3;
- УЗ-2;
- УЗ-1.

{note:warn}

Пайдаланушылардың барлық дербес деректері Ресей Федерациясының аумағында сақталады.

{/note}

Қауіпсіздік үшін жауапкершілікті VK Cloud пен пайдаланушы арасында бөлу қағидаты қолданылады. Сонымен қатар, дербес деректерді сақтау үшін жауапкершілік те бөлінеді.

Жауапкершілікті бөлу туралы ақпаратпен [Бұлт қауіпсіздігі](https://cloud.vk.com/cloud-security/) бетінде танысуға болады.

VK Cloud [жеке кабинетте](https://kz.cloud.vk.com/app/) пайдаланушыларға құқық қорғау органдары тарапынан ПДн-ді ашуға қатысты заңды міндеттеуші кез келген сұрау туралы мұндай сұрау келіп түскен сәттен бастап 72 сағат ішінде хабарлайды.

VK Cloud үшінші тұлғаларға ПДн ашудың барлық фактілерін тіркейді, соның ішінде қандай ПДн, кімге және қай уақытта ашылғаны туралы ақпаратты да.

VK Cloud тиісті пайдаланушыларды мынадай жағдайларда хабардар етеді:

- олардың ПДн-іне рұқсатсыз қол жеткізу;
- ПДн жоғалуына, ашылуына немесе өзгертілуіне әкелетін жабдыққа немесе өңдеу құралдарына рұқсатсыз қол жеткізу фактілері.

{note:info}

Бұлтты қызметтерді ұсыну кезінде ПДн өңдеуге байланысты барлық сұрақтар бойынша [техникалық қолдауға](/kz/contacts) жүгініңіз.

{/note}
