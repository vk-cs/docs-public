## Саясаттың мақсаты

Осы саясат VK тобына кіретін компаниялардың ақпараттық қауіпсіздік саясаттарының талаптарын (бұдан әрі — VK), VK Cloud бұлттық қызметтерін ұсыну кезінде ақпараттық қауіпсіздікті (бұдан әрі — ИБ) басқару мен қамтамасыз етуге қатысты нақтылайды.

## Саясаттың қолданылу аясы

Қолданылуы: VK Cloud

Қолданыла бастайтын күн: 01.09.2023

## VK Cloud сервистерін ұсыну кезінде ИБ-ны басқару және қамтамасыз ету

### ИБ саласындағы жауапкершілікті бөлу

- Бұлттық қызметтерді пайдалану кезінде ИБ саласындағы міндеттер мен жауапкершілік VK Cloud пен бұлттық қызметтерді тұтынушылар (бұдан әрі — клиент) арасында бөлінеді. Жауапкершілікті бөлу шекараларының сипаттамасы VK Cloud әлеуетті және қолданыстағы клиенттеріне қолжетімді жария құжаттама аясында өзекті күйде сақталады.

### Клиенттермен коммуникация

- VK Cloud клиенттерді сервистер жұмысының мәртебесі, оларға әсер етуі мүмкін ақаулар мен өзгерістер туралы уақтылы хабардар етуді қамтамасыз етеді. Мұндай хабарлау VK Cloud сайтындағы [Сервистер жұмысының мәртебесі](https://status.kz.cloud.vk.com) бөлімі арқылы, сондай-ақ жеделдікті арттыру үшін [арнайы Telegram-арнада](https://t.me/vk_cloud_alerts) жүргізіледі.

### ИБ бұзушылықтары туралы хабарламалар

- ИБ бұзушылықтары кезінде VK Cloud пен клиенттер арасындағы тиімді өзара іс-қимыл ИБ инциденттерінің ықтимал салдарын азайтудың маңызды факторы болып табылады. VK-да ИБ инциденттерін басқару процесі енгізілген және тұрақты түрде жетілдіріліп отырады.
- VK Cloud өз клиенттеріне ИБ оқиғалары мен инциденттері туралы хабарлау тетіктерін ұсынады. VK Cloud пайдаланушыларды оларға әсер еткен инциденттер туралы хабардар етуге міндеттенеді. Инциденттерді тергеу қажет болған жағдайда VK Cloud клиенттермен өзара іс-қимылды және қажетті ақпаратты ұсынуды қамтамасыз етуге кепілдік береді.

### VK Cloud бұлттық сервистерін әзірлеу және енгізу кезіндегі ИБ

- VK Cloud бұлттық сервистерін әзірлеу және енгізу ИБ талаптарын ескере отырып орындалады.
- VK Cloud бұлттық сервистері үшін ИБ талаптары әзірленетін және енгізілетін сервистер архитектурасының қауіпсіздігін талдау және ИБ саласындағы үздік практикалар негізінде қалыптастырылады.
- Әзірлеу нәтижелері бойынша әзірленген сервистерді немесе оларға енгізілген өзгерістерді жан-жақты тестілеу, оның ішінде ИБ талаптарының іске асырылуын және код қауіпсіздігін бақылау жүргізіледі.
- Өнімдік командалар мен әзірлеу командалары үшін қауіпсіз әзірлеу бойынша тренингтер мен хабардарлықты арттыру шаралары өткізіледі.

### Уәкілетті инсайдерлерден туындайтын тәуекелдер, қол жеткізуді басқару

- VK Cloud уәкілетті инсайдерлердің клиент деректеріне рұқсатсыз қол жеткізу тәуекелін ескереді және оны барынша азайту үшін шаралар қабылдайды. VK Cloud сақтау қоймалары мен дерекқорларға қол жеткізуді қатаң бақылайды.
- VK Cloud қызметкерлері мен контрагенттерінің қол жеткізуін басқару саясаты енгізілген, ИБ оқиғалары тіркеледі және бақыланады, аутентификация әдістері регламенттелген. VK Cloud ресурстарына тек VK немесе VK мердігерлерінің өкілетті қызметкерлері ең аз артықшылықтар қағидаты негізінде қол жеткізеді.
- VK Cloud ақпараттық ресурстарына қатысты артық құқықтардың жиналуы, рұқсатсыз қол жеткізу және заңсыз пайдалану салдарынан туындайтын ИБ тәуекелдерін азайту үшін есептік жазбалар мен олардың құқықтары қайта қаралып отырады.
- Клиент деректеріне қол жеткізу қатаң шектелген және тек лауазымдық міндеттерін орындау үшін осы қол жеткізу қажет болатын қызметкерлерге ғана беріледі. Барлық қызметкерлермен құпиялылық туралы келісімге қол қойылады, ИБ саласында тұрақты оқыту және хабардарлықты арттыру жүргізіледі.

### Клиенттердің өмірлік циклін және қол жеткізуін басқару

- Клиенттің өмірлік циклі VK Cloud платформасында есептік жазбаны жасаудан басталады. Бұдан әрі клиент пайдаланушыларының қол жеткізу құқықтарын басқаруды, оның ішінде пайдаланушыларды құруды, қол жеткізу деңгейін және екі факторлы аутентификацияны баптауды және т.б. клиенттің өзі жүзеге асырады.
- VK Cloud өз клиенттеріне олардың өмірлік циклінің барлық кезеңдерінде техникалық қолдау аясында қолдау көрсетеді.
- Клиент деректері не VK Cloud платформасындағы есептік жазба жойылған кезде, не ұсынылатын сервистер үшін төлем жасалмаған жағдайда клиент ажыратылған кезде жойылады.

### Виртуализация қауіпсіздігі

- VK Cloud виртуализация қауіпсіздігін қамтамасыз етуге назар аударады. Ол үшін VK Cloud виртуалды инфрақұрылымға қол жеткізуді бақылау мен басқаруды, виртуалды инфрақұрылымға рұқсатсыз қол жеткізуден қорғауды, желілік шабуылдардан қорғауды қамтамасыз етеді және осалдықтарды басқарады.

### Бұлттық сервистер клиенттерін оқшаулау

- VK Cloud инфрақұрылымдық ресурстары виртуализация ортасының жұмысы үшін пайдаланылмайтын арнайы бөлінген физикалық серверлерде іске қосылған. Клиенттік желілерден сервистік желілерге қол жеткізу мүмкін емес.
- VK Cloud клиентінің виртуалды ортасы басқа клиенттерден оқшауланған. Ақпараттық ағындарды басқару клиенттік виртуалды машиналар деңгейінде жүзеге асырылады.
- Клиенттер арасындағы оқшаулау мына деңгейлерде қамтамасыз етіледі: желі деңгейінде, гипервизор деңгейінде және қолданба деңгейінде. Виртуалды машинаны жасау кезінде клиент желіаралық экрандау қағидаларын өзі баптайды немесе ұсынылатын желілік профильді таңдайды.
