# {heading(Жылдам бастау)[id=billing-quick-start]}

{include(/kz/_includes/_translated_by_ai.md)}

Жылдам бастау сервистер ақысын төлеу үшін жаңа жобаны баптауға көмектеседі.

Жылдам бастау қадамдарының барлығын орындағаннан кейін, сіз жоба дербес шотын алғаш рет толтырасыз.
