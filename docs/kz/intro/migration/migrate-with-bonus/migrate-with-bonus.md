Ресурстарды VK Cloud ішіне VK Cloud мамандарының қолдауымен көшіруге болады.

VK Cloud клиенттерімен жұмыс жөніндегі менеджерден кеңес алыңыз. Ол көшіру туралы ақпарат береді және қажеттіліктеріңізге сай инфрақұрылымның дұрыс нұсқасын таңдауға көмектеседі.

## Көшіру қалай жүреді

1. _Жоспарлау_. Сіздің техникалық сарапшыларыңызбен бірге VK Cloud менеджерлері:

   - тиімді инфрақұрылымды құрады;
   - көшірудің қадамдық стратегиясын жасайды.
   
1. _Көшіру процесі_. VK Cloud мамандары сіздің ресурстарыңызды платформаға көшіреді.

1. _Тестілеу_. Барлығы дұрыс жұмыс істейтінін тексересіз.

Көшіру сәтті аяқталғаннан кейін жеке кабинеттегі шотқа 50000-ға дейін [бонус](/kz/intro/billing/concepts/balance#bonuses) аласыз. Сондай-ақ 30 күн бойы [базалық техникалық қолдауға](/kz/start/support/support-info) қосымша ретінде VK Cloud көшіру жөніндегі мамандарынан тікелей жеке консультациялар қолжетімді болады.

Өтінімді [152-ФЗ бұлты](https://cloud.vk.com/solutions/152-fz/) бетінде қалдыруға болады.
