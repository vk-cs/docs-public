Сіз VK Cloud ішінде өзіңіздің [жобаларыңызды](/kz/tools-for-using-services/account/concepts/projects) жасай аласыз. Жоба жасағаннан кейін сіз оның [иесі](/kz/access/iam/concepts/rolesandpermissions#roli_dlya_obshchego_upravleniya_proektom) боласыз.

{note:info}

[Базалық квоталар](/kz/tools-for-using-services/account/concepts/quotasandlimits#quotasandlimits-general) аясында бір пайдаланушы 5 жобадан артық жасай алмайды. Бұл санды қолданыстағы [лимит](/kz/tools-for-using-services/account/concepts/quotasandlimits) шегінде [ұлғайтуға](/kz/tools-for-using-services/account/instructions/project-settings/manage#project-increase-quota) болады.
