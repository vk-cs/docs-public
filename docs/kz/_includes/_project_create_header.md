Бұл санды қолданыстағы [лимит](/kz/tools-for-using-services/account/concepts/quotasandlimits#kvoty_i_tehnicheskie_limity) шегінде [ұлғайтуға](/kz/tools-for-using-services/account/instructions/project-settings/manage#increase-quota) болады.
