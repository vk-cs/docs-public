1. Аддонды баптау және онымен жұмыс істеу кезінде қателерді болдырмау үшін [Argo CD ресми құжаттамасын](https://argo-cd.readthedocs.io/en/stable/) зерттеңіз. Атап айтқанда, аддонмен жұмыс істеу кезінде сізге келесілер қажет болады:

    - Argo CD веб-интерфейсіне қол жеткізу;
    - кластер конфигурациясы бар Git-репозиторийді қосу;
    - кластер күйін Git-репозиториймен қолмен немесе автоматты түрде синхрондауды іске қосу.
