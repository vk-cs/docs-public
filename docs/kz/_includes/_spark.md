{includetag(intro)}

Apache Spark — үлкен деректерді үлестірілген өңдеуге арналған сервис. Ол Java, Scala, Python және R тілдеріндегі API интерфейстерінен, сондай-ақ SQL үшін [Spark SQL](https://spark.apache.org/docs/latest/sql-programming-guide.html) өңдеу құралдарынан, [Pandas API](https://spark.apache.org/docs/latest/api/python/getting_started/quickstart_ps.html), машиналық оқытуға арналған [MLlib](https://spark.apache.org/docs/latest/ml-guide.html), графтарды өңдеуге арналған [GraphX](https://spark.apache.org/docs/latest/graphx-programming-guide.html) және ағындық өңдеуге арналған [Structured Streaming](https://spark.apache.org/docs/latest/structured-streaming-programming-guide.html) құралдарынан тұрады. Көбіне Spark Hadoop кластерінің құрамында қолданылады.

Cloud Spark — [Apache Spark Operator](https://github.com/GoogleCloudPlatform/spark-on-k8s-operator) және VK Cloud ұсынатын [PaaS Kubernetes](/kz/kubernetes/k8s) негізіндегі шешім. Ол Hadoop кластерін пайдаланбай, Docker Registry ішіндегі образдың көмегімен Spark-ты Kubernetes ішінде өрістетуге мүмкіндік береді.

## Сервис қандай міндеттерге жарайды

- Үлкен деректерді үлестірілген өңдеу.
- Объектілік сақтау орнынан деректерді оқып, оларды әрі қарай өңдеу үшін БД-ға экспорттау (ClickHouse, Greenplum, PostgreSQL). Сондай-ақ деректерді БД-дан объектілік сақтау орнына беруге болады.
- Үлкен деректерді пайдаланып, ML модельдерін үлестірілген түрде оқыту.
- [GraphX](https://spark.apache.org/docs/latest/graphx-programming-guide.html) компонентін қолданып графтық есептеулерді орындау.

{/includetag}

{includetag(open)}

1. [Өтіңіз](https://kz.cloud.vk.com/app/) VK Cloud жеке кабинетіне.
1. **Data Platform** → **Сервис даналары** бөліміне өтіңіз.
1. Қажетті дананың атауын басыңыз.

{/includetag}

{includetag(login_password)}

1. Spark-қа қол жеткізу үшін әкімші логинін көрсетіңіз. Логинге қойылатын талаптар:

    - тек цифрлар, латын әріптері және `_` таңбасы рұқсат етіледі;
    - бірінші таңба — кез келген регистрдегі латын әрпі немесе `_`;
    - рұқсат етілмейтін атаулар: `os_admin`, `root`, `dataplatform_moth`.

1. Spark-қа қол жеткізу үшін әкімші құпиясөзін көрсетіңіз. Құпиясөзді орнату үшін **Генерациялау** батырмасын басыңыз немесе өз құпиясөзіңізді көрсетіңіз.

   Құпиясөзге қойылатын талаптар:

    - кемінде 16 таңба;
    - латын әліпбиінің кемінде бір бас және бір кіші әрпі;
    - кемінде бір цифр;
    - мына таңбалардың кемінде бірі: `!`, `?`, `%`, `#`, `/`, `(`, `)`, `-`, `+`, `*`.

   {note:warn}

   Құпиясөзді сақтап қойыңыз. Құпиясөзді қалпына келтіру мүмкіндігі қарастырылмаған.

   {/note}

{/includetag}

{includetag(connection)}

1. **Қосылым қосу** батырмасын басыңыз.
1. Дереккөзді таңдаңыз:

    - `S3 сыртқы` — сыртқы S3 объектілік сақтау орнына қосылу.
    - `S3 VK Cloud` — сол жобадағы VK Object Storage сақтау орнына қосылу.
    - `PostgreSQL` — PostgreSQL дерекқорына қосылу.

1. Дереккөзге қосылу параметрлерін баптаңыз:

   {tabs}

   {tab(S3 сыртқы)}

    - **Сыртқы S3 қосылымының атауы**: қосылым атауын беріңіз. Атау кіші латын әрпінен басталуы және тек бас және кіші латын әріптерін, цифрларды және `_` таңбасын қамтуы керек.
    - **Аймақ**: S3 сақтау орныңыз орналасқан аймақ.
    - **Access Key**: сақтау орнына қол жеткізу кілтінің бірегей идентификаторы.
    - **Secret Key**: сақтау орнына қол жеткізудің құпия кілті.
    - **S3 URL**: S3 сақтау орныңыздың URL мекенжайы.
    - **Бакет**: Spark данасы қосылатын S3 сақтау орнындағы бакет атауы.
    - **S3 бакетіндегі жол**: Spark данасына қолжетімді болатын бакеттегі алдын ала жасалған бумаға дейінгі жол.

   {/tab}

   {tab(S3 VK Cloud)}

    - **Ішкі S3 қосылымының атауы**: қосылым атауын беріңіз. Атау кіші латын әрпінен басталуы және тек бас және кіші латын әріптерін, цифрларды және `_` таңбасын қамтуы керек.
    - **Бакет**: Spark данасы қосылатын VK Object Storage бакетінің атауы.
    - **S3 бакетіндегі жол**: Spark данасына қолжетімді болатын бакеттегі алдын ала жасалған бумаға дейінгі жол.

   {/tab}

   {tab(PostgreSQL)}

    - **PostgreSQL**: қосылым атауын беріңіз. Атау кіші латын әрпінен басталуы және тек бас және кіші латын әріптерін, цифрларды және `_` таңбасын қамтуы керек.
    - **Дерекқор атауы**: Spark данасы қосылатын дерекқордың атауы.
    - **БД хостының атауы**: қосылуға арналған сервер адресі.
    - **Пайдаланушы аты**: дерекқор пайдаланушысының есептік жазба атауы.
    - **Пайдаланушы құпиясөзі**: дерекқор пайдаланушысының есептік жазба құпиясөзі.

   {/tab}

   {/tabs}

1. **Өзгерістерді сақтау** батырмасын басыңыз.

   Техникалық жұмыстарды, соның ішінде резервтік көшіруді жүргізу ұзақтығы — 4 сағат. Осы уақытта сервис қолжетімсіз болуы мүмкін.
