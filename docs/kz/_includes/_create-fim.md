VK Cloud куәлік федерацияларын тек [рөлі](/kz/tools-for-using-services/account/concepts/rolesandpermissions) жоба иесі болып табылатын пайдаланушылар ғана жасай алады.

1. [Өтіңіз](https://cloud.vk.com/account) VK Cloud аккаунтына.
1. **Куәлік федерациясы** бөліміне, **Федерациялар** қойындысына өтіңіз.
1. **Жасау** батырмасын басыңыз.
1. **IdP метадеректерін жүктеу** батырмасын басыңыз.

   {includetag(ad-fs)}
1. AD FS метадеректері бар XML файлын қосып, **Жүктеу** батырмасын басыңыз.
   {/includetag}

   {includetag(keycloak)}
1. Keycloak метадеректері бар XML файлын қосып, **Жүктеу** батырмасын басыңыз.
   {/includetag}

1. Федерация метадеректері бар XML файлын сақтау үшін **Метадеректерді жүктеп алу** батырмасын басыңыз. Кейін бұл файл relying party trust арқылы сенім қатынасын жасау үшін пайдаланылады.

**Федерациялар** қойындысында федерация деректері пайда болады:

- VK Cloud жеке кабинетіне федеративті пайдаланушылардың кіру URL мекенжайы, мысалы: `https://cloud.vk.com/v1/federation/saml/54f0267b-31f6-XXXX-XXX-2a24c5f436fb/signin`.
- Федерация ID-і, мысалы: `54f0267b-31f6-XXXX-XXX-2a24c5f436fb`.
