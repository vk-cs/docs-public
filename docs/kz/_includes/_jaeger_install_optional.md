1. (Қосымша) [Танысыңыз](/kz/kubernetes/k8s/how-to-guides/jaeger) с практикалық нұсқаулықпен по использованию Jaeger с микросервистік демо-қосымшамен [HotROD](https://github.com/jaegertracing/jaeger/tree/main/examples/hotrod). Нұсқаулықта мыналар көрсетілген:

    - OpenTelemetry-ді микросервистік қосымшаға интеграциялау, ол Jaeger-ге трассалау үшін қажетті сұрау деректерін жіберуі үшін.
    - Query UI көмегімен Jaeger жинаған деректерді визуализациялау және интерпретациялау.
