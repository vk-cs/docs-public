## {counter(vm)}. ВМ жасаңыз
{/includetag}

{includetag(vm_onboarding)}
1. [Өтіңіз](https://kz.cloud.vk.com/app/) VK Cloud жеке кабинетіне.
1. **Облачные вычисления → Виртуальные машины** бөліміне өтіңіз.
1. **Создать инстанс** немесе **Добавить** батырмасын басыңыз.
1. **Конфигурация** блогын ашып, ВМ конфигурациясын көрсетіңіз:

   1. **Операционная система** өрісінде `Ubuntu 22.04` таңдаңыз.
   1. Қалған параметрлердің мәндерін әдепкі қалпында қалдырыңыз. 

1. **Сеть и настройки firewall** блогын ашып, желі параметрлерін көрсетіңіз:

   1. **Сеть** өрісінде `Внешняя сеть (internet)` таңдаңыз.

      Сыртқы желіде виртуалды машинаға сыртқы IP-мекенжай автоматты түрде тағайындалады.

   1. **Ключ виртуальной машины** өрісінде `Создать новый ключ` таңдаңыз.
   1. **Добавить группу безопасности** батырмасын басып, `ssh` қосымша қауіпсіздік тобын таңдаңыз.

1. **Создать** батырмасын басыңыз. `.pem` кілт файлы компьютерге автоматты түрде жүктеледі.

   {note:warn}
   Виртуалды машинаны жасау процесі аяқталғанша бетті жаппаңыз.
   {/note}

1. Виртуалды машинаны жасау процесі аяқталғанша күтіңіз. Жаңа ВМ **Виртуальные машины** бөліміндегі тізімде пайда болады.
1. ВМ сәтті жасалғаннан кейін 10–15 минут күтіңіз. Осы уақытта жасалған ВМ жұмысқа дайындалады: оның дискісіне операциялық жүйе жайылады, ВМ-ге таңдалған баптауларды қолданатын жүйелік құралдар жұмыс істейді.
1. SSH протоколы арқылы ВМ-ге қосылыңыз:

## {counter(vm)}. ВМ-ге қосылыңыз

Ubuntu образынан жасалған Linux негізіндегі виртуалды машинаға SSH протоколы арқылы қосылуға болады.
   {/includetag}

   {tabs}

   {tab(Linux/macOS)}

   1. Жүктелген `.pem` кілт файлын тауып, оған дейінгі жолды көшіріңіз.
   1. Кілтті тек ағымдағы пайдаланушыға қолжетімді етіңіз:

      ```console
      chmod 400 <ПУТЬ_К_КЛЮЧУ>
      ```
   1. Терминалда команданы орындаңыз:

      ```shell
      ssh -i <ПУТЬ_К_КЛЮЧУ> ubuntu@<IP-АДРЕС>
      ```
      Мұнда:

      - `<ПУТЬ_К_КЛЮЧУ>` — `.pem` кілт файлына дейінгі жол.
      - `<IP-АДРЕС>` — ВМ сыртқы IP-мекенжайы. Оны VK Cloud [жеке кабинетінде](https://kz.cloud.vk.com/app/) **Облачные вычисления → Виртуальные машины** бөлімінде көруге болады.

      Мысалы:

      ```shell
      ssh -i ubuntu-key.pem ubuntu@213.219.212.130
      ```

      {include(/kz/_includes/_ssh-warning.md)[tags=new-host]}

   {/tab}

   {tab(Windows 10/11)}

   1. Жүктелген `.pem` кілт файлын тауып, оған дейінгі жолды көшіріңіз.
   1. Пәрмен жолында команданы орындаңыз:

      ```shell
      ssh -i <ПУТЬ_К_КЛЮЧУ> ubuntu@<IP-АДРЕС>
      ```
      Мұнда:

      - `<ПУТЬ_К_КЛЮЧУ>` — `.pem` кілт файлына дейінгі жол.
      - `<IP-АДРЕС>` — ВМ сыртқы IP-мекенжайы. Оны VK Cloud [жеке кабинетінде](https://kz.cloud.vk.com/app/) **Облачные вычисления → Виртуальные машины** бөлімінде көруге болады.

      Мысалы:

      ```shell
      ssh -i ubuntu-key.pem ubuntu@213.219.212.130
      ```

      {include(/kz/_includes/_ssh-warning.md)[tags=new-host]}

   {/tab}

   {tab(Windows 7/8)}

   Windows 7/8 жүйесінде SSH арқылы қосылу үшін PuTTY қолданбасын пайдаланыңыз.

   1. PuTTY-ді [орнатыңыз](https://www.putty.org/).
   1. Жинаққа кіретін PuTTYgen қолданбасын іске қосыңыз:

      1. **Conversions** мәзірінде **Import Key** тармағын таңдап, бар `.pem` кілтін көрсетіңіз.
      1. **Save private key** батырмасын басып, директорияны таңдап, `.ppk` кілтінің қалаған атауын көрсетіңіз.

   1. Жинаққа кіретін Pageant қолданбасын іске қосыңыз:

      1. Тапсырмалар панеліндегі Pageant белгішесін тінтуірдің оң жақ батырмасымен басыңыз.
      1. Контекстік мәзірден **Add key** тармағын таңдаңыз.
      1. `.ppk` форматындағы жасалған жеке кілтті таңдаңыз.

   1. PuTTY-ді іске қосыңыз:

      1. **Host Name (or IP address)** өрісіне виртуалды машинаның сыртқы IP-мекенжайын енгізіңіз. `22` портын және `SSH` қосылым түрін көрсетіңіз.

         {note:info}
         ВМ сыртқы IP-мекенжайын VK Cloud [жеке кабинетінде](https://kz.cloud.vk.com/app/) **Облачные вычисления → Виртуальные машины** бөлімінде көруге болады.
         {/note}

      1. Сол жақтағы ағашта **Connection → SSH → Auth** санатына өтіңіз.
      1. **Allow agent forwarding** жалаушасын орнатыңыз.
      1. Сол жақтағы ағашта **Auth → Credentials** тармағын ашыңыз.
      1. **Private key file for authentication** өрісінде жеке кілті бар `.ppk` файлын таңдаңыз.
      1. **Open** батырмасын басыңыз.

         Егер бұл виртуалды машинаға алғашқы қосылу болса, ашылған терезеде оны растаңыз.

      1. Ашылған терминал терезесінде `ubuntu` логинін енгізіңіз.

         Егер бәрі дұрыс бапталған болса, сервермен байланыс орнатылады.

   {/tab}

   {/tabs}
