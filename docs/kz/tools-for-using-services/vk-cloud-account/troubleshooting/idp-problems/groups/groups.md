Идентификация федерациясы арқылы кіруге әрекет жасағанда келесі хабар көрсетіледі:

```plaintext
Forbidden: role mapping misconfiguration: IdP group list is empty
```

Қате куәліктер жеткізушісі (IdP) жағында топтарға мүшелік туралы бекітімді жіберу дұрыс бапталмаған болса немесе пайдаланушы ешбір топқа кірмесе пайда болады.

## Шешім

1. Пайдаланушының IdP жүйеңіздегі кемінде бір топқа кіретініне көз жеткізіңіз.
1. Егер пайдаланушы топта болса немесе оны топқа қосу көмектеспесе, IdP жүйесінде топтарға мүшелік туралы бекітімді жіберуді тексеріп, қажет болса баптаңыз.
1. Егер мәселе сақталса, IdP жүйеңіздің техникалық қолдау қызметіне хабарласыңыз.
