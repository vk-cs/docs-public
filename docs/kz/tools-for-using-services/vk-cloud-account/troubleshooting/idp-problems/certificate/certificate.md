Идентификация федерациясы арқылы кіруге әрекет жасағанда келесі хабар көрсетіледі:

```plaintext
Forbidden: received invalid saml response: cannot validate signature on Assertion: Could not verify certificate against trusted certs
```

Қате жеткізуші куәліктері (IdP) жағында сертификат өзгерген жағдайда пайда болады. Мысалы, сертификат автоматты түрде жаңартылған кезде.  

## Шешім

Федерация үшін IdP метадеректерін қайта жүктеңіз:

1. [Өтіңіз](https://cloud.vk.com/account) VK Cloud аккаунтына.
1. **Идентификация федерациясы** бөліміне өтіп, **Федерациялар** қойындысын ашыңыз.
1. **IdP метадеректерін жүктеу** түймесін басып, куәліктер жеткізушісінің метадеректері бар XML-файлды қосыңыз.
1. **Жүктеу** түймесін басыңыз.
1. Егер мәселе сақталса немесе метадеректерді VK Cloud аккаунты арқылы жүктей алмасаңыз, куәліктер жеткізушісінің метадеректері бар XML-файлды қоса тіркеп, [техникалық қолдау қызметіне](/kz/contacts) хабарласыңыз.
