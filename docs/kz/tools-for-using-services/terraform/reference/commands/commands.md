[Terraform командалық жолы](https://www.terraform.io/cli/commands) — бұл көптеген ішкі командаларды қабылдайтын `terraform` командасы.

Негізгі қолданылатын командалар:

- `terraform init` — Terraform жұмыс каталогын инициализациялайды.
- `terraform validate` — Terraform файлы синтаксисінің дұрыстығын растайды.
- `terraform plan` — өзгерістер файлын жасайды және іске қосу кезінде не өзгеретінін көрсетеді. Нәтижелер ниетіңізге сәйкес келетініне көз жеткізу үшін бұл команданы `apply` командасын іске қоспас бұрын орындау ұсынылады.
- `terraform apply` — инфрақұрылымды құрады немесе өзгертеді. Команда орындау жоспарын көрсетеді және «иә» немесе «жоқ» деп жауап беруді сұрайды (`--auto-approve` жалауы қолданылмаса; бұл жалаумен команда автоматты түрде орындалады).
- `terraform refresh` — нақты ресурстарға қатысты жергілікті күй файлын жаңартады. Бұл Terraform-да ағымдағы ортада не бар екені туралы нақты түсінік болуын қамтамасыз етеді.
- `terraform destroy` — Terraform басқаратын инфрақұрылымды жояды. Бұл кластерден күй файлында жасалған және сақталғанның бәрінің қайтарымсыз жойылуына әкеледі.

Командаларды қолданбас бұрын `.tf` конфигурация файлдары бар жұмыс директориясына ауысыңыз.

Командаларды автоматтандыру сценарийлерінде қолданған кезде, ыңғайлы болу үшін команданы қолданар алдында жұмыс директориясын ауыстыруға арналған `-chdir=` опциясы пайдаланылады:

```console
terraform -chdir=terraform/configuration apply
```
