Қолданба инстансының ВМ-інде SSL-сертификат жаңартылғаннан кейін `ivkcs_cert` ресурсының сипаттамасында көрсетілген скрипт орындалмайды.

Мәселенің туындауының ықтимал себептері:

- Жүйе скриптті таба алмайды.
- Скрипт орындалатын пайдаланушының бұл үшін құқықтары жеткіліксіз.

## Шешімі

1. `ivkcs_cert` ресурсының `renewal_hook_path` аргументіндегі скриптке жолдың дұрыс көрсетілгенін тексеріңіз.
1. Егер `ivkcs_cert` ресурсының сипаттамасында `cert_files_owner` аргументі берілсе, онда онда көрсетілген пайдаланушыда мыналар бар екенін тексеріңіз:

    - скрипт орналасқан бумаға қол жеткізу,
    - скриптті орындау құқығы.

1. Егер тексерулер теріс нәтиже берсе, қолданбаңыздың конфигурациясына [қажетті өзгерістерді енгізіңіз](/kz/tools-for-using-services/vendor-account/how-to-guides/ssl-cert-usage#adjust_vm_image).
1. Егер мәселені өз бетіңізше шеше алмасаңыз, [техникалық қолдау қызметіне хабарласыңыз](/kz/contacts).
