# {heading(Агент)[id=ib_image_agent]}

{include(/kz/_includes/_translated_by_ai.md)}

Агент — бұл сервисті өрістету барысында ВМ-ге орнатылатын бағдарламалық қамтамасыз ету. Негізгі функциялары:

* Скрипттерді орындайды.
* Скрипттердің орындалуын бақылайды.
* Сервис инстансын өрістету кезінде скрипттердің орындалу нәтижелерін пайдалануға мүмкіндік береді.
* Скрипттердің орындалу нәтижелерін пайдаланушыға жіберуге мүмкіндік береді.
* Сервис инстансының мониторингін баптауға мүмкіндік береді.

{note:warn}

Агент кластерде өрістетілетін сервис образының құрамында міндетті түрде болуы керек.

{/note}

Бір ВМ-де өрістетілетін сервис образында агент болмауы мүмкін. Бұл жағдайда сервисте келесі мүмкіндіктер болмайды:

* Скрипттердің орындалуын бақылау.
* Скрипттердің орындалу нәтижелерін пайдаланушыға жіберу.
* Мониторинг.

Мысалы, агентсіз скрипт жасаған сервис инстансын бастапқы баптауға арналған бір реттік парольді беру мүмкін болмайды.

Агентті пайдалану iVK CS провайдері арқылы қамтамасыз етіледі (толығырақ — {linkto(../../../ibservice_add/tf_manifest/tf_manifest_settings/#tf_manifest_settings)[text=%text]} бөлімінде). Агент жұмысының логтары мен нәтижелерін қарау {linkto(../../../ibservice_add/ibservice_upload/ibservice_upload_deploysystemtest/#deploysystemtest_log)[text=%text]} бөлімінде сипатталған.
