# {heading(Қызмет образын жариялау)[id=ibservice_upload_publish_image]}

{include(/kz/_includes/_translated_by_ai.md)}

Бұлтты платформада қызмет образын жариялау үшін:

1. [marketplace@cloud.vk.com](mailto:marketplace@cloud.vk.com) мекенжайына хат жіберіңіз.

   Хатта қызмет образының ID мәнін көрсетіңіз.

   Көрсетілген образ жарияланады. Жауап хатта қызметтің жария образының ID мәні жіберіледі.

   {note:info}

   Қызметтің жария образы бұлтты платформаның барлық пайдаланушыларына қолжетімді болады.

   {/note}
1. `plans/<PLAN_NAME>/deployment/deploy.tf` манифестерінде қызмет образының ID мәнін қызметтің жария образының ID мәніне ауыстырыңыз.
1. `service.yaml` файлында қызметтің жаңа ревизиясын көрсетіңіз.
1. Қызметтің жария образының ID мәні бар сервистік пакетті дүкенге жүктеу үшін {linkto(../ibservice_upload_package#ibservice_upload_package)[text=%text]} бөлімінде келтірілген Infra API сервисіне сұрауды орындаңыз.
