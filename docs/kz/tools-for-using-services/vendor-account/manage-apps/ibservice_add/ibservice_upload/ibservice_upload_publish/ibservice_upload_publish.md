# {heading(Қызметті жариялау)[id=ibservice_upload_publish]}

{include(/kz/_includes/_translated_by_ai.md)}

Тұтынушыларға қызмет жарияланғаннан кейін ғана қолжетімді болады.

Дүкенде image-based қолданбасын жариялау үшін [marketplace@cloud.vk.com](mailto:marketplace@cloud.vk.com) мекенжайына хат жіберіңіз. Хатта қызметтің ID мәнін және оның ревизиясын көрсетіңіз.

Сервистік пакет модерациядан өтеді. Осыдан кейін қызмет дүкенде жарияланады.

Жарияланатын қызмет ревизиясы дүкеннің тестілік namespace-інен сервистік кілтте көрсетілген ашық namespace-ке ауыстырылады. Қызмет осы ашық namespace-ке қолжетімділігі бар барлық пайдаланушыларға қолжетімді болады.

{note:warn}

Ашық namespace-терде қызметтің тек соңғы жарияланған ревизиясы ғана қолжетімді.

{/note}
