# {heading(Image-based қолданбаның конфигурациялық файлдарының тізімі)[id=ibservice_files]}

{include(/kz/_includes/_translated_by_ai.md)}

Брокерге арналған сервис конфигурациясы `YAML` және `MD` форматындағы файлдардың көмегімен сипатталады. YAML-файлдар тиісті конфигурациялық файлдардың сипаттамасы бар бөлімдерде келтірілген белгілі бір құрылым мен параметрлерге ие болуы тиіс.

Келесі конфигурациялық файлдарды дайындаңыз:

1. {linkto(../ibservice_param#ibservice_param)[text=%text]}.
1. {linkto(../ibservice_full_description#ibservice_full_description)[text=%text]}.
1. `parameters/<OPTION_NAME>.yaml` тарифтік опциялар файлдары (толығырақ — {linkto(../ibopt_fill_in#IB_option_fill_in)[text=plans/<PLAN_NAME>/plan.yaml]} бөлімінде).
1. Тарифтік жоспарлардың {linkto(../ibplan#ibplan)[text=%text]} файлдары.
1. Әрбір тарифтік жоспар үшін тарифтік жоспарды конфигурациялау шеберінің {linkto(../ib_display#ib_display)[text=plans/<PLAN_NAME>/display.yaml]} файлдары.
