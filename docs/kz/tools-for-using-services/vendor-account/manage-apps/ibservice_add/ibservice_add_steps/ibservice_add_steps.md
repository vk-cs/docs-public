# {heading(Әрекеттер реті)[id=ibservice_add_steps]}

{include(/kz/_includes/_translated_by_ai.md)}

Дүкенге image-based қолданбасын қосу үшін:

1. [Сервис образын жасап, оны дүкенге жүктеңіз](../ib_image_create).

   Ұсынылатын тәсіл — Packer көмегімен.

   {note:err}

   Сервисті дүкенде жарияламас бұрын образ бұлтты платформада жарияланады (толығырақ — {linkto(../ibservice_upload/ibservice_upload_publish_image#ibservice_upload_publish_image)[text=%text]} бөлімінде). Бұлтты платформа пайдаланушыларындағы сервис инстанстары жария образ негізінде өрістетіледі. Образ деректері жалпыға қолжетімді болады.

   {/note}
1. [Сервис пакеті файлдарының құрылымын](../ib_structure) жасаңыз.
1. [Сервис конфигурациясын сипаттайтын](../ibservice_configure) файлдарды дайындаңыз (тарифтік жоспарлар, опциялар).
1. [Сервис инфрақұрылымының конфигурациясын сипаттайтын](../tf_manifest) файлдарды дайындаңыз.
1. [Сервисті жүктеп, жариялаңыз](../ibservice_upload).
