# {heading(Әрекеттер тәртібі)[id=saas_add_steps]}

SaaS-қосымшасын дүкенге қосу үшін:

1. [Әзірлеңіз](../saas_broker) сервиске арналған брокерді.
1. [Сипаттаңыз](../saas_configure) сервис конфигурациясын (тарифтік жоспарлар, опциялар).
1. [Жүктеңіз](../saas_broker_add_configuration) сервис конфигурациясын брокерге.
1. [Жүктеп, жариялаңыз](../saas_upload) сервисті дүкенде.
