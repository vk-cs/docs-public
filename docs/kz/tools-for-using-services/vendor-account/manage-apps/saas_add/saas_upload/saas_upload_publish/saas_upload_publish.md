# {heading(Сервисті жариялау)[id=saas_upload_publish]}

Сервис тұтынушыларға тек жарияланғаннан кейін ғана қолжетімді болады.

Marketplace дүкенінде SaaS-қолданбаны жариялау үшін [marketplace@cloud.vk.com](mailto:marketplace@cloud.vk.com) мекенжайына хат жіберіңіз. Хатта сервис ID-ін және оның ревизиясын көрсетіңіз.

Сервис конфигурациясына модерация жүргізіледі. Осыдан кейін сервис дүкенде жарияланады.

Жарияланатын сервис ревизиясы сервис конфигурациясындағы `metadata` параметрінде көрсетілген дүкеннің тестілік атау кеңістігінен ашық атау кеңістігіне ауыстырылады. Сервис осы ашық атау кеңістігіне қол жеткізе алатын барлық пайдаланушыларға қолжетімді болады.

{note:warn}

Ашық атау кеңістіктерінде сервистің тек соңғы жарияланған ревизиясы ғана қолжетімді.

{/note}
