# {heading(Қызметтің бұлтты платформадағы жұмыс істеуі)[id=cloud_integration]}

{include(/kz/_includes/_translated_by_ai.md)}

Қызметтің бұлтты платформада қалай жұмыс істейтінін тексеру үшін қызметті дүкенде жарияламас бұрын тестілеу ұсынылады. Бұл үшін бұлтты платформа жобасының бонустық шотына есептелетін бонустар беріледі.

Тестілеу процесінде қызмет инстансымен жасалатын негізгі операциялар мен қызметтің пайдаланушылық сценарийлері тексеріледі.

Қызметті бұлтты платформада тестілеу және қызметті дүкенде жариялау туралы толық ақпарат [Image-based қосымшасын дүкенге жүктеу](../../../ibservice_add/ibservice_upload) және [SaaS-қосымшасын дүкенге жүктеу](../../../saas_add/saas_upload) бөлімдерінде келтірілген.
