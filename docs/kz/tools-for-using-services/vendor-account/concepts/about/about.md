Шешім жеткізушілері — {var(cloud)} ішінде орналастыруға арналған дайын бағдарламалық өнімдер мен сервистерді ұсынатын компаниялар мен әзірлеушілер.

*Жеткізуші кабинеті* {var(cloud)} — бұл [қолданбалар дүкенінен](/kz/applications-and-services/marketplace/concepts/about) сервис жеткізушілеріне арналған жеке құрал, ол мыналарға мүмкіндік береді:

* жаңа сервистерді қосу;
* қолданыстағы сервистерді жаңарту;
* сервистерді мұрағатқа көшіру;
* негізгі метрикалар бойынша статистиканы қарау;
* сервистерді пайдалану туралы есептерді жүктеп алу.

Жеткізуші үшін аталған мүмкіндіктерге қолжетімділік [рөлдік модельмен](#xaas_vendor_roles) анықталады.

Жеткізуші кабинеті арқылы қосылған сервистер бұлттық платформаның жеке кабинетіндегі **Қолданбалар дүкені** бөлімінде көрсетіледі.

Басқа жеткізуші пайдаланушылардың рөлін тек әкімшілік рөл болған жағдайда ғана өзгертуге болады:
