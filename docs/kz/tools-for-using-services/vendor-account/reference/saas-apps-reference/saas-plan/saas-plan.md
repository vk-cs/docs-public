Кестеде келтірілген параметрлер [сервис конфигурациясының JSON-файлында](../../../manage-saas-apps/saas-add#service_config) `plans` секциясында көрсетіледі.

[cols="2,4,2,1,2", options="header"]
|===
|Атауы
|Сипаттамасы
|Форматы
|Міндетті
|Әдепкі мәні

|`id`
|
UUID4 генераторы арқылы қалыптастырылған тарифтік жоспардың UUID4 идентификаторы (ID)
|string (UUID4)
| ![](/kz/assets/check.svg "inline")
| ![](/kz/assets/no.svg "inline")

|`revision`
|
Тарифтік жоспардың ревизиясы. Ревизия мен тарифтік жоспар ID-інің тіркесімі оның сервис шеңберіндегі бірегейлігін анықтайды. Қалған параметрлер тарифтік жоспардың нақты ревизиясының сипаттамаларын сипаттайды.

{note:err}

Тарифтік жоспар ID-і мен ревизиясының тіркесімі сервис шеңберінде бірегей болуы керек. Егер сервис ішінде дәл сондай ID-і мен ревизиясы бар жоспар бұрыннан бар болса, тарифтік жоспар жаңартылмайды

{/note}
|string, 255 таңбаға дейін
| ![](/kz/assets/check.svg "inline")
| ![](/kz/assets/no.svg "inline")

|`name`
|
Дүкен интерфейсінде көрсетілмейтін тарифтік жоспардың техникалық атауы. Бос орындардың орнына төменгі сызықша белгісін пайдаланып, латын әріптерімен көрсетілуі керек
|string, 255 таңбаға дейін
| ![](/kz/assets/check.svg "inline")
| ![](/kz/assets/no.svg "inline")

|`description`
|
Дүкен интерфейсінде көрсетілетін тарифтік жоспар атауы
|string, 255 таңбаға дейін
| ![](/kz/assets/check.svg "inline")
| ![](/kz/assets/no.svg "inline")

|`free`
|
Тарифтік жоспардың тегін екенін не еместігін анықтайды
|boolean
| ![](/kz/assets/check.svg "inline")
| ![](/kz/assets/no.svg "inline")

|`plan_updateable`
|Пайдаланушы сервисті жоймай-ақ бір тарифтік жоспардан екіншісіне өте ала ма, соны анықтайды.

Қызметтің аттас параметрінде берілген мәнді қайта анықтайды
|boolean
| ![](/kz/assets/no.svg "inline")
| ![](/kz/assets/no.svg "inline")

|`metadata`
|Тарифтік жоспар қолжетімді болатын дүкеннің тестілік және ашық namespace-терін анықтайды.

Тестілік namespace-тер `test_ns` кілтінде беріледі.

Ашық namespace-тер `prod_ns` кілтінде беріледі.

Егер namespace-тер берілмесе, [қызмет параметрінде](../saas-param) аттас параметрде көрсетілген мәндер пайдаланылады
|map, кілттер — string
