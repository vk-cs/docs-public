Серіктестің клиенті VK Cloud жүйесіне оның поштасына келетін [шақыру](/kz/tools-for-using-services/partner-platform/instructions/invite-client) арқылы қосылады.

Шақыру хаты **Сізді VK Cloud-тағы серіктестік жобаға қосылуға шақырады** тақырыбымен келеді.

Егер шақыру **Кіріс** бумасында болмаса, **Спам** бумасын тексеріңіз.

## Шақыруды қабылдау

Кіру тәртібі сіздің поштаңызға VK Cloud аккаунты тіркелген-тіркелмегеніне байланысты.

{tabs}

{tab(Аккаунт тіркелмеген)}

1. Хаттағы **Шақыруды қабылдау** сілтемесіне өтіңіз.
1. Ашылған терезеде жаңа құпиясөзді орнатып, **Тіркелу** түймесін басыңыз.

    Құпиясөздің ең аз ұзындығы — 8 таңба. Құпиясөзде мыналар болуы керек:

    - латын әліпбиінің бас және кіші әріптері;
    - кемінде бір сан немесе `?`, `!`, `~`, `@`, `#`, `$`, `%`, `^`, `&`, `_`, `-`, `+`, `*`, `=`, `;`, `:`, `,`, `.`, `/`, `\`, `|`, `` ` ``, `[`, `]`, `{`, `}`, `(`, `)` арнайы таңбаларының бірі.

    Құпиясөзде қайталанатын таңбалар топтары болмауы керек — мысалы, `aaa123123123`.

{note:info}

Егер VK Cloud-қа кіргеннен кейін **Екі факторлы аутентификацияны қосу** беті ашылса, жеке кабинетке қол жеткізу үшін осы опцияны [баптаңыз](/kz/access/iam/instructions/manage-2fa).

{/note}

{/tab}

{tab(Аккаунт тіркелген)}

1. Хаттағы **Шақыруды қабылдау** сілтемесіне өтіңіз.

1. Жобаға шақырудың ашылған терезесінде **Қабылдау** түймесін басыңыз.

1. Авторизация терезесінде логин ретінде шақыру келген мекенжайды және ағымдағы VK Cloud құпиясөзін енгізіңіз.

{note:warn}
Егер сіз серіктестің шақыруын қолданыстағы аккаунтта қабылдасаңыз, барлық жобаларыңыз VK Cloud серіктесіне ауыстырылады. Төлем тек серіктес арқылы жүзеге асырылады.
{/note}

{/tab}

## Одан кейін не болады?

Кіргеннен кейін сіз жоба иесінің [рөлімен](/kz/tools-for-using-services/account/concepts/rolesandpermissions) сіз үшін автоматты түрде құрылған жобаға өтесіз.

Серіктес клиентінің жеке кабинетінің мүмкіндіктерімен [танысыңыз](/kz/tools-for-using-services/partner-platform/concepts/client-account).

VK Cloud сервистерінің техникалық қолдауы туралы ақпаратпен [танысыңыз](/kz/start/support).
