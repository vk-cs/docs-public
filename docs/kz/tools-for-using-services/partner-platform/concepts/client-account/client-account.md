Серіктес клиентінің жеке кабинетінде VK Cloud [пайдаланушының жеке кабинетіндегідей](/kz/tools-for-using-services/account) VK Cloud пайдаланушылық инфрақұрылымын басқару құралдары бар.
Егер сіз [White Label](/kz/start/partners/cooperation-models#model_kalay_zhumys_isteydi_fb7fed6c) серіктесінің клиенті болсаңыз, сізге VK Cloud сервистерінің шектеулі жиынтығы қолжетімді болуы мүмкін. Шектеулерді сізді платформаға шақырған White Label серіктесі белгілейді.

Серіктес клиенті жеке кабинетте [балансты](/kz/intro/billing/concepts/balance) толықтыра алмайды, ол қызметтерді тікелей серіктеске төлейді. Серіктес өз кезегінде клиенттің VK Cloud-тан алған қызметтерін өзінің серіктестік жеңілдігін шегере отырып төлейді.

Серіктес клиентіне шығыстар мен тұтынудың [егжей-тегжейі](/kz/intro/billing/instructions/detail) бар бөлім қолжетімді. Серіктес өз клиенттерінің баланс пен тұтыну егжей-тегжейін [серіктестің жеке кабинетінде](/kz/tools-for-using-services/partner-platform/instructions/reports/balance) көреді. Егер клиенттің жоба бонустық шотында виртуалды машиналарға және сервистердің басқа топтарына арналған бөлек бонустар болса, серіктес сондай-ақ бонустық шот топтарының әрқайсысы бойынша бонустардың қалдығы мен қолданылу мерзімін көреді.

Клиенттің балансы автоматты түрде толықтырылмайды, қаражат серіктестің сұрауы бойынша қолмен есепке алынады. Сервистердің құны [прайс-парақта](https://cloud.vk.com/pricelist) берілген, бірақ егер сіз [White Label](/kz/start/partners/cooperation-models#model_kalay_zhumys_isteydi_fb7fed6c) серіктесінің клиенті болсаңыз, бағаларыңыз көрсетілгеннен өзгеше болуы мүмкін.

Серіктес клиенттері тіркелу кезінде қарсы алу [бонусын](/kz/intro/billing/concepts/balance#bonuses) алады.
