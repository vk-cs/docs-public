# {heading(Қолжеткізу сеансы және пайдаланушыны бұғаттау)[id=tools-account-user-session]}

{include(/kz/_includes/_translated_by_ai.md)}

{ifdef(private,private-pg,private-pdf,private-pg-pdf)}
- Әдепкі бойынша бір мезгілде орындалатын қолжеткізу сеанстары санына шектеу жоқ. Әрбір есептік жазба үшін бір белсенді қолжеткізу сеансына шектеуді тек әкімші ғана орната алады.
{/ifdef}

{ifdef(private-cert)}
- Әрбір есептік жазба үшін бір мезгілде тек бір қолжеткізу сеансы іске қосыла алады.
{/ifdef}

- 1 сағат әрекетсіздіктен кейін қолжеткізу сеансы аяқталады. Қайтадан аутентификациядан өтіңіз.

- Есептік жазба 45 күннен астам пайдаланылмаса, бұғатталуы мүмкін.

  {ifdef(private,private-pg,private-pdf,private-pg-pdf)}
  {note:warn}
  {var(cloud)} әкімшісі әдепкі бойынша берілген пайдаланушы әрекетсіздігі кезеңін өзгерте алады.
  {/note}
  {/ifdef}

  Бұғатталған есептік жазба арқылы жеке кабинетке кірген кезде экранда «403» кіру қатесі пайда болады.

  Жоба иесінің есептік жазбасын бұғаттау жобадағы басқа пайдаланушылардың жұмысына шектеу қоймайды. Есептік жазбаны бұғаттан шығару үшін {var(cloud)} әкімшісіне хабарласыңыз.
