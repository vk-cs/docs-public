# {heading(Жоба балансы)[id=account-balance]}

{include(/kz/_includes/_translated_by_ai.md)}

{include(../../../../_includes/_balance.md)}

Бонустар болған жағдайда, қаражат алдымен бонустық шоттан, содан кейін негізгі шоттан есептен шығарылады.

Тарифтеу «pay as you go» қағидаты бойынша жүргізіледі: төлем тек тұтынылған ресурстар үшін алынады. Ақша қаражаты жұмыс істеп тұрған виртуализация нысандарынан минут сайын есептен шығарылады. Егер виртуализация нысандары тоқтатылған болса, есептен шығару тек жалға алынған диск кеңістігі мен бар резервтік көшірмелерді сақтау үшін жалғасады.

{note:warn}
Қорытынды құн **үтірден кейінгі екі таңбаға** дейін дөңгелектенеді.
{/note}
