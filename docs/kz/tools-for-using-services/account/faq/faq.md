{cut(VK Cloud сервисінде қалай тіркелуге болады?)}

[VK Cloud бетіне](https://cloud.vk.com) өтіп, **Тіркелу** түймесін басыңыз. Есептік жазбаны пайдалануды бастау үшін барлық қажетті өрістерді толтырыңыз. Толығырақ [VK Cloud сервисінде тіркелу](/kz/intro/onboarding/account) мақаласында берілген.

{/cut}

{cut(Басқа елдің телефон нөмірімен қалай тіркелуге болады?)}

Телефон нөмірін қолмен енгізу туралы өтінішпен `support@mcs.mail.ru` поштасы арқылы техникалық қолдауға жүгініңіз.

{note:warn}

VK Cloud жеке кабинеті арқылы тек `+7` нөмірінен басталатын телефон нөмірлерін ғана тіркеуге болады.

{/note}

{/cut}

{cut(Мен тіркелдім, бірақ жеке кабинеттегі сервистер әлі де қолжетімсіз)}

Сервистер [тіркелу](/kz/intro/onboarding/account) рәсімінен толық өткеннен кейін ғана қолжетімді болады: телефонды, поштаны және төлем картасын байланыстырдыңыз.

{/cut}

{cut(Жоба қатысушыларының қандай құқықтары бар?)}

Жоба қатысушыларының рөлдік моделі [Жеке кабинет пайдаланушыларының рөлдері мен құқықтары](../concepts/rolesandpermissions) мақаласында сипатталған.

{/cut}

{cut(Жобаға қосымша қатысушыны қалай қосуға болады?)}

Рәсім [Қолжетімділіктерді басқару](../instructions/project-settings/access-manage#zhobaga_zhana_katysushyny_shakyru) мақаласында сипатталған. Қосқан кезде [рөлдер матрицасына](../concepts/rolesandpermissions) сүйеніп, рөлді көрсетіңіз.

{/cut}

{cut(Есептік жазбаның құпиясөзін қалай қалпына келтіруге болады?)}

1. Жеке кабинетке кіру экранында **Құпиясөзді ұмыттыңыз ба?** түймесін басыңыз.
1. **Тіркелу кезінде көрсетілген Email** өрісіне поштаны енгізіңіз.
1. **Құпиясөзді жіберу** түймесін басыңыз. Құпиясөзді қалпына келтіру нұсқаулығы бар хат көрсетілген поштаға жіберіледі.
1. VK Cloud жеке кабинетінің құпиясөзін қалпына келтіру туралы хаттағы сілтеме бойынша өтіңіз.
1. Аттас өріске жаңа құпиясөзді енгізіңіз.
1. **Сақтау** түймесін басыңыз.

{/cut}

{cut(Есептік жазба үшін телефон нөмірін қалай өзгертуге болады?)}

[Техникалық қолдауға](/kz/contacts) жүгініп, мына деректерді көрсетіңіз:

- ағымдағы телефон нөмірі;
- жаңа телефон нөмірі.

Нөмір пайдаланушының жеке басы сәтті расталғаннан кейін өзгертіледі.

{/cut}

{cut(VK Cloud есептік жазбасының иесін қалай ауыстыруға болады?)}

[Жобаны басқару](../instructions/project-settings/manage#zhoba_iesin_auystyru) мақаласындағы нұсқаулықты пайдаланыңыз.

{/cut}

{cut(Жоба иесін басқа заңды тұлғаға қалай өзгертуге болады (шартты көшіру)?)}

[Техникалық қолдауға жүгініңіз](/kz/contacts), мыналарды көрсетіп:

- [Жобаның PID идентификаторы](/kz/tools-for-using-services/account/instructions/project-settings/manage#zhoba_identifikatoryn_alu);
- ағымдағы және жаңа иесінің ЖСН/БСН.
{/cut}

{cut(GeekBrains бағдарламасының қатысушысы VK Cloud платформасында квоталарды ұлғайтуды сұрай ала ма?)}

Иә, сұрай алады. Ол үшін тағайындалған GeekBrains кураторына жүгініңіз.

{/cut}

{cut(Егер бос ресурстар жеткіліксіз болса, VK Cloud жобасының квоталарын қалай ұлғайтуға болады?)}

[Жоба квоталарын ұлғайту жөніндегі нұсқаулықты](/kz/tools-for-using-services/account/instructions/project-settings/manage#increase-quota) пайдаланыңыз.

{/cut}

{cut(Жоба менеджері кім және онымен қалай байланысуға болады?)}

[Техникалық қолдауға](/kz/contacts) жүгініп, ұйымыңыздың ЖСН/БСН көрсетіңіз.

{/cut}

{/cut}
