# {heading(Тарификация)[id=tools-account-tariffication]}

{include(/kz/_includes/_translated_by_ai.md)}

Жеке кабинетті пайдалану тарификацияланбайды.

Сервистердің бағалары [прайс-парақта](https://cloud.vk.com/pricelist) және [жоба баптауларында](https://kz.cloud.vk.com/app/project/prices) көрсетілген. Таңдалған сервистер конфигурациясының құнын есептеу үшін [калькуляторды](https://cloud.vk.com/pricing/) пайдаланыңыз. Платформа сервистерін қалай төлеу керектігі туралы [Биллинг](/kz/intro/billing) бөлімінен оқыңыз.
