# {heading(Қатысушыны жобаға шақыру)[id=tools-account-project-invite]}

Қатысушыны шақырып, оған қолжетімділікті реттей алатын пайдаланушылар: иесі, суперадминистратор және пайдаланушылар әкімшісі (IAM). Жоба қатысушыларын көру жоба әкімшісі мен бақылаушыға да қолжетімді.
