Сервис жұмысы кезінде пайда болған оқиғалар мен қателер туралы ақпарат VK Cloud логтарында сақталады. Бұл деректер туындайтын инциденттерді талдау кезінде техникалық қолдау қызметімен пайдаланылады.

{note:info}

Егер іске қосылған виртуалды машинаның жұмысында ақаулар туындаса, техникалық қолдауға [ВМ хабарламаларының логтарын](/kz/computing/iaas/instructions/vm/vm-console#vm_habarlamalarynyn_logtary) ұсыныңыз.

{/note}

Логтарды жергілікті түрде жүктеп алу үшін:

{tabs}

{tab(Жеке кабинет)}

1. VK Cloud жеке кабинетіне [өтіңіз](https://kz.cloud.vk.com/app/).
1. Бет жоғарғы жолағындағы пайдаланушы атына басыңыз.
1. Ашылмалы тізімнен **Отладка приложения** тармағын таңдаңыз.

   {note:info}

   JSON форматындағы әрбір оқиға туралы толық ақпарат оны басқан кезде қолжетімді.

   {/note}

1. Көрсетілетін жазбаларды баптаңыз:

   - **Тип бекенда**: ашылмалы тізімнен сервисті таңдаңыз. Қажетті сервис тізімде болмаса, **Неизвестный бэкенд** опциясын таңдаңыз.
   - **Показывать только 4хх и 5хх**: тек `4XX` немесе `5XX` түріндегі қателермен аяқталған оқиғаларды көрсету қажет болса, осы опцияны қосыңыз.

1. **Скачать log** түймесін басыңыз. `dpp-log-<дата>.json` түріндегі файл жүктеледі.

{/tab}
