Шақыру хаты **Сізді VK Cloud жүйесіндегі жобаға қосылуға шақырады** тақырыбымен келеді. Хатта сіз шақырылған [рөлдер](../../concepts/rolesandpermissions) мен жоба идентификаторы көрсетіледі. Егер сіз бірнеше [жобаға](../../concepts/projects) шақырылсаңыз, бірнеше хат келеді.

Егер шақыру **Кіріс жәшігінде** болмаса, **Спам** бумасын тексеріңіз.

## Шақыруды қабылдау

Кіру рәсімі сіздің мекенжайыңызбен VK Cloud жүйесінде аккаунт тіркелген-тіркелмегеніне байланысты.

{tabs}

{tab(Аккаунт тіркелмеген)}

1. Хаттағы **Шақыруды қабылдау** сілтемесіне өтіңіз.

1. Ашылған терезеде жаңа құпиясөзді орнатып, **Тіркелу** түймесін басыңыз.

    Құпиясөздің ең аз ұзындығы — 8 таңба. Құпиясөзде мыналар болуы керек:

    - латын әліпбиінің бас және кіші әріптері;
    - кемінде бір сан немесе `?`, `!`, `~`, `@`, `#`, `$`, `%`, `^`, `&`, `_`, `-`, `+`, `*`, `=`, `;`, `:`, `,`, `.`, `/`, `\`, `|`, `` ` ``, `[`, `]`, `{`, `}`, `(`, `)` арнайы таңбаларының бірі.

    Құпиясөзде қайталанатын таңбалар топтары болмауы керек — мысалы, `aaa123123123`.

{note:info}

Егер VK Cloud жүйесіне кіргеннен кейін **Екі факторлы аутентификацияны қосу** беті ашылса, жеке кабинетке қол жеткізу үшін осы опцияны [баптаңыз](/kz/tools-for-using-services/vk-cloud-account/instructions/account-manage/manage-2fa).

{/note}

{/tab}

{tab(Аккаунт тіркелген)}

1. Хаттағы **Шақыруды қабылдау** сілтемесіне өтіңіз.

1. Жобаға шақырудың ашылған терезесінде **Қабылдау** түймесін басыңыз.

1. Авторизация терезесінде логин ретінде шақыру келген мекенжайды және VK Cloud құпиясөзін енгізіңіз.

1. Егер жобада міндетті екі факторлы аутентификация қосылған болса, сізге кодты пайдаланып авторизациядан өту немесе VK CLoud профиліңізде тиісті опцияны [баптау](/kz/tools-for-using-services/vk-cloud-account/instructions/account-manage/manage-2fa) ұсынылады.

1. Жеке кабинетте сіз шақырылған жобаға өтіңіз:

    1. Бет жоғарғы жолағында ағымдағы жобаның атауын басыңыз.

    2. Тізімнен қажетті жобаны таңдаңыз.

{/tab}

## Әрі қарай не істеу керек?

- Жеке кабинеттің мүмкіндіктерімен [танысыңыз](/kz/tools-for-using-services/account).
- VK Cloud жүйесіндегі профиліңізді [баптаңыз](/kz/tools-for-using-services/vk-cloud-account/instructions/account-manage/editinfo).
