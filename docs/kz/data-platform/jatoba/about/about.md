# {heading(Сервис туралы)[id=jatoba_info]}

{include(/kz/_includes/_translated_by_ai.md)}

Jatoba — PostgreSQL ашық кодты ДҚБЖ-ның жаңғыртылған өзегіне негізделген қорғалған ДҚБЖ. Оны шағын ұйымдарда да, ірі компанияларда да пайдалануға болады. Jatoba туралы толығырақ [ресми құжаттамада](https://jatoba.ru/).

Сервис мүмкіндіктері:

- ДҚБЖ-ның географиялық үлестірілген архитектурасы бар автоматты ақауға төзімді және жоғары өнімді кластер;
- Ыңғайлы графикалық интерфейсі бар кеңейтілген әкімшілендіру функциялары, сондай-ақ "vanilla" PostgreSQL басқаруы;
- Дашбордтары бар дамыған мониторинг жүйесі;
- Дерекқорлар мен кестелерді ағып кетулер кезінде рұқсатсыз оқудан қорғау, SQL Firewall және киберқауіпсіздіктің озық функциялары.
