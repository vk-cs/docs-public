# {heading(Cloud Redis)[id=redis_metrics]}

{include(/kz/_includes/_translated_by_ai.md)}

Cloud Redis сервисінде метрикалар Prometheus форматында жиналады. Әрбір метрика үшін дашбордта көрсетуге болатын панель автоматты түрде жасалады. Метрикалар тізімі {linkto(#redis_tab_metrics)[text=%number кестеде]} келтірілген.

{caption({counter(table)[id=numb_redis_tab_metrics]} кесте — Redis метрикалары)[align=right;position=above;id=redis_tab_metrics;number={const(numb_redis_tab_metrics)}]}
[cols="1,1,1", options="header"]
|===
|Метрика атауы
|Сипаттамасы
|Дашборд панеліндегі деректер

|`% CPU`
|CPU жүктемесі пайызбен
|Таңдалған уақыт аралығындағы белгіленген шекке қатысты CPU жүктемесі пайызбен

|`% Memory`
|Жадты пайдалану пайызбен
|Таңдалған уақыт аралығындағы белгіленген шекке қатысты жадты пайдалану пайызбен

|`Read IOPS`
|Дискіден оқу операцияларының саны
|Таңдалған уақыт аралығындағы дискіден оқу операцияларының жалпы саны

|`Write IOPS`
|Дискке жазу операцияларының саны
|Таңдалған уақыт аралығындағы дискке жазу операцияларының жалпы саны

|`Read MB`
|Дискіден оқылған деректер көлемі, МБ
|Таңдалған уақыт аралығында дискіден оқылған деректер көлемі, МБ

|`Write, Mbytes`
|Дискке жазылған деректер көлемі, МБ
|Таңдалған уақыт аралығында дискке жазылған деректер көлемі, МБ

|`Network received, Mbytes`
|Кіріс желілік трафик көлемі, МБ
|Таңдалған уақыт аралығындағы кіріс желілік трафик көлемі, МБ

|`Network sent, Mbytes`
|Шығыс желілік трафик көлемі, МБ
|Таңдалған уақыт аралығындағы шығыс желілік трафик көлемі, МБ

|`Connected clients`
|Қосылған клиенттер саны
|Белсенді клиенттік қосылымдардың ағымдағы саны

|`Total commands per second`
|Секундына командалар саны
|Таңдалған уақыт аралығындағы секундына орындалған командалардың жалпы саны

|`Blocked clients`
|Бұғатталған клиенттер саны
|Бұғатталған клиенттік қосылымдардың ағымдағы саны

|`Memory fragmentation ratio`
|Жад фрагментациясының дәрежесі
|Жад фрагментациясы коэффициентінің ағымдағы мәні

|`Keyspace hits per second`
|Секундына кэшке түсу санының жалпы саны
|Таңдалған уақыт аралығындағы секунд сайын деректерге сұраулардың кэшке түсу саны

|`Keyspace misses per second`
|Секундына кэштегі өткізіп алулар саны
|Таңдалған уақыт аралығындағы секунд сайын деректерге сұраулардың кэшті өткізіп алуы саны

|`Hit ratio`
|Кэшке түсу үлесі
|Таңдалған уақыт аралығындағы деректерге сұраулардың кэшке түсу үлесі

|`Expired keys (per sec)`
|Секундына мерзімі өткен кілттер саны
|Таңдалған уақыт аралығындағы секундына мерзімі өткен кілттер саны

|`Evicted keys per sec`
|Секундына жад жетіспеуіне байланысты жойылған кілттер саны
|Таңдалған уақыт аралығындағы секундына жад жетіспеуіне байланысты жойылған кілттер саны

|`Connected slaves`
|Қосылған репликалар саны
|Қосылған репликалардың ағымдағы саны

|`Replication offsets for master`
|Мастер-тораптағы репликация ығысуы
|Мастер-тораптағы репликация ығысуының ағымдағы мәні, байтпен

|`Replication offsets for slave`
|Реплика тораптарындағы репликация ығысуы
|Реплика торабындағы репликация ығысуының ағымдағы мәні, байтпен

|===
