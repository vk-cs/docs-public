# {heading(Private Cloud құрамындағы Data Platform мониторингі)[id=monitoring-private_cloud]}

{include(/kz/_includes/_translated_by_ai.md)}

Private Cloud құрамында жайылған {var(data-p)} сервистерінің бір бөлігі Cloud Monitoring мониторинг жүйесімен біріктірілген. Жиналған метрикалардың мониторингі {var(data-p)} сервистерінің күйін бақылауға, ақаулар мен сервистер жұмысындағы іркілістерді дер кезінде анықтауға мүмкіндік береді. Cloud Monitoring жұмысы туралы толығырақ **VK Private Cloud әкімшісі нұсқаулығы** құжатындағы **Компоненттер** → [Cloud Monitoring](https://cloud.vk.com/docs/on-premises/private-cloud/kz/4_3/admin-guide/components/cloud_monitoring) бөлімінен қараңыз.

Cloud Monitoring-пен біріктірілген сервистер тізімі:

- Cloud Kafka;
- Cloud Trino;
- Redis.
