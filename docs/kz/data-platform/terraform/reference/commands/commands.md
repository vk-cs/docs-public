# {heading(Негізгі Terraform командалары)[id=terraform_commands]}

{include(/kz/_includes/_translated_by_ai.md)}

[Terraform командалық жолы](https://www.terraform.io/cli/commands) — бұл көптеген ішкі командаларды қабылдайтын `terraform` командасы.

Негізгі пайдаланылатын командалар:

- `terraform init` — Terraform жұмыс каталогын инициализациялайды.
- `terraform validate` — Terraform файлы синтаксисінің дұрыстығын растайды.
- `terraform plan` — өзгерістер файлын жасайды және іске қосу кезінде не өзгеретінін көрсетеді. Нәтижелер ниетіңізге сәйкес келетініне көз жеткізу үшін бұл команданы `apply` командасын іске қоспас бұрын орындау ұсынылады.
- `terraform apply` — инфрақұрылымды құрады немесе өзгертеді. Команда орындау жоспарын көрсетеді және «иә» немесе «жоқ» жауабын талап етеді (егер `--auto-approve` жалауы пайдаланылмаса; бұл жалаумен команда автоматты түрде орындалады).
- `terraform refresh` — нақты ресурстарға қатысты жергілікті күй файлын жаңартады. Бұл Terraform-ның ағымдағы ортада не бар екенін дәл түсінуіне кепілдік береді.
- `terraform destroy` — Terraform басқаратын инфрақұрылымды жояды. Бұл кластерден күй файлында жасалған және сақталған барлық нәрсенің қайтарымсыз жойылуына әкеледі.

Командаларды пайдаланбас бұрын `.tf` конфигурация файлдары бар жұмыс директориясына өтіңіз.

Автоматтандыру скрипттерінде командаларды пайдалану жағдайында ыңғайлылық үшін команданы қолданар алдында жұмыс директориясын ауыстыруға арналған `-chdir=` опциясы қолданылады:

```console
terraform -chdir=terraform/configuration apply
```
