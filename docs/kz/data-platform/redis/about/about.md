# {heading(Қызмет туралы)[id=redis_info]}

{include(/kz/_includes/_translated_by_ai.md)}

Redis — ашық бастапқы коды бар, деректерді жедел жадта кілт/мән жұптары түрінде сақтайтын жоғары тиімді және жоғары жылдамдықты NoSQL түріндегі СУБД. Redis туралы толығырақ — [ресми құжаттамада](https://redis.io/).

Redis сұрауларды өңдеу кезінде микросекундтық кідірісті қамтамасыз етеді, секундына миллиондаған операцияны өңдей алады және жылдамдық басым болатын міндеттерге жарайды: деректерді кэштеу, пайдаланушы сессияларын сақтау, чаттарды, кезектерді, хабарлама брокерлерін, нақты уақыттағы аналитиканы және басқаларын іске асыру.

Қызметті қолдану салалары:

- веб-қосымшалар;
- нақты уақыт жүйелері;
- геймификация;
- аналитикалық жүйелер;
- микросервистік архитектура;
- таратылған қосымшалар.
