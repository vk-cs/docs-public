# {heading(ClickHouse Data Lakehouse архитектурасында)[id=dlh_clickhouse]}

{include(/kz/_includes/_translated_by_ai.md)}

{linkto(../../../../clickhouse/concepts/about#clickhouse_info)[text=Cloud ClickHouse]} — бұл құрылымдалған үлкен деректер бойынша аналитикалық SQL-сұрауларды нақты уақыт режимінде жылдам өңдеуге арналған ашық бастапқы коды бар бағандық реляциялық СУБД. Деректерді S3 қоймасында сақтаусыз кіріс оқиғалар ағынында нақты уақыттағы жедел аналитиканы орындау қажет болған жағдайда DLH архитектурасына қосымша түрде енгізіледі.

Компонент келесі функцияларды орындайды:

- Негізгі DLH қоймасына сұрау жасамай-ақ BI-аналитиканы орындау үшін деректерді блоктық-бағандық форматта ұзақ мерзімді сақтау.
- Жеделдетілген толық сканерлеумен BI-аналитикаға арналған күрделі SQL оқу сұрауларын өңдеу.
- Бірыңғай кластерлік СУБД шеңберінде [OLTP](https://ru.wikipedia.org/wiki/OLTP)- және OLAP-сценарийлерін қолдау.
- Деректерді шардтауды және 100 серверге дейін көлденең масштабтауды қолдау.
- Деректер көздерінің реляциялық СУБД-ларынан деректерді тікелей импорттау және ETL мен ELT процестерін ұйымдастыру мүмкіндігі.
- Көздерден келетін кіріс оқиғалар ағынында жедел аналитиканы орындау үшін үздіксіз мониторинг.
- Архитектурада ClickHouse болуы DLH құнын арттырады, бірақ сонымен бірге DLH компоненттерінің толық жиынтығын өрістетпей-ақ нақты уақыттағы бірқатар бизнес-сценарийлерді іске қосуға мүмкіндік береді, бұл ірі Enterprise-инсталляциялар үшін шешімнің ресурстық сыйымдылығын және T2M көрсеткішін едәуір қысқартады.
