# {heading(Сервис туралы)[id=postgresql_info]}

{include(/kz/_includes/_translated_by_ai.md)}

Cloud PostgreSQL — деректерді сақтау және басқару үшін пайдаланылатын объектілік-реляциялық СУБД. Сервис PostgreSQL негізінде әзірленген. PostgreSQL туралы толығырақ [ресми құжаттамада](https://www.postgresql.org/docs/).

Cloud PostgreSQL үлкен көлемдегі деректерді сақтау үшін қолайлы. СУБД деректердің көптеген түрлерін қолдайды және өз түрлерін жасау мүмкіндігін береді.

Cloud PostgreSQL-дің негізгі артықшылықтары:

- ANSI-SQL:2008 стандартына жақын деректер тұтастығын сақтау;
- күрделі таңдауларды жасау мүмкіндігі: көріністер, ережелер, ішкі сұраулар;
- көпнұсқалылық (MVCC) арқылы параллель қолжетімділікті басқару.
