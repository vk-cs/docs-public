      HDFS қоймасы бар Hive Metastore кластеріне қосылу үшін Kerberos есептік деректері қолданылады. Жаңа клиент принципалын (principal) жасау және оған арналған Keytab-файлды алу үшін Hive Metastore әкімшісіне жүгініңіз. Бұл файлды base64 форматына қайта кодтау қажет.

      {tabs}

      ```bash
      base64 -i <ПУТЬ_ДО_KEYTAB-ФАЙЛА>
      ```

      {tab(Windows)}

      ```ps
      [Convert]::ToBase64String([IO.File]::ReadAllBytes("<ПУТЬ_ДО_KEYTAB-ФАЙЛА>"))
      ```
