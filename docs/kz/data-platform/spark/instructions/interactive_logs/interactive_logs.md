# {heading(Интерактивті логтарды қарау)[id=kafka_iceberg]}

{include(/kz/_includes/_translated_by_ai.md)}

Cloud Spark сервисінде Spark тапсырмаларының орындалуы туралы толық ақпараты бар интерактивті логтарды мәтіндік немесе JSON форматында қарау қолжетімді.

{note:warn}

Логтар тек аяқталған Spark тапсырмалары үшін, соның ішінде қателікпен аяқталған тапсырмалар үшін, егер олардың кодында `job.delete` тапсырманы жою функциясы қолданылмаса ғана көрсетіледі.

{/note}

Интерактивті логтарды қарау үшін:

1. **Data Platform** → **Қызмет даналары** бөліміне өтіңіз.
1. Cloud Spark сервисі данасының атауын басыңыз.
1. **Жалпы ақпарат** қойындысында **Spark Service** тармағындағы URL мекенжайын көшіріп алыңыз.
1. Көшірілген URL мекенжайына өтіңіз. Аяқталған Spark тапсырмаларының оқиғаларын визуализациялауға арналған веб-интерфейс — History Server ашылады.
1. Қызықтыратын тапсырманы `App Name` мәні бойынша тауып, оның App ID мәнін басыңыз.
1. **Executors** қойындысына өтіңіз.
1. Executors кестесіндегі `logs` бағанының оң жағынан **log** түймесін басыңыз.

    Spark тапсырмасының орындалу интерактивті логтары бар бет ашылады, ол мәтіндік немесе JSON форматында болады (параметрлерге байланысты).

    Егер логтар жоқ болып, `failed to get pod logs` хабарламасы көрсетілсе, Spark тапсырмасының кодында `job.delete` функциясы қолданылмайтынына көз жеткізіңіз. Егер функция бар болса, оны алып тастап, тапсырманы қайта іске қосыңыз.
