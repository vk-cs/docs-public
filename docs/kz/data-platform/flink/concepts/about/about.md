Cloud Flink — деректерді көздерден мақсатты жүйелерге кез келген көлемде ағындық өңдеуге және тасымалдауға арналған сервис. Сервис [Apache Flink](https://flink.apache.org/) пен Kubernetes негізінде әзірленген.

Apache Flink сервисінің жұмыс істеу қағидаты үлестірілген ағындық есептеулер тұжырымдамасына негізделген: деректер параллель және үлестірілген түрде өңделетін үздіксіз ағын ретінде қарастырылады. Бұл тәсіл жоғары өнімділік пен ауқымдылыққа қол жеткізуге мүмкіндік береді.

Cloud Flink сервисі автоматтандыратын процестер:

- оқиғаларды ағындық өңдеу;
- үлкен деректер жиынтықтарын өңдеу;
- деректерді агрегаттау және талдау;
- деректерді әртүрлі жүйелерге тасымалдау (мысалы, деректерді Kafka-дан дерекқорға, дерекқордан деректер қоймасына тасымалдау);
- ETL-міндеттерін өңдеу: деректерді шығару (Extract), түрлендіру (Transform) және жүктеу (Load) процестері.

Сервисті пайдаланудың ықтимал сценарийлері:

- Қаржылық аналитика: транзакцияларды қадағалау, алаяқтық әрекеттерді анықтау.
- Веб-сайттарды мониторингтеу: пайдаланушылар белсенділігін нақты уақыт режимінде талдау.
- Құрылғылардан алынған деректерді талдау: әртүрлі датчиктер мен құрылғылардан алынған бастапқы деректерді өңдеу.
- Машиналық оқыту: машиналық оқыту модельдерін жаттықтыру және бағалау.
- Big Data аналитикасы: трендтерді, болжамдарды және т.б. анықтау үшін үлкен деректерді өңдеу.

Пайдаланушы Cloud Flink ресурстарын VK Cloud жеке кабинеті арқылы басқара алады, Flink кластерлерімен жұмыс істеу [Flink CLI](https://docs.kafka-ui.provectus.io/overview/readme) арқылы қолжетімді.

Cloud Flink сервисімен жұмысты бастау үшін қосылуға [өтінім қалдырыңыз](https://cloud.vk.com/flink/).
