# {heading(Сервис туралы)[id=opensearch_info]}

{include(/kz/_includes/_translated_by_ai.md)}

OpenSearch — іздеу жүйелері мен талдау шешімдерін құруға арналған бағдарламалық кешен. Сервис OpenSearch іздеу платформасын және деректерді визуализациялау мен талдауға арналған OpenSearch Dashboards веб-интерфейсін қамтиды. Толығырақ [ресми құжаттамадан](https://docs.opensearch.org/docs/latest/) қараңыз.

Сервисті қолдану салалары:

- іздеу жүйелері;
- логтар талдауы;
- қолданбалар мониторингі;
- пайдаланушы әрекетін талдау;
- бизнес-талдау шешімдерін құру.

OpenSearch данасын тек **Кластер** конфигурациясында {linkto(../instructions/create#spark_create)[text=құруға]} болады (бірнеше түйіннен тұрады). Бұл шешім үлкен көлемдегі деректерді өңдеуге жарайды және жоғары өнімділік пен істен шығуға төзімділікті қамтамасыз етеді: бір түйіннің істен шығуы дана жұмысының тоқтауына әкелмейді.
