VK Data Platform UI Console (DP Console) — VK Data Platform платформасымен жұмысты оңтайландыруға арналған сервис.

Сервис VK Data Platform ішіндегі дата-сервистерді орталықтандырылған түрде басқару және топтастыру үшін қосымша пайдаланушы интерфейсін ұсынады.

Сервис мүмкіндіктері:

- Қажетті дата-сервис данасын немесе сервистер тобын іздеу.
- Дата-сервистерді логикалық топтарға біріктіру. Бұл ретте топ әртүрлі кластерлерде жайылған дата-сервистерді қамтуы мүмкін.
- Дата-сервистер топтары бойынша навигация, сервистердің жекелеген даналары арасындағы желілік байланыстылықты қарау.
- Сервистер тобы деңгейінде деректер мамандарының жекелеген командаларына құқықтар тағайындау. Деректер инженерлерінің әрбір командасы дата-сервистердің бекітілген жиынтығымен жұмыс істеуге қол жеткізеді және деректер пайплайнының нақты бір бөлігіне жауап береді.
- Топтармен орындалған әрекеттер, соның ішінде топтарға құқық беру немесе қайтарып алу деңгейіндегі әрекеттер тіркелетін оқиғалар журналын қарау.
