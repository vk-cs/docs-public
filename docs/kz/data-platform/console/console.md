VK Cloud инфрақұрылымындағы деректер платформасымен жұмысты жеңілдетуге арналған сервис. VK Cloud жеке кабинетінің интерфейсі арқылы деректер платформасының дата-сервистерін орталықтандырылған түрде басқаруды және топтастыруды қамтамасыз етеді. Сервис VK Data Platform сервисінің барлық пайдаланушылары үшін тегін.
