# {heading(Сервис туралы)[id=nifi_info]}

{include(/kz/_includes/_translated_by_ai.md)}

NiFi — деректер ағындарын автоматтандыруға және басқаруға арналған платформа. Сервис Apache NiFi негізінде әзірленген. Толығырақ [ресми құжаттамада](https://nifi.apache.org).

Сервис әртүрлі жүйелер арасында ақпаратты жинау, өңдеу және беру процестерін автоматтандыруға арналған және нақты уақыт режимінде мониторингтеу мен басқару мүмкіндігі бар күрделі деректер конвейерлерін құруға мүмкіндік береді.

Сервистің мүмкіндіктері:

- Көптеген көздерден деректерді жинау.
- Деректерді түрлендіру және тазалау.
- Деректерді жүйелер арасында тасымалдау.
- Деректерді өңдеуге арналған бас тартуға төзімді шешімдерді құру.
- Деректер ағындарын визуализациялау және мониторингтеу.
