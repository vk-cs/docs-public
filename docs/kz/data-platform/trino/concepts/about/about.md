# {heading(Қызмет туралы)[id=trino_info]}

{include(/kz/_includes/_translated_by_ai.md)}

Cloud Trino [Trino](https://trino.io/) SQL-қозғалтқышының алдын ала бапталған данасын әртүрлі дереккөздерден алынған үлкен көлемдегі деректерді параллель өңдеу үшін пайдалануға мүмкіндік береді. 

Cloud Trino деректерді өзінде сақтамайды, бірақ оларға сұраныстарды үйлестіреді және оңтайландырады. Сұраныстар сервис түйіндері арасында динамикалық түрде бөлінеді және параллель орындалады. Бұл ретте бір сұраныс олардың орналасқан жері мен деректерді сақтау пішіміне қарамастан бірнеше дереккөзге жіберілуі мүмкін. Алынған деректер бірыңғай аналитиканы алу үшін жинақталып, өңделеді.

Cloud Trino сервисі автоматтандыратын процестер:

- бір SQL-сұраныста көптеген әртүрлі дереккөздерден деректерді шығарып алу;
- үлкен көлемдегі деректерді өңдеу;
- деректерді түрлендіру және агрегаттау; 
- ETL-міндеттерін өңдеу: деректерді шығарып алу (Extract), түрлендіру (Transform) және жүктеу (Load) процестері;
- сұраныстардың орындалуын оңтайландыру, соның ішінде параллель орындау үшін сұранысты ұсақ ішкі сұраныстарға бөлу.

Сервисті пайдаланудың ықтимал сценарийлері:

- Деректерді талдау: нақты уақыт режимінде ақпарат алу үшін ағындық деректерге аналитикалық сұраныстарды орындау.
- Аналитикалық панельдерді жасау (dashboarding): пайдаланушыларға әртүрлі дереккөздерден алынған өзекті ақпаратқа қол жеткізуді қамтамасыз ететін интерактивті аналитикалық панельдерді жасауға арналған деректерді алу.
- Есептерді әзірлеу: SQL-сұраныстар көмегімен көптеген дереккөздерден есептер жасау.
- Машиналық оқыту: машиналық оқыту модельдері үшін деректерді дайындау.

Trino туралы толығырақ [ресми құжаттамада](https://trino.io/docs/current/overview/concepts.html).
