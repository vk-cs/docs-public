## Инстансқа қосылу

ADB инстансына қосылу үшін [дерекқор инстансына қосылу жөніндегі нұсқаулықты](../../../dbaas/connect) пайдаланыңыз.

## Деректерді жүктеу

Дерекқорға қосылу үшін коннектор таңдалып, байланыс орнатылғаннан кейін, деректерді жүктеуге кірісуге болады.

Greenplum жүйесіне деректерді жүктеу және Greenplum жүйесінен деректерді шығару үшін негізгі құрал gpfdist утилитасы болып табылады. Бұл — Greenplum бірден барлық түйіндері арқылы өзара әрекеттесетін http-сервер, соның арқасында деректерді жүктеу де, шығару да жоғары жылдамдықпен орындалады.

gpfdist бойынша ресми құжаттама [ресми ресурста қолжетімді](https://docs.vmware.com/en/VMware-Greenplum/7/greenplum-database/utility_guide-ref-gpfdist.html).

gpfdist не үшін қолданылады? Greenplum жүйесінде external table (сыртқы кестелер) ұғымы бар, олардың сипаттамасы кәдімгі кестелердің сипаттамасына өте ұқсас, бірақ соңында LOCATION (‘gpfdist://hostname:8080/database_name/table_name.csv’) сияқты шарт қосылады, ол external table ішінде сипатталған құрылымы бар файлдың нақты қай жерде сақталатынын көрсетеді. External table не оқу үшін, не жазу үшін болуы мүмкін.

Егер external table оқу үшін жасалса, онда, біріншіден, файл атауының орнына бүркеншік үлгіні пайдалануға болады (яғни бірден көп файлды оқуға немесе жүктеуге болады), ал екіншіден, архивтелген файлдардан (.gz, .zip немесе .bz2) оқуға болады, бұл файлдарды оқу және жүктеу жылдамдығын едәуір арттырады, себебі дискілерден деректерді оқу көлемі архивтелмеген файлдарды оқумен салыстырғанда бірнеше есе азаяды.

External table әртүрлі түйіндерде іске қосылған бірнеше gpfdist данасымен бірден жұмыс істей алады. Бұл жүйенің өнімділігін одан әрі арттырады — деректерді жүктеу үшін де, шығару үшін де.
