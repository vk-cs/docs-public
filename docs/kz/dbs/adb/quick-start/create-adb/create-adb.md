Дерекқорды жасау үшін [жеке кабинеттегі "Аналитикалық БД" бөліміне](https://kz.cloud.vk.com/app/services/analytic-databases/) өтіп, «Arenadata DB» лицензия нұсқасын таңдап, «БД жасау» түймесін басыңыз.

Келесі қадамда қажетті кластер түрін, ArenadataDB инстансының нұсқасын және инстанс конфигурациясын таңдаңыз.

«Кластер» конфигурациясы — бұл деректердің синхронды репликациясы бар кластер. Ол жүйенің сенімділігі мен істен шығуға төзімділігіне қойылатын жоғары талаптар болған кезде қолданылады.

Келесі қадамдарда конфигурацияның қажетті параметрлерін таңдаңыз:

| Параметр атауы | Сипаттама |
|----------------------------|---------------------------------------------|
| Кластер атауы | Әдепкі атауды қалдырыңыз немесе латын әріптерімен өз атауыңызды енгізіңіз |
| Сығылмаған таза деректер саны | БД-ның ең үлкен көлемі |
| Түйіндер конфигурациясы | Мастердің CPU / RAM / HDD (SSD) конфигурациясы.<br>Реплика/slave дәл осындай конфигурациямен автоматты түрде жасалады |
| Резервтеуді пайдалану | Істен шығуға төзімділікті қамтамасыз ету үшін артық қайталанатын түйіндерді жасау |
| monitoring-түйінді қосу | Кластердің жұмысқа қабілеттілік метрикаларын визуализациялау үшін осы опцияны белгілеңіз |
| Дискілерді автоауқымдау | Дискілер өлшемі асқан кезде олардың көлемі кластердің барлық ВМ-дарында автоматты түрде ұлғаюы үшін осы опцияны белгілеңіз. Ұлғайтылатын дискінің ең үлкен мөлшері, ұлғайту қадамы, сондай-ақ бос емес кеңістіктің шегі автоматты түрде есептеледі |
| Диск түрі                  | Жақсырақ өнімділік үшін SSD немесе Hi-IOPS SSD диск түрін таңдауды ұсынамыз |
| Желі                       | Әдепкі бойынша қалдырыңыз немесе өзіңіздің жекеменшік желіңізді таңдаңыз |
| Сыртқы IP тағайындау | Кластерге сыртқы желілерден қосылу мүмкіндігін алу үшін оған сыртқы IP тағайындаңыз |
| Қолжетімділік аймағы           | Қолжетімділік аймағын көрсетіңіз |

Келесі қадамда дерекқорды инициализациялау параметрлерін орнатыңыз:

| Параметр атауы | Сипаттама |
|----------------------------|---------------------------------------------|
| Жасалатын дерекқор атауы | Әдепкі мәнді қалдырыңыз немесе өз атауыңызды енгізіңіз |
| Пайдаланушы аты | Қашықтан қол жеткізу үшін пайдаланушы атын көрсетіңіз |
| Пайдаланушы құпиясөзі | Қашықтан қол жеткізу үшін пайдаланушы құпиясөзін орнатыңыз |

Инстанс бірнеше минут ішінде жасалады. Осыдан кейін инстанс туралы ақпарат және қосылу тәсілдері пайда болады.
