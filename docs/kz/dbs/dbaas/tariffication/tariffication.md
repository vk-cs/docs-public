# {heading(Тарификация)[id=dbaas-tariffication]}

{include(/kz/_includes/_translated_by_ai.md)}

Тарификация «pay as you go» қағидаты бойынша жүргізіледі: төлем тек тұтынылған ресурстар үшін минуттық дәлдікпен
алынады.

Қызметтердің бағасы [прайс-парақта](https://cloud.vk.com/pricelist) келтірілген. Қызметтердің жалпы бағасын есептеу
үшін [калькуляторды](https://cloud.vk.com/pricing) пайдалануға болады. Платформа қызметтерінің ақысын төлеуге байланысты
құралдардың қалай жұмыс істейтіні туралы [Биллинг](../../../intro/billing) бөлімінен оқыңыз.

# {heading(Тарификацияланады)[id=dbaas-tariffication-yes]}

- CPU (vCPU) — әр ядро үшін. 1 vCPU виртуализация серверінің 1 физикалық ядросына сәйкес келеді.
- Жедел жад (RAM) — жедел жадтың әрбір 1 ГБ-ы үшін.
- Дискілер — дискілік кеңістіктің әрбір 1 ГБ-ы үшін, бағасы диск түріне байланысты (SSD немесе High-IOPS SSD).
- Диск снапшоты (резервтік көшірме) — снапшот көлемінің әрбір 1 ГБ-ы үшін.
- IP мекенжайлары мен жүктеме теңгергіштері, толығырақ {linkto(
  ../../../networks/vnet/tariffication#vnet-tariffication)[text=Виртуалды желілер]} бөлімінде.
- Жүйелік дискілер.
- ДҚБЖ түріне байланысты: транзакциялар журналына арналған диск.

# {heading(Тарификацияланбайды)[id=dbaas-tariffication-no]}

- Резервтік көшірмелерді автоматты түрде жасау.
- Кіріс және шығыс трафик.
- Мониторинг.
