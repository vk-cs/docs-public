Cloud Databases — бұлттық дерекқорлар (Database as a service). Сервис бұлтта MySQL, PostgreSQL, MongoDB, Redis,
ClickHouse, Tarantool негізіндегі шешімдерді өрістетуге мүмкіндік береді. Дерекқорлар Single, Master-Replica және
Кластер конфигурацияларында қолжетімді. Сервисті пайдалану дерекқорларды әкімшілендіру жүктемесін сізден алып тастайды.
