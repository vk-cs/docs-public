# {heading(Жылдам бастау)[id=dbaas-quick-start]}

{include(/kz/_includes/_translated_by_ai.md)}

Жылдам бастау сізге PostgreSQL мысалында сервисті пайдалануды бастауға және оның мүмкіндіктерімен танысуға көмектеседі.

Жылдам бастау қадамдарының барлығын орындағаннан кейін, сіз:

1. Бір хосттан тұратын PostgreSQL БД инстансын жасайсыз.
1. Инстансқа [TimescaleDB](https://docs.timescale.com) кеңейтімін орнатасыз.
1. SQL-сұрауларды {linkto(#db_connect)[text=орындау]} үшін инстансқа қосылуды үйреніңіз.
1. Кеңейтімнің жұмыс істейтінін тексеру үшін TimescaleDB үшін тестілік деректер мен сұраулар жасайсыз.
1. БД инстансының жұмыс істеуі барысында жиналған мониторинг деректерімен танысасыз.

Тестілік деректердің мысалы ретінде автоматты түрде жасалатын, заттар интернеті (IoT) сенсорларынан алынған ақпаратты:
температура мен CPU жүктемесін білдіретін деректер жиыны пайдаланылады. Тестілік деректер жиыны туралы
толығырақ [Timescale құжаттамасынан](https://docs.timescale.com/tutorials/latest/simulate-iot-sensor-data/) оқыңыз.
Осындай деректер жиындарын жасау
процедурасы [Timescale блогында](https://www.timescale.com/blog/how-to-create-lots-of-sample-time-series-data-with-postgresql-generate_series/)
егжей-тегжейлі қарастырылады.

{note:warn}

БД инстансы тарификацияланады және есептеу ресурстарын тұтынады. Жылдам бастаудан өткеннен кейін, егер олар енді қажет
болмаса, [инстансты және оған тағайындалған Floating IP мекенжайын жойыңыз]().

{/note}
