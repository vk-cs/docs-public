# {heading(Қызмет туралы)[id=dbaas-about]}

{include(/kz/_includes/_translated_by_ai.md)}

Cloud Databases қызметі бұлтта сізге қажет {linkto(../types#dbaas-types)[text=түрлердегі]} ДҚБЖ-ны әртүрлі {linkto(../work-configs#dbaas-work-configs)[text=конфигурацияларда]} жылдам өрістетуге мүмкіндік береді.

Бұлттағы ДҚБЖ-ларды басқару әкімшілендіру бойынша терең білімді талап етпейді. ДҚ инстансы жеке кабинетте бірнеше шерту арқылы жасалады. VK Cloud инстансты өрістету және оны техникалық сүйемелдеу бойынша барлық жұмысты өз мойнына алады, соның ішінде конфигурация мен таңдалған опцияларға байланысты келесілер бапталады:

- хосттар арасындағы деректер репликациясы;
- жүктемені теңгеру;
- шардтау;
- резервтік көшіру;
- деректер көлемі ұлғайған сайын дисктерді масштабтау.

{linkto(../../../../tools-for-using-services/account/concepts/regions#tools-account-concepts-regions)[text=Москва аймағында]} 16-нұсқадағы {linkto(../types/postgresql#dbaas-concepts-postgresql)[text=PostgreSQL]} үшін әртүрлі [қолжетімділік аймақтарында](../../../../start/concepts/architecture#az) хосттары бар географиялық үлестірілген конфигурацияларды жасап, істен шығуға төзімділікті қосымша арттыра аласыз.

ДҚ инстансын өрістеткеннен кейін онда қажетті дерекқорларды жасау, оларды мазмұнмен толтыру, оларға өз қолданбаларыңызды қосу және қолжетімділіктерді баптау ғана қалады.

PostgreSQL үшін VK Cloud-тың кірістірілген {linkto(../../monitoring/postgresql#dbaas-monitoring-postgresql)[text=мониторингі]} автоматты түрде қосылады.

Кейбір ДҚБЖ түрлері үшін {linkto(../extensions#dbaas-concepts-extensions)[text=кеңейтімдер]} қолжетімді, оларды жеке кабинет арқылы орнатуға болады.
