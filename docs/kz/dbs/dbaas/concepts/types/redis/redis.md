Redis (Remote Dictionary Server) — деректерді жедел жадта кілт/мән жұптары түрінде сақтайтын, бастапқы коды ашық, NoSQL түріндегі жоғары тиімді СУБД. Redis-тің басты артықшылығы — өнімділік бойынша, тіпті ұқсас жүйелердің арасында да, оған тең келетіндер жоқ.

Redis жылдамдық басым болатын міндеттер үшін оңтайлы: деректерді кэштеу, пайдаланушы сессияларын сақтау, чаттарды, кезектерді, хабарламалар брокерлерін, нақты уақыттағы аналитиканы және басқаларын іске асыру үшін.

Толығырақ [Redis ресми құжаттамасында](https://redis.io/docs/).

VK Cloud-та қолдау көрсетілетін Redis нұсқаларының тізімі осы деректер қоры инстансын [құру](../../../instructions/create) кезінде қолжетімді.

## Әрі қарай не істеу керек?

- Деректер қоры инстансын [құрыңыз](../../../instructions/create).
- Деректер қорына [қосылыңыз](../../../connect).
- Қолжетімді деректер қоры кеңейтімдерімен [танысыңыз](../../extensions).
