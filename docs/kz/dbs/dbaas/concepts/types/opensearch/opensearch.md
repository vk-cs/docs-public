# {heading(OpenSearch)[id=dbaas-opensearch]}

{include(/kz/_includes/_translated_by_ai.md)}

[OpenSearch](https://opensearch.org/) — бұл Apache 2.0 лицензиясымен таратылатын, деректерді алуды, іздеуді, визуализациялауды және талдауды жеңілдететін бастапқы коды ашық бағдарламалық кешен. OpenSearch өз жұмысында [Apache Lucene](https://lucene.apache.org/) қозғалтқышын пайдаланады. OpenSearch-та деректермен жұмыс Elasticsearch-тағыдай ұйымдастырылған: деректер JSON-құжаттар түрінде сақталады, оларға метадеректер қосылады. Құжаттар мен метадеректер, өз кезегінде, индекстерде сақталады.

OpenSearch деректер бойынша толықмәтінді іздеу қажет болғанда тиімді. Мысалы, оны «Wikipedia-дағыдай» іздеу функционалдығы қажет болатын интернет-дүкендер сияқты қолданбалар үшін бэкенд ретінде пайдалануға болады. Тағы бір танымал қолданылу саласы — логтар аналитикасы.

Толығырақ [OpenSearch ресми құжаттамасында](https://opensearch.org/docs/latest/).

VK Cloud-та қолдау көрсетілетін OpenSearch нұсқаларының тізімі осы деректер қоры инстансын {linkto(../../../instructions/create#dbaas-create)[text=құру]} кезінде қолжетімді.
