## {heading(Профильдер)[id=gpu-vgpu-profiles]}

Стандартты сипаттамалардан (CPU, RAM және т.б.) бөлек, {linkto(../../../../computing/gpu/concepts/about#gpu-about-vgpu-flavors)[text=vGPU бар ВМ конфигурациялары қалыптарының]} атауында *vGPU профилі* де көрсетіледі.

vGPU профилі екі бөліктен тұрады:

* Профиль түрі — GPU қандай міндеттер үшін пайдаланылатынын анықтайды. Латын әліпбиінің әрпі түрінде беріледі. Cloud GPU-де Q сериясындағы әмбебап профильдер қолданылады. Олар GPU пайдаланылатын кез келген міндеттерге жарайды, мысалы, қолданбалар интерфейстерін және күрделі 3D жобаларды рендерлеу үшін немесе CUDA негізіндегі күрделі есептеулер (AI, ML, HPC) үшін.
* Бейнежад көлемі (VRAM) — профиль шеңберінде қолжетімді болатын ГБ санын білдіреді.

Мысалы, 2 ГБ VRAM бар vGPU профилі `2Q` деп аталады.

## {heading(Лицензиялау)[id=gpu-vgpu-licensing]}

vGPU жұмысы үшін NVIDIA® vGPU Software лицензиясы қажет. Cloud GPU Q сериясындағы профильдер үшін арнайы жасалған NVIDIA® vGPU Virtual Workstation лицензиясын пайдаланады.

VK Cloud-та лицензияны тексеру жергілікті NVIDIA® License Server арқылы орындалады. Пайдаланылатын ОС образына байланысты қосымша баптау қажет болуы мүмкін:

- VK Cloud образы: қосымша әрекеттер қажет емес, лицензиялау сервері әлдеқашан бапталған.
- Жеке образ: жүйені vGPU-мен жұмыс істеу үшін {linkto(../../../../computing/gpu/how-to-guides/vgpu-setup#vgpu-setup)[text=баптау]} қажет.

Тексеру сәтсіз болған жағдайда, ВМ сынақ режимінде 20 минутқа дейін жұмыс істейді.
