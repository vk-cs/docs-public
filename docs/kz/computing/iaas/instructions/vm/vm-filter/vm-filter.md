Қажетті виртуалды машиналарды жылдам табу үшін сүзгілерді пайдаланыңыз:

1. [Өтіңіз](https://kz.cloud.vk.com/app/) VK Cloud жеке кабинетіне.
1. **Бұлтты есептеулер → Виртуалды машиналар** бөліміне өтіңіз.
1. **Инстанстар бойынша іздеу** өрісін басыңыз. Қолжетімді сүзгілер тізімі ашылады:

    - **Виртуалды машиналардың түрлері** — қажетті түрін таңдаңыз: виртуалды машиналар, Kubernetes инстанстары немесе ML Platform.
    - **Мәртебелері** — тізімде тек белсенді инстанстарды қалдыру үшін **Инстанс іске қосылған** опциясын таңдаңыз.
    - **Бэкаптар** — ВМ автоматты резервтік көшіру жоспарына енгізілгенін тексеру үшін опцияны пайдаланыңыз:

        - **Иә** — тізімде тек резервтік көшірмелерді автоматты түрде жасау жоспарларына енгізілген ВМ қалады.
        - **Жоқ** — тізімде ешбір автоматты резервтік көшіру жоспарына енгізілмеген ВМ ғана қалады.

        {note:info}
        ВМ резервтік көшірмелерін қолмен жасау сүзгілеуге әсер етпейді.
        {/note}

    - **Тегтер** — қажетті тегтерді таңдаңыз. Тізімде таңдалған барлық тегтері бар ВМ ғана қалады.
    - ВМ атауы бойынша сүзгілеу — іздеу өрісін толтыра бастаңыз. Тізімде атауы енгізілген таңбалардан басталатын ВМ ғана қалады.

   Әртүрлі сүзгілерді ВМ-ді дәлірек іздеу үшін біріктіріп қолдануға болады.
