Қажетті виртуалды машиналарды жылдам табу үшін сүзгілерді пайдаланыңыз:

1. [Өтіңіз](https://kz.cloud.vk.com/app/) VK Cloud жеке кабинетіне.
1. **Облачные вычисления → Виртуальные машины** бөліміне өтіңіз.
1. **Поиск по инстансам** өрісін басыңыз. Қолжетімді сүзгілер тізімі ашылады:

    - **Типы виртуальных машин** — қажетті түрін таңдаңыз: виртуалды машиналар, Kubernetes инстанстары немесе ML Platform.
    - **Статусы** — тізімде тек белсенді инстанстарды қалдыру үшін **Инстанс запущен** опциясын таңдаңыз.
    - **Бэкапы** — ВМ автоматты резервтік көшіру жоспарына енгізілгенін тексеру үшін опцияны пайдаланыңыз:

        - **Да** — тізімде тек резервтік көшірмелерді автоматты түрде жасау жоспарларына енгізілген ВМ қалады.
        - **Нет** — тізімде ешбір автоматты резервтік көшіру жоспарына енгізілмеген ВМ ғана қалады.

        {note:info}
        ВМ резервтік көшірмелерін қолмен жасау сүзгілеуге әсер етпейді.
        {/note}

    - **Теги** — қажетті тегтерді таңдаңыз. Тізімде таңдалған барлық тегтері бар ВМ ғана қалады.
    - ВМ атауы бойынша сүзгілеу — іздеу өрісін толтыра бастаңыз. Тізімде атауы енгізілген таңбалардан басталатын ВМ ғана қалады.

   Әртүрлі сүзгілерді ВМ-ді дәлірек іздеу үшін біріктіріп қолдануға болады.
