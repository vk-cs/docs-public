# {heading(ВМ жасау кезіндегі кідірістер)[id=iaas-vm-creation-time]}

{include(/kz/_includes/_translated_by_ai.md)}

ВМ әдеттегіден ұзағырақ жасалады.

Мәселе әртүрлі себептерден туындауы мүмкін.

### {heading(Шешім)[id=iaas-vm-creation-time-decision]}

1. {var(cloud)} сервистері тұрақты жұмыс істеп тұрғанына [көз жеткізіңіз](/kz/tools-for-using-services/account/how-to-guides/systemstate).
1. ВМ-ді басқа қолжетімділік аймағында [жасап көріңіз](/kz/computing/iaas/instructions/vm/vm-create).
1. Егер мәселе шешілмесе, [техникалық қолдау қызметіне хабарласыңыз](/kz/contacts) және ВМ жасалған нақты уақыт аралығын және жоба идентификаторын хабарлаңыз.
