# {heading(Диск read-only күйіне өтеді)[id=iaas-disk-read-only]}

{include(/kz/_includes/_translated_by_ai.md)}

ВМ дискісі `read-only` күйіне өтеді.

Бұл ОС-тың деректердің жоғалуынан немесе бүлінуінен қорғайтын механизмі. Мәселе әртүрлі себептерден туындауы мүмкін.

### {heading(Шешім)[id=iaas-disk-read-only-decision]}

1. Жүйелік логтарда енгізу-шығару қателері (I/O error) бар-жоғын тексеріңіз. Қателерді іздеу үшін `dmesg` немесе `journalctl` командаларын пайдаланыңыз.
1. Қайта жүктеу кезінде `fsck` утилитасымен тексеру жүргізіп, файлдық жүйенің тұтастығын тексеріңіз.
1. [Техникалық қолдауға хабарласыңыз](/kz/contacts) және өтінімге қателері бар логтарды қоса тіркеңіз.

Егер мәселе аппараттық ақаудан туындаса, қолжетімділік аймағын ауыстыру көмектесуі мүмкін.
