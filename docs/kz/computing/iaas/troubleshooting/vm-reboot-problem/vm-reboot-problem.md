Виртуальная машина зависла в процессе перезагрузки.

### Решение

[Выполните](/kz/computing/iaas/instructions/vm/vm-manage#iaas-vm-manage-restart) принудительную перезагрузку ВМ.

Если проблема сохраняется в течение 10–15 минут, [обратитесь в техническую поддержку](https://cloud.vk.com/contacts/) для проверки состояния хоста.
