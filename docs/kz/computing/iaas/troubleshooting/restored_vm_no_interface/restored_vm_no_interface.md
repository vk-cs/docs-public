Резервтік көшірмеден қалпына келтірілген ВМ-де интерфейстер бапталмаған.

### Шешім

Резервтік көшірмеден қалпына келтіргеннен кейін ВМ операциялық жүйесіндегі интерфейс баптауларын [тексеріп, қажет болса түзетіңіз](/kz/computing/iaas/troubleshooting/linux-vm-network#1_zhelilik_interfeys_baptaularyn_tekseriniz). ВМ-нің ішкі және сыртқы мекенжайларының баптаулары [жеке кабинеттегі](https://kz.cloud.vk.com/app/) баптауларға сәйкес келуі керек.

{note:info}

Сыртқы IP болуы тиіс барлық ВМ-ді қосу үшін [Floating IP мекенжайларын](/kz/networks/vnet/concepts/ips-and-inet#floating-ip) пайдаланыңыз. Егер ВМ жойылса немесе оны қайта құру қажет болса, Floating IP мекенжайы жобада қалады және оны қалпына келтірілген ВМ-ге тағайындауға болады.

{/note}
