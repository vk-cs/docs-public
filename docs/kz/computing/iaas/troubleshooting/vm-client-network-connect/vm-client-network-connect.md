# {heading(ВМ сыртқы желіні көрмейді)[id=iaas-vm-client-network-connect]}

{include(/kz/_includes/_translated_by_ai.md)}

Виртуалды машинаның сыртқы желіге қосылу мәселелері {var(cloud)} виртуалды желісі мен сыртқы желі арасында желілік байланыстылықтың (VPN) болмауына байланысты болуы мүмкін.

### {heading(Шешімі)[id=iaas-vm-client-network-connect-decision]}

{var(cloud)} виртуалды желісі мен сыртқы желі арасында VPN-туннель жасаңыз. 

Бұлт пен шағын қашықтағы желі арасында VPN-туннель ұйымдастыру үшін [нұсқаулықты](/kz/networks/vnet/how-to-guides/onpremise-connect/vpn-tunnel) пайдаланыңыз. Немесе қашықтағы инфрақұрылымға қосылудың басқа [нұсқаларын](/kz/networks/vnet/concepts/onpremise-connect) қараңыз. 

Қосылу нұсқасын таңдау жоба SDN-іне, қашықтағы инфрақұрылымнан интернетке қолжеткізуге және қосылымның істен шығуға төзімділігіне қойылатын талаптарға байланысты.
