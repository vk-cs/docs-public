# {heading(Cannot continue with installation қатесі)[id=iaas-error-cannot-continue-with-installation]}

{include(/kz/_includes/_translated_by_ai.md)}

ВМ жасау әрекеті кезінде `Cannot continue with installation, no valid storage devices detected` хабарламасы пайда болады.

Қате орнату бейнесінде VirtIO диск контроллеріне арналған драйверлердің жоқтығына байланысты. 

### {heading(Шешім)[id=iaas-error-cannot-continue-with-installation-decision]}

Орнату бейнесіне VirtIO драйверлерін қосыңыз.
