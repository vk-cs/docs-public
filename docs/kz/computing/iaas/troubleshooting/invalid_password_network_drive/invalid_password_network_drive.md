Желілік дискіні қосу кезінде `Недействительный пароль` қатесі дұрыс емес есептік деректерді немесе қол жеткізу баптауларын білдіруі мүмкін.

### Шешім

Файлдық қоймаға арналған қол жеткізу ережелерін тексеріңіз:

1. **Бұлттық есептеулер** → **Файлдық қойма** бөліміне өтіңіз.
1. Қажетті файлдық қойманың атауын басыңыз.
1. **Қол жеткізу ережелері** қойындысына өтіңіз.

CIFS және SMB протоколдары үшін пайдаланушыға немесе IP мекенжайына қол жеткізуге рұқсат беріліп, дұрыс құпиясөз орнатылуы керек.

Егер мәселе сақталса, [техникалық қолдау қызметіне хабарласыңыз](/kz/contacts).
