Жеке кабинетте ВМ әкімшісінің құпиясөзін өзгерткен кезде **Құпиясөзді орнату мүмкін болмады. Техникалық қолдау қызметіне хабарласыңыз** қатесі пайда болады.

Бұл ВМ құпиясөзі орнатылатын QEMU қонақ агентінің қолжетімсіздігіне байланысты болуы мүмкін.

### Шешімі

1. Виртуалды машина консоліне қосылып, авторизациядан өтіңіз.

1. Келесі команданы орындап, QEMU қонақ агентін жаңартыңыз:

   ```console
   sudo sh -c "apt install -y qemu-guest-agent;systemctl enable qemu-guest-agent;systemctl start qemu-guest-agent"
   ```

1. ВМ қайта іске қосып, әкімші құпиясөзін тағы бір рет өзгертіп көріңіз. Егер қате сақталса, [техникалық қолдау қызметіне хабарласыңыз](/kz/contacts).
