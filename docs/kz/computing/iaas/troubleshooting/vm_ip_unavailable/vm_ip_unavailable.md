# {heading(Қызмет IP мекенжайы бойынша қолжетімсіз)[id=iaas-vm-ip-unavailable]}

{include(/kz/_includes/_translated_by_ai.md)}

Виртуалды машинада орналастырылған қызметке IP мекенжайы бойынша қолжетімділік жоқ.

Қате әртүрлі себептерден туындауы мүмкін.

### {heading(Шешім)[id=iaas-vm-ip-unavailable-decision]}

1. **Бұлттық есептеулер → Виртуалды машиналар** бөліміндегі [жеке кабинетте {var(cloud)}](https://kz.cloud.vk.com/app/) ВМ күйін тексеріңіз. ВМ күйі `Белсенді` екеніне көз жеткізіңіз.
1. Жоба балансын [тексеріңіз](/kz/intro/billing/instructions/payment#billing-payment-balance-view). Теріс баланс кезінде қызметтердің жұмысы тоқтатылады.
1. Қауіпсіздік топтарын [тексеріңіз](/kz/networks/vnet/instructions/secgroups#vnet-secgroups-view). Қажетті портқа кіріс трафигіне рұқсат беретін ереже бар екеніне көз жеткізіңіз.
1. ВМ ішіндегі желілік қызметтерді тексеріңіз. Қолданбаның іске қосылғанына және қажетті портты тыңдап тұрғанына көз жеткізіңіз.
1. Порталға корпоративтік қолжетімділікті тексеріңіз. Корпоративтік желіден бұлтқа қосылулар бұғатталмағанына көз жеткізіңіз.
