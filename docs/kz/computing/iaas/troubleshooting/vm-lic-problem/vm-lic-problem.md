# {heading(RDS лицензияларымен мәселе)[id=iaas-vm-lic-problem]}

{include(/kz/_includes/_translated_by_ai.md)}

ВМ-ге [RDS](/kz/computing/vm-licenses/ms-lic#vm-licenses-microsoft-rds) арқылы қосылған кезде лицензиялауға қатысты мәселе және сеанстың 60 минуттан кейін тоқтатылатыны туралы ақпараттық терезе пайда болады.

Мәселе топтық саясаттарда клиенттік қолжеткізуге арналған [лицензия түрі](/kz/computing/vm-licenses/ms-lic#vm-licenses-microsoft-rds) қате орнатылған болса туындауы мүмкін.

### {heading(Шешімі)[id=iaas-vm-lic-problem-decision]}

Топтық саясаттарда клиенттік қолжеткізу лицензиясының түрін өзгертіңіз:

{include(/kz/_includes/_vm-lic-manage.md)}

1. RDS серверін қайта жүктеңіз.
1. Егер мәселе сақталса, [техникалық қолдау қызметіне хабарласыңыз](/kz/contacts).
