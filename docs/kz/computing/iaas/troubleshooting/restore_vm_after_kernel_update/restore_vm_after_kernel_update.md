# {heading(Ядро жаңартылғаннан кейін ВМ жүктелмейді)[id=iaas-restore-vm-after-kernel-update]}

{include(/kz/_includes/_translated_by_ai.md)}

ОЖ-ның жаңа ядросын орнатқаннан кейін виртуалды машина жүктелмей қалды.

Мәселе әртүрлі себептерден туындауы мүмкін: үйлесімсіздік, конфигурация қателері, қажетті модульдердің болмауы немесе ядроның жаңа нұсқасындағы қателер.

### {heading(Шешім)[id=iaas-restore-vm-after-kernel-update-decision]}

Жүйені жүктеу кезінде ядроның алдыңғы нұсқасын таңдаңыз. Егер мұны істеу мүмкін болмаса, [техникалық қолдауға хабарласыңыз](/kz/contacts).
