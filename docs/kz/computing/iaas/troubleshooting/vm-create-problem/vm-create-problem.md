# {heading(ВМ жасалмайды)[id=iaas-vm-create-problem]}

{include(/kz/_includes/_translated_by_ai.md)}

Жеке кабинетте виртуалды машинаны жасау процесінде қате пайда болады.

Қате әртүрлі себептерден туындауы мүмкін.

### {heading(Шешімі)[id=iaas-vm-create-problem-decision]}

ВМ жасау процесінде {var(cloud)} панелінің жоғарғы оң жақ бұрышындағы қалқымалы терезеге назар аударыңыз. Онда қате туралы хабарлама көрсетіледі. Қатенің сипаттамасын талдап, оны өз бетіңізше түзетіп көріңіз. Қажет болса, құжаттаманы пайдаланыңыз.

Егер қатені түзету мүмкін болмаса немесе жасау шебері қате туралы хабарлағанымен, хабарлама пайда болмаса, [техникалық қолдау қызметіне](/kz/contacts) хабарласыңыз.
