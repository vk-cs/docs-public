# {heading(ВМ образын жүктеу мүмкін емес)[id=iaas-image-loading-problem]}

{include(/kz/_includes/_translated_by_ai.md)}

Виртуалды машина образын жүктеу мәселелері келесі себептерге байланысты туындауы мүмкін:

- жүктелетін файлдың көлемі үлкен;
- файл пішімі {var(cloud)} жүйесімен үйлеспейді;
- образдардың жалпы көлеміне арналған лимит таусылған.

### {heading(Шешім)[id=iaas-image-loading-problem-decision]}

1. Жүктелетін файл пішімі RAW екеніне көз жеткізіңіз. Егер образыңыз басқа пішімде болса, [оны түрлендіріңіз](../../how-to-guides/packer#iaas-packer-convert-image).
1. Жүктелетін файл көлемі 500 ГБ-тан аспайтынына көз жеткізіңіз. Егер одан үлкен образды жүктеу қажет болса, [нұсқаулықты](/kz/storage/s3/how-to-guides/load-large-image) пайдаланыңыз.
1. [Образды](/kz/computing/iaas/instructions/images/images-manage#iaas-images-manage-import) OpenStack CLI көмегімен жүктеп көріңіз.
1. Егер мәселе сақталса, [техникалық қолдауға хабарласыңыз](/kz/contacts). 

{note:info}
Жүктелген образдардың жалпы көлеміне арналған лимит 2 ТБ құрайды. Лимитті арттыру үшін [техникалық қолдауға хабарласыңыз](/kz/contacts).
{/note}
