# {heading(Жылдам бастау)[id=iaas-quick-start]}

{include(/kz/_includes/_translated_by_ai.md)}

Cloud Servers көмегімен {var(cloud)} жеке кабинетінде Linux виртуалды машинасы (ВМ) жасалып, оған қосылу орындалады.

Жылдам бастау сервисті пайдалануды бастауға және оның мүмкіндіктерімен танысуға көмектеседі.

Жылдам бастаудың барлық қадамдарынан өткеннен кейін, сіз:

1. Жаңа виртуалды машина (ВМ) жасайсыз.
1. Жасалған ВМ-ге SSH арқылы қосыласыз.

{note:info}
Жұмыс істеп тұрған ВМ есептеу ресурстарын тұтынады.

Жылдам бастаудан өткеннен кейін, егер ВМ енді қажет болмаса, оны тоқтатыңыз немесе жойыңыз.
{/note}

## {heading(Дайындық қадамдары)[id=iaas-quick-start-preparatory-steps]}

1. [{var(cloud)} платформасында тіркеліңіз](/kz/intro/onboarding/account).
1. Жеке шот теңгерімі оң екеніне, ал [квоталардың](/kz/tools-for-using-services/account/concepts/quotasandlimits) виртуалды машинаның мақсатты конфигурациясын жасауға жеткілікті екеніне көз жеткізіңіз.
