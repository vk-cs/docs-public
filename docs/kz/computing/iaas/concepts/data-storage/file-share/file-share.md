Файлдық сақтау орны — бірнеше виртуалды машинаға қосуға және деректерге ортақ қолжеткізу үшін пайдалануға болатын виртуалды файлдық жүйе. [Файлдық сақтау орнын жасау](../../../instructions/fs-manage/fs-create) кезінде оған желілік қолжеткізу протоколы (NFS немесе CIFS) көрсетіледі.

{note:info}

Файлдық сақтау орындарына қолжеткізу тек VK Cloud жобасының ішіндегі виртуалды машиналардан жүзеге асырылады.

{/note}

Файлдық сақтау орнының өлшемі жасау кезеңінде беріледі. Қажет болса, файлдық сақтау орнының өлшемін ұлғайтуға болады, бірақ оны кішірейтуге болмайды.

Файлдық сақтау орындары үшін снимок жасау және снимоктан қалпына келтіру операциялары қолжетімді.
