## {heading(Виртуалды машиналар санаттары)[id=vm_categories]}

Жеке кабинетте ВМ қалыптары санаттар бойынша топтастырылған:

[cols="1,2,2", options="header"]
|===
| Санат
| Сипаттама
| Көрсетілу шарты

| Intel Cascade Lake (Intel Xeon Gen 2)
| Intel Cascade Lake CPU бар серверлердегі ВМ
| Әдепкі бойынша қолжетімді

| Intel Ice Lake (Intel Xeon Gen 3)
| Intel Ice Lake CPU бар серверлердегі ВМ

| vGPU бар виртуалды машиналар

| Сәйкес ВМ түрлеріне [Cloud GPU](https://cloud.vk.com/cloud-gpu/) сервисі арқылы тапсырыс берілгенде көрсетіледі

| Бөлінген серверлердегі ВМ

| Сәйкес ВМ түрлеріне менеджеріңіз арқылы тапсырыс берілгенде көрсетіледі
|===

Әдепкі бойынша тек өзекті конфигурация қалыптары көрсетіледі: Intel Cascade Lake және Intel Ice Lake CPU бар серверлердегі ВМ, сондай-ақ графикалық картасы қосылған, жоғары өнімді немесе бөлінген серверлердегі ВМ, егер олар жобаға қосылған болса.

Ресурс көлемі 16 vCPU және 64 ГБ RAM-нан асатын ВМ қоса алғанда, ВМ үшін жеке конфигурацияларды тізімге қосу үшін [техникалық қолдауға](/kz/contacts) хабарласыңыз.

ВМ-нің ескі конфигурация қалыптары **Архивтік ВМ түрлері** санатын таңдағанда қолжетімді болады. Ескі конфигурациялы ВМ жасауға арналған сервер кездейсоқ таңдалады: Intel Cascade Lake немесе Intel Ice Lake, бірақ [тарифтеледі](../../../tariffication) Intel Ice Lake бағасы бойынша. Сервер қолжетімсіз болған жағдайда (мысалы, профилактикалық жұмыстар кезінде) бұл ВМ басқа буындағы серверге көшірілуі мүмкін.

{cut(Архивтік ВМ түрлері)}

| Атауы | Конфигурация параметрлері | Сипаттама |
| --- | --- | --- |
| Basic | 2 vCPU дейін <br/> 4 ГБ RAM дейін | Өнімділігі төмен ВМ жасауға арналған базалық конфигурациялар |
| Standard | 2-ден 4 vCPU дейін <br/> 4 ГБ-тан 16 ГБ RAM дейін | CPU саны мен RAM көлемі ұлғайтылған конфигурациялар |
| Advanced | 8-ден 16 vCPU дейін <br/> 16 ГБ-тан 64 ГБ RAM дейін | Жоғары өнімді ВМ жасауға арналған конфигурациялар |
| Heavy | 16 vCPU бастап <br/> 64 ГБ RAM бастап | Жоғары өнімді ВМ жасауға арналған жеке конфигурациялар |
| Custom | Шектелмеген | Жеке конфигурациялар |

{/cut}

Қосымша ВМ қалыптарын жасау үшін (мысалы, ресурстар көлемі үлкен болса) [техникалық қолдауға хабарласыңыз](/kz/contacts). Бұл жағдайда ВМ құны әрбір сұрау үшін жеке есептеледі.

{note:info}

CPU саны мен түрін, сондай-ақ бұрыннан жасалған виртуалды машинаның RAM көлемін ВМ қалыбын ауыстыру арқылы өзгертуге болады. Бұл процесс ВМ-ді қайта жүктеуді талап етеді.

{/note}

## {heading(ВМ конфигурация қалыптарының атауы)[id=flavor_names]}

Конфигурация қалыбы атауының құрылымы ВМ санатына байланысты өзгереді, бірақ жалпы түрі мынадай:

```plaintext
<ВМ_САНАТЫ>-<CPU_САНЫ>-<RAM_КӨЛЕМІ>-...
```

Белгілі бір ВМ санаттарына тән қосымша параметрлер болса, олар соңында көрсетіледі.

Мысалдар:

|ВМ қалыбының атауы | Сипаттама |
| --- | --- |
| STD2-2-4 | Intel Cascade Lake процессоры, 2 vCPU және 4 ГБ жедел жады бар ВМ |
| STD3-4-8 | Intel Ice Lake процессоры, 4 vCPU және 8 ГБ жедел жады бар ВМ |
| GPU1A-32-96-A100-1 | AMD EPYC 7662 процессоры және бір қосылған Nvidia Tesla A10040GB графикалық картасы бар ВМ |
