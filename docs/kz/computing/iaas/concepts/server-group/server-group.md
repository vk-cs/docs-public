# {heading(Серверлер топтары)[id=iaas-server-group]}

{include(/kz/_includes/_translated_by_ai.md)}

Серверлер тобы — белгілі бір міндетті шешу үшін біріктірілетін серверлер жиынтығы. Серверлер деп мыналар түсініледі:

- виртуалды машиналар;
- ДБ инстанстары;
- K8s кластерлері;
- қолданбалар дүкеніндегі қолданбалардың инстанстары.

{var(cloud)} платформасының есептеу тораптарында (compute nodes) серверлер топтарын орналастырудың келесі саясаттары қолданылады:

- `affinity` — топтың барлық серверлерін бір есептеу торабында орналастыру;
- `anti-affinity` — топ серверлерін әртүрлі есептеу тораптарында орналастыру;
- `soft-affinity` — мүмкіндігінше топ серверлерін бір есептеу торабында орналастыру (егер бір торапта орын жеткіліксіз болса, `affinity` жағдайындағыдай жасау қатесі орнына сервер басқа есептеу торабында жасалады);
- `soft-anti-affinity` — мүмкіндігінше топ серверлерін әртүрлі есептеу тораптарында орналастыру.

Серверлер тобын пайдалану мысалы: бірнеше ВМ, мысалы, қолданба серверін және осы қолданбаға арналған дерекқор серверін, осы ВМ арасындағы байланысты жеделдету үшін бір торапта орналастыру.
