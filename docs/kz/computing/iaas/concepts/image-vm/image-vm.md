Виртуалды машинаның конфигурациясы мен деректері бір немесе бірнеше файл түрінде сақталады. Бұл файлдар жергілікті ортада ВМ іске қосу үшін образ ретінде немесе жаңа ВМ жасау үшін пайдаланылуы мүмкін.

VK Cloud ішінде ВМ дискісінің негізінде [образ жасауға](../../instructions/images/images-manage#obrazdy_zhasau) немесе [образ файлын жүктеуге](../../instructions/images/images-manage#obrazdy_importtau) болады. Алынған образды жаңа виртуалды машинаны жасау үшін пайдалануға болады.

Дереккөзі диск болып табылатын образ орнатылған және бапталған операциялық жүйесі мен қосымшалары бар виртуалды машинаның көшірмесін жасауға мүмкіндік береді.

Файлдан жүктелген образды ВМ көші-қоны үшін немесе өзіңіздің операциялық жүйеңізбен виртуалды машина жасау үшін пайдалануға болады.

{note:info}

Cloud Servers `raw` форматындағы образдарды қолдайды.

{/note}
