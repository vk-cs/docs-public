# {heading(Тарификация)[id=iaas-tariffication]}

{include(/kz/_includes/_translated_by_ai.md)}

Тарифтендіру «pay as you go» қағидаты бойынша жүзеге асырылады: төлем тек тұтынылған ресурстар үшін минут дәлдігімен алынады, ақылы ОС пайдаланғаны үшін төлем күніне бір рет алынады.

Сервистердің бағасы [прайс-парақта](https://cloud.vk.com/pricelist) көрсетілген. Сервистердің жалпы бағасын есептеу үшін [калькуляторды](https://cloud.vk.com/pricing/) пайдалануға болады. Платформа сервистерінің ақысын төлеуге байланысты құралдар қалай жұмыс істейтіні туралы [Биллинг](/kz/intro/billing) бөлімінен оқыңыз.

## {heading(Тарифицируется)[id=iaas-tariffication-list]}

- CPU (vCPU) — әрбір ядро үшін. 1 vCPU виртуализация серверінің 1 физикалық ядросына сәйкес келеді.
- Жедел жад (RAM) — жедел жадтың әрбір 1 ГБ үшін.
- Лицензиялар (мысалы, Windows ОС үшін).
- Дискілер — дискілік кеңістіктің әрбір 1 ГБ үшін, баға диск түріне байланысты (SSD, HDD, High-IOPS, Low Latency NVMe).
- Файлдық сақтау орындары (NFS/CIFS) — дискілік кеңістіктің әрбір 1 ГБ үшін.
- Диск снимоктары — снимок көлемінің әрбір 1 ГБ үшін.
- [Жария IP мекенжайлары](/kz/networks/vnet/tariffication#vnet-tariffication-about).
- Сіз {var(cloud)} ішінде жасаған немесе импорттаған диск [образдары](../concepts/image-vm). Бағасы сақтау орнына байланысты:

  - [VK Object Storage](/kz/storage/s3) ішінде сақтау (әдепкі бойынша) — [Hotbox тарифі](/kz/storage/s3/tariffication) бойынша образ көлемінің әрбір 1 ГБ үшін және образды жүктеп алу көлемінің әрбір 1 ГБ үшін, мысалы ВМ жасау кезінде.
  - [Блоктық сақтау орнында](../concepts/data-storage/storage-types) сақтау — образ көлемінің әрбір 1 ГБ үшін HDD-диск бағасымен.

CPU (vCPU) және жедел жад (RAM) тек виртуалды машина іске қосылған жағдайда ғана тарифтеледі.

Виртуалды машина тоқтатылған кезде, қаражат есептен шығарылуын жалғастырады:

- Ақылы лицензияларды пайдалану үшін (мысалы, Windows ОС немесе RDS лицензиясы үшін, егер бұл опция белсендірілген болса).
- Дискілік кеңістікті жалға алу үшін.
- Қолда бар резервтік көшірмелерді сақтау үшін.
- Жария IP мекенжайлары үшін.

## {heading(Не тарифицируется)[id=iaas-tariffication-not-list]}

- Кіріс және шығыс трафик.
- Мониторинг.
- {var(cloud)} ішінде алдын ала бапталған жария диск образдары.

## {heading(Пример расчета цены)[id=iaas-tariffication-prices]}

Мысалы, Ubuntu Linux ОС, 4 CPU, 16 ГБ RAM, 100 ГБ SSD-дискі және жария IP бар виртуалды машинаның бағасы былай есептеледі:

`4 x 1 vCPU үшін баға` + `16 x 1 ГБ RAM үшін баға` + `100 x SSD үшін 1 ГБ баға` + `1 x жария IP үшін баға`
