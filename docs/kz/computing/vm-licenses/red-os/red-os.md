# {heading(РЕД ОС)[id=vm-licenses-red-os]}

{include(/kz/_includes/_translated_by_ai.md)}

{var(cloud)} аясында [РЕД ОС](https://redos.red-soft.ru) компаниясының бейнелерін қолдайтын виртуалды машиналарды орнату қолданылады.

Операциялық жүйенің қолжетімді редакциялары {var(cloud)} [жеке кабинет](https://kz.cloud.vk.com/app/services/infra/servers/add) арқылы ВМ жасау кезінде **Операциялық жүйе** ашылмалы тізімінде көрсетіледі.

РЕД ОС — толықтай отандық операциялық жүйе, RPM-пішіміндегі Linux дистрибутиві. Ресей Цифрлық даму министрлігінің Ресейлік бағдарламалық қамтамасыз ету тізіліміне енгізілген (№3751) және Ресей ФСТЭК-тің қолданыстағы сертификатына ие. Қосымша артықшылықтары:

- Нарық реттеушілерінің нормативтік-құқықтық актілеріне сәйкес келеді.
- Үйлесімді шешімдердің толыққанды экожүйесіне ие.
- IT-инфрақұрылымды орталықтандырылған басқаруға арналған құралдарға ие.
- Жұмыс үстелі мен графикалық ортаның интуитивті түсінікті интерфейсіне ие.
- ФОИВ, РОИВ және мемлекеттік корпорациялардың корпоративтік стандартына қабылданған.

«Сервер» конфигурациясының Стандартты және Сертификатталған редакциялары бар:

- Стандартты редакцияда пакеттердің ең жаңа және өзекті жиынтығы бар.
- Сертификатталған редакция Ресей ФСТЭК-імен сертификатталған. Қолданылуы мүмкін салалар:

  - ИСОП (прокуратура органдарының ақпараттық жүйесі) II классқа дейін қоса алғанда.
  - АСУ ТП (технологиялық процесті басқарудың автоматтандырылған жүйесі) I қорғалу класына дейін қоса алғанда.
  - ГИС (мемлекеттік ақпараттық жүйелер) I қорғалу класына дейін қоса алғанда.
  - ИСПДн (дербес деректердің ақпараттық жүйесі) I қорғалу деңгейіне дейін қоса алғанда.
  - КИИ (сыни ақпараттық инфрақұрылым) I маңыздылық санатына дейін қоса алғанда.

«Сервер» конфигурациясы мыналарды баптауға мүмкіндік береді:

- APM жаңарту репозиторийінің айнасы;
- желілік жүктеу сервері;
- деректер базалары;
- мониторинг жүйесі;
- прокси-сервер;
- домен контроллері;
- веб-сервер;
- пошта сервері;
- DNS, DHCP қызметтері;
- виртуализация жүйелері.

«Сервер» конфигурациясының барлық редакциялары мына архитектура түрлерін қолдайды:

- x86_64, i686,
- aarch64:
  
  - Raspberry Pi,
  - Байкал–М,
  - Kunpeng (Huawei Taishan).

- e2k (Эльбрус).
