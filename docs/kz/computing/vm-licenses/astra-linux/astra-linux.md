# {heading(Astra Linux)[id=vm-licenses-astra-linux]}

{include(/kz/_includes/_translated_by_ai.md)}

{var(cloud)} аясында [Astra Linux](https://astralinux.ru) компаниясының образдарын қолдайтын виртуалды машиналарды орнатуға қолдау көрсетіледі.

Операциялық жүйенің қолжетімді редакциялары {var(cloud)} [жеке кабинет](https://kz.cloud.vk.com/app/services/infra/servers/add) арқылы ВМ жасау кезінде **Операциялық жүйе** ашылмалы тізімінде көрсетіледі.

Astra Linux Speсial Edition 1.7 келесі қолдану сценарийлеріне жарамды:

- гипервизор (2-типті);
- ДҚБЖ сервері;
- домен контроллері / MS AD домен контроллерінің эмуляторы;
- пошта сервері;
- веб-сервер;
- файл сервері;
- proxy-сервер.

Ресейде платформа пайдаланушыларына вендордан қолдау және анықталған осалдықтар жабылатын әрі функционалдық жақсартулар іске асырылатын жаңартуларды тұрақты түрде алу мүмкіндігі қолжетімді. Astra Linux ашық репозиторийінде әзірлеуші сынақтан өткізген және үйлесімділігіне кепілдік берілген қолданбалардың кең жиынтығы бар.

## {heading(Astra Linux Special Edition 1.7 «Орел»)[id=vm-licenses-astra-linux-orel]}

Astra Linux Special Edition 1.7 «Орел» — бұл жалпыға қолжетімді желілерге қосылған инфрақұрылымдардың ашық сегменттерінде, білім беру мекемелерінде және құпия ақпаратты өңдемейтін басқа да IT-жүйелерде қолдануға ұсынылатын, арнайы мақсаттағы ресейлік толық функционалды Astra Linux операциялық жүйесін лицензиялау нұсқасы.

Шешім толыққанды серверлік платформа болып табылады, өйткені оның құрамында серверлік жүктемеге оңтайландырылған ядро, сондай-ақ көптеген міндеттерге арналған қолданбалардың кең жиынтығы бар репозиторий бар.

## {heading(Astra Linux Special Edition 1.7 «Воронеж»)[id=vm-licenses-astra-linux-voronezh]}

Astra Linux Special Edition 1.7 «Воронеж» — бұл кіріктірілген сертификатталған СЗИ кешені бар арнайы мақсаттағы Astra Linux операциялық жүйесін лицензиялау нұсқасы, ол құпия сипаттағы мәліметтерді, соның ішінде ГИС-та, ИСПДн-де және кез келген қорғалу сыныбының (маңыздылық санатының) КИИ маңызды объектілерінде қорғауды қамтамасыз етуге мүмкіндік береді.

Бұл лицензиялау нұсқасында пайдаланушыға ақпаратты қорғауды қамтамасыз ететін және ОС құралдарымен келесі міндеттерді шешуге мүмкіндік беретін сертификатталған функцияларды қолдану құқығы беріледі:

- пайдаланушыларды сәйкестендіру және аутентификациялау;
- қолжетімділікті дискрециялық басқару;
- қауіпсіздік оқиғаларын тіркеу;
- процестерді оқшаулау;
- жадты қорғау.

Сондай-ақ Astra Linux Special Edition 1.7 «Воронеж» құрамында тұтастықты бақылауды, бағдарламалық ортаны шектеуді және деректерді кепілді жоюды қамтамасыз ететін «Астра» ТК өз әзірлемесінің күшейтілген қорғаныс құралдары бар.

Astra Linux Special Edition 1.7 «Воронеж» толыққанды серверлік платформа ретінде қолданыла алады, өйткені оның құрамында серверлік жүктемеге оңтайландырылған ядро, сондай-ақ көптеген міндеттерге арналған қолданбалардың кең жиынтығы бар репозиторий бар.
