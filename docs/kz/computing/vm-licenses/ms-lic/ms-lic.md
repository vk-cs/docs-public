VK Cloud платформасы Microsoft Windows Server операциялық жүйелерінің алдын ала орнатылған лицензиялық көшірмелері бар виртуалды машиналарды жасауға, сондай-ақ Microsoft-тың қосымша бағдарламалық өнімдерін пайдалануға мүмкіндік береді — толық тізім [прайс-парақта](https://cloud.vk.com/pricelist) қолжетімді.

{note:warn}

VK Cloud Microsoft БҚ-сын Service Provider License Agreement (SPLA) лицензиялық келісімі бойынша ұсынады. Осы келісім аясында Microsoft БҚ-сын тек VK Cloud бұлтындағы виртуалды машиналарда пайдалануға болады.

{/note}

## Пайдалану шарттары

Microsoft БҚ-сын лицензиялау және пайдалану құқықтары бойынша сұрақтар туындаса, заң бөліміңізге немесе Microsoft сауда өкілдеріне жүгініңіз.

Осы өнімді пайдалану арқылы сіз келесі өнімдерді пайдалану шарттарымен келісесіз: [Microsoft](https://www.microsoft.com/licensing/docs/view/Services-Provider-Use-Rights-SPUR?lang=1).

## Бағасы

БҚ-ны бір ай пайдалану бағасын мына жерден білуге болады:

- виртуалды машинаны (ВМ) жасау кезінде;
- [жоба баптауларында](https://kz.cloud.vk.com/app/project/) **Бағалар** қойындысында.

## {heading(Лицензияларды автоматты түрде белсендіру үшін ВМ-ге қойылатын талаптар)[id=requirements]}

VK Cloud компаниясынан сатып алынған Microsoft Windows Server және Microsoft Office өнімдеріне лицензияларды автоматты түрде белсендіру үшін:

- ВМ интернет арқылы қолжетімді болуы керек. Егер қолжетімділік VK Cloud ұсынған IP-мекенжай арқылы жүзеге асырылмаса, виртуалды машинаның IP-мекенжайын [техникалық қолдауға](/kz/contacts) хабарлау қажет.
- ВМ-де TCP 1688 порты ашық болуы керек.

Windows ОС-і бар ВМ өзінің белсендіруін тұрақты түрде ұзартып отыратындықтан, ВМ өмір сүруінің бүкіл мерзімі ішінде белсендіру серверімен байланысты қамтамасыз ету қажет.

## Windows Server

Microsoft Windows ОС-і бар ВМ жасау кезінде таңдалған редакциядағы ОС-тің лицензиялық көшірмесі автоматты түрде өрістетіледі. Егер [ВМ-ге қойылатын талаптар](#requirements) сақталса, лицензия ВМ қосылған кезде автоматты түрде белсендіріледі.

Microsoft Windows Server жүйесінің қолжетімді редакциялары ВМ-ді VK Cloud [жеке кабинеті](https://kz.cloud.vk.com/app/services/infra/servers/add) арқылы жасау кезінде **Операциялық жүйе** ашылмалы тізімінде көрсетіледі.

Windows ОС-і бар виртуалды машинаның әр 2 vCPU-і үшін бір лицензияны пайдалану қажет. Мысалы, 5 vCPU бар ВМ жасағанда 3 Windows Server лицензиясы қажет болады. Виртуалды машинаның түрі өзгертілген кезде лицензиялар саны өзгертілген конфигурациядағы виртуалды ядролар санына сәйкес өзгертіледі.

{note:info}

Лицензиялар төлемі қосымша әрекеттерді талап етпейді және жасалған виртуалды машинаның конфигурациясына сәйкес автоматты түрде есептен шығарылады.

{/note}

## Remote Desktop Services

Әдепкі бойынша Windows Server ОС-ін орнатқаннан кейін виртуалды машинаға тек екі бір мезгілдегі қосылым қолжетімді болады (консоль және RDP).

Бұл екі қосылымды тек серверді баптау және әкімшілендіру үшін пайдалануға болады. Қосылымдардың көбірек санын (әкімшілік немесе бірнеше пайдаланушыға арналған қосылымдарды) рұқсат ету үшін:

1. Қашықтағы жұмыс үстелдері қызметтерінің тиісті клиенттік лицензияларының қолда бар екеніне көз жеткізіңіз.
1. Мақсатты ВМ-ге қашықтағы жұмыс үстелдері сеанстары торабының рөлін орнатыңыз.

Клиенттік қол жеткізу лицензияларының (CAL) екі түрі бар:

- Пайдаланушыға — бір пайдаланушыға шектеусіз құрылғылар санында қол жеткізуді береді. Бұл лицензия түрі тек RDS-сервер Active Directory доменіне кірген жағдайда ғана қолжетімді.
- Құрылғыға — серверіңізге жүгінетін бір құрылғыға қол жеткізуді береді. Құрылғымен жұмыс істейтін пайдаланушылар саны шектелмейді. Құрылғыға берілген лицензиялар бірнеше қызметкер бір құрылғыны пайдалана алатын компанияларда, мысалы бірнеше ауысымда жұмыс істегенде, шығындарды азайтуға және әкімшілендіруді жеңілдетуге мүмкіндік береді.

Егер RDS-сервер Active Directory доменіне кірсе, лицензияның екі түрі де қолжетімді.

Лицензия қашықтағы жұмыс үстелдері протоколын (RDP) пайдаланып қосылатын ОС-тің әрбір пайдаланушысы үшін қажет болады. Қашықтағы жұмыс үстелдері сервисін белсендіру әдепкі бойынша қолжетімді RDP арқылы қосылуды толық алмастырады. Клиенттік қол жеткізу лицензияларын белсендіру үшін [нұсқаулықты пайдаланыңыз](/kz/computing/iaas/instructions/vm/vm-lic-manage).

{note:warn}
Remote Desktop Services лицензиялары тек VK Cloud жүйесінде сатып алынған MS Windows операциялық жүйесінің лицензиялық көшірмесіне ғана берілуі мүмкін.
{/note}

## SQL Server

SQL Server деректер базасын басқару жүйесі SQL Server алдын ала орнатылған көшірмесі бар дайын бейне түрінде орнатуға қолжетімді. Ол пайдаланылатын ВМ идентификаторын [техникалық қолдауға](/kz/contacts) хабарлаңыз.

«Ядро бойынша» лицензиялау тәсілі қолданылады: ВМ-ге қосылған әрбір келесі екі ядро үшін баға артады. Лицензия сатып алынатын ядролардың ең аз саны — төртеу.
