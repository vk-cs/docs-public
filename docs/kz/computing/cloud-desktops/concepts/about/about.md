# {heading(Сервис туралы)[id=desktops-about]}

{include(/kz/_includes/_translated_by_ai.md)}

Cloud Desktop сервисі VDI (Virtual Desktop Infrastructure) технологиясына негізделген және пайдаланушыларға виртуалды жұмыс үстелдерін ұсынуға арналған. Жұмыс үстелі — оған қонақ ОС және басқа БҚ орнатылған виртуалды жұмыс орны (ВРМ). Жұмыс үстелдері {linkto(#desktops-about-pool)[text=жұмыс үстелдері пулы]}. аясында қашықтағы серверде жасалады. Жобада пайдаланушылардың әртүрлі топтары үшін бірнеше пул болуы мүмкін. Пайдаланушылар тобын сервис әкімшісі пулды жасау кезінде тағайындайды.

Жұмыс үстелдері виртуализация компоненті орнатылып, бапталған алдын ала дайындалған образдардан сервис арқылы автоматты түрде жасалады. Пулдағы жұмыс үстелдеріне арналған образ жұмыс үстелі пайдаланушылары орындайтын міндеттерге байланысты таңдалады. {var(cloud)} ұсынатын образдарды пайдалана аласыз немесе {linkto(../../../../computing/cloud-desktops/concepts/desktop-image#desktops-image)[text=жеке образдарыңызды]} дайындай аласыз.

Образдан жұмыс үстелі жасалған кезде оған автоматты режимде бағдарламалық компонент — ВРМ агенті орнатылады. Жұмыс үстелі пайдаланушысы өз компьютеріне басқа бағдарламалық компонентті — Cloud Desktop Client қолданбасын (клиентті) орнатады. Агент клиентпен өзара әрекеттесе отырып, пайдаланушыға ВРМ-ге және оның инфрақұрылымына қосылуға мүмкіндік береді. ВРМ-мен жұмыс істеу үшін пайдаланушының компьютері {linkto(../../../../computing/cloud-desktops/concepts/system-reqs#desktops-system-reqs)[text=жүйелік талаптарға]} сай болуы керек.

ВРМ-ге қосылған кезде пайдаланушы өзі жұмыс істейтін ұйымның каталог қызметінде авторизациядан өтеді. Cloud Desktop сервисі Active Directory (AD) және OpenLDAP каталог қызметтерімен интеграцияны қолдайды.

## {heading(Жұмыс үстелдері пулы)[id=desktops-about-pool]}

Жұмыс үстелдері пулы — бұл бұлттағы ортақ есептеу ресурстарын пайдаланатын бірнеше жұмыс үстелінің бірігуі. Жұмыс үстелдері пулы арқылы сервис әкімшісі жұмыс үстелдері үшін ортақ баптауларды {ifdef(public,private,private-pg)}[орната](../../instructions/desktops-pool/manage){/ifdef}{ifdef(private-pdf,private-pg-pdf)}{linkto(../../../../computing/cloud-desktops/instructions/desktops-pool/manage#desktops-pool-manage)[text=орната]}{/ifdef} алады. Cloud Desktop ішінде екі түрлі пул жасауға болады:

- Сессиялық — мұндай пулда жұмыс үстелі қажеттілікке қарай автоматты режимде жасалады, оған бірінші болып қосылған пайдаланушыға тағайындалады және пайдаланушы жұмыс үстелінің қонақ ОС-інен шыққаннан кейін жойылады. Бірдей конфигурациясы және бірдей БҚ жиынтығы бар көп мөлшердегі жұмыс үстелдерін өрістету қажет болса, сессиялық пулды пайдаланыңыз. Сессиялық пулда жұмыс үстелдерінің ыстық резерві қарастырылған, яғни пайдаланушының қосылуына алдын ала дайындалған жұмыс үстелдерінің белгілі бір саны болады.
- Дербестендірілген — мұндай пулда әрбір жұмыс үстелі оған бірінші болып қосылған пайдаланушыға бекітіледі. Пайдаланушы бұл жұмыс үстеліне оны әкімші жойғанға дейін иелік етеді және оны өз міндеттерін шешуге бейімдей алады. Бұл пул түрі көбінесе қолданылады.

## {heading(Сервистің рөлдік моделі)[id=desktops-about-roles]}

Cloud Desktop сервисінде келесі рөлдер қарастырылған:

- Әкімші.

  Сервисті {var(cloud)} {ifdef(public,private,private-pg)}[жеке кабинеті](https://kz.cloud.vk.com/app){/ifdef}{ifdef(private-pdf,private-pg-pdf)}{linkto(../../../../intro/authorization/lk_entry#prerequisites_vkc_ui)[text=жеке кабинеті]}{/ifdef} арқылы басқарады. Сервис инфрақұрылымы мен жұмыс үстелдері пулдарына арналған желілерді, каталог қызметімен интеграцияны және екі факторлы аутентификацияны {ifdef(public,private,private-pg)}[баптайды](../../instructions/config){/ifdef}{ifdef(private-pdf,private-pg-pdf)}{linkto(../../../../computing/cloud-desktops/instructions/config#desktops-config)[text=баптайды]}{/ifdef}. Жұмыс үстелдерінің {linkto(../../../../computing/cloud-desktops/instructions/desktops-pool/add#desktops-pool-add)[text=пулдарын жасайды]} және оларды каталог сервисіндегі пайдаланушылар топтарына тағайындайды.

- Пайдаланушы.

  Виртуалды жұмыс үстелдерінің тікелей пайдаланушысы. Жұмыс үстеліне қол жеткізуге арналған клиентті өз компьютеріне өз бетінше {linkto(../../../../computing/cloud-desktops/user-instructions#desktops-install-client)[text=орнатады]}.

## {heading(Сервис қандай міндеттерге сай келеді)[id=desktops-about-task]}

Cloud Desktop сервисі бір жұмыс орнын әртүрлі адамдар пайдалана алатын кәсіпорынның немесе кәсіпорындар желісінің коммуникациялық инфрақұрылымын құруға немесе жаңғыртуға қолайлы.

Сервис шешетін міндеттер:

- Жұмыс үстелдерін өрістету және оларды пулдар арқылы әрі қарай әкімшілендіру.
- Жұмыс үстелдері пулдарын басқару және олардағы жүктемені бөлу.
- Пайдаланушыларға арналған виртуалды жұмыс үстелдерін ұйымдастыру және оларға үздіксіз қолжетімділікті қамтамасыз ету.
- Жұмыс үстелдерін рұқсатсыз қол жеткізуден қорғау.

## {heading(Сервистің мүмкіндіктері)[id=desktops-about-service-features]}

- Виртуалды жұмыс үстелдері іске қосылатын қонақ ОС-терді қолдау: Windows ОС, Linux ОС.
- [AD](https://learn.microsoft.com/ru-ru/windows-server/identity/ad-ds/get-started/virtual-dc/active-directory-domain-services-overview)/[LDAP](https://docs.altlinux.org/ru-RU/archive/2.4/html-single/master/alt-docs-master/ch06s11.html) каталог қызметтерін қолдау.
- RDP протоколы бойынша деректерді беруді қолдау.
- ВРМ-нің екі түрін қолдау: дербестендірілген және сессиялық.
- ВРМ-ге қосылған {linkto(../../../../computing/cloud-desktops/instructions/desktops-pool/add#desktops-pool-configure-peripherals)[text=перифериялық құрылғыларды]} пайдалануды бақылау.
- 1000-нан астам бір мезгілдегі пайдаланушылық қосылымдарды қызмет көрсету.
- Пайдаланушыларды қашықтағы жұмыс үстелдеріне қорғалған протокол арқылы қосу.
- Сервис өрістетілген кезде SSL сертификаттарын автоматты түрде қосу және {linkto(../../../../computing/cloud-desktops/instructions/config/ssl-certificates#desktops-ssl-certificates)[text=өз SSL сертификаттарын жүктеу]} мүмкіндігі.
- Алдын ала дайындалған мастер-образдан виртуализация платформасында ВМ жасау және баптау.
- {var(cloud)} ұсынатын образдардан жоғары өнімді графикалық процессорлары бар ({linkto(../../../gpu/concepts/about#gpu-about-flavors)[text=GPU]}) жұмыс үстелдерін өрістету.
- Басқару, лицензиялау, сақтау серверлері және конфигурация өзгерістерін есепке алу үшін ақауға төзімділік механизмдерін қолдау.
