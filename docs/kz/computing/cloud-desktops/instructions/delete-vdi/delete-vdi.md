Жойылады:

- қызмет инфрақұрылымы: виртуалды машиналар, жүктеме теңгергіштері;
- қызметке қосылу баптаулары;
- баптауларды тексеру үшін өрістетілген виртуалды машиналар.

Жойылмайды:

- VPN-ге қосылған VDI сервистік желісі;
- пайдаланушы жасаған желілер.

{note:err}

Жою операциясын болдырмау мүмкін емес!

{/note}

Инфрақұрылым мен баптауларды жою үшін:

1. [Өтіңіз](https://kz.cloud.vk.com/app) VK Cloud жеке кабинетіне.
1. **Cloud Desktop** → **Қызмет баптаулары** бөліміне өтіңіз.
1. **VDI жою** батырмасын басыңыз.
1. Ашылған терезеде келісімді белгілеп, **Жою** батырмасын басыңыз.
