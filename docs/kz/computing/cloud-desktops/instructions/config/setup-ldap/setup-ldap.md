# {heading(Пайдаланушылар каталогына қосылуды баптау)[id=desktops-setup-ldap]}

{include(/kz/_includes/_translated_by_ai.md)}

{note:warn}
Cloud Desktop жұмысы AD/LDAP пайдаланушылар каталогы қызметінсіз мүмкін емес. Егер желіңізде каталог қызметі болмаса, оны өзіңіз жасаңыз.
{/note}

Пайдаланушылар каталогына қосылу әр жоба үшін бөлек бапталады. Бір жоба үшін тек бір ғана қосылымды баптауға болады.

## {heading(1. Пайдаланушылар каталогына қосылуды баптаңыз)[id=desktops-setup-ldap-step1]}

1. {ifdef(public)}[Өтіңіз](https://kz.cloud.vk.com/app/){/ifdef}{ifdef(private,private-pg,private-pdf,private-pg-pdf)}{linkto(../../../../../intro/authorization/lk_entry#prerequisites_vkc_ui)[text=Өтіңіз]}{/ifdef} {var(cloud)} жеке кабинетіне.
1. **Cloud Desktop** → **Қызмет баптаулары** бөліміне өтіңіз.
1. **Каталогты баптау** қойындысында параметрлерді орнатыңыз:

   - **Хост**: сыртқы тіркелгі каталогы серверінің IP-мекенжайын немесе FQDN атауын көрсетіңіз.
   - **Порт**: протокол бойынша қол жеткізу портының нөмірін көрсетіңіз (`389` LDAP үшін).
   - **Base DN**: AD немесе LDAP ішінен объектілерді іздеуге арналған түбірлік каталогты орнатыңыз.
   - **LDAP каталогына қосылуға арналған пайдаланушының DN**: OpenLDAP базасындағы тіркелгіге толық жолды көрсетіңіз. Бұл тіркелгі тіркелгілер каталогымен синхрондау үшін пайдаланылады.

     Active Directory үшін мұндай тіркелгі **Domain Admins** тобының, ал ALD Pro үшін **Computers Administrators** тобының мүшесі болуы керек.

   - **Құпиясөз**: **LDAP каталогына қосылуға арналған пайдаланушының DN** өрісінде көрсетілген тіркелгінің құпиясөзін орнатыңыз.
   - **Таңдау өрісі**: LDAP каталогындағы бірегей пайдаланушы атрибутын орнатыңыз, сол бойынша пайдаланушы базадан таңдап алынады (мысалы, `UID` немесе `SamAccountName`).

1. **Сақтау** батырмасын басыңыз.

{note:info}
Егер Cloud Desktop жұмыс істеп тұрған кезде пайдаланушылар каталогына қосылу баптауларын өзгертсеңіз, барлық бар пулдар автоматты түрде жаңа қосылу баптауларына ауысады.
{/note}

## {heading(2. Қосылу баптауларын тексеріңіз)[id=desktops-setup-ldap-step2]}

Пайдаланушылар каталогына қосылу баптауларын тексеру үшін, баптаулардағы өзгерістерді сақтаудың қажеті жоқ.

{note:warn}
Тексеру үшін виртуалды машина өрістетіледі, оның атауы `vdi-checker` деп басталады. {ifdef(public)}Осы ВМ пайдалану {linkto(../../../../../computing/iaas/tariffication#iaas-tariffication)[text=тарифтеледі]}.{/ifdef}
{/note}

Қызмет үшін каталог баптауларының дұрыстығын тексеру үшін:

1. {ifdef(public)}[Өтіңіз](https://kz.cloud.vk.com/app/){/ifdef}{ifdef(private,private-pg,private-pdf,private-pg-pdf)}{linkto(../../../../../intro/authorization/lk_entry#prerequisites_vkc_ui)[text=Өтіңіз]}{/ifdef} {var(cloud)} жеке кабинетіне.
1. **Cloud Desktop** → **Қызмет баптаулары** бөліміне өтіңіз.
1. **Каталогты баптау** қойындысында **Каталог баптауларын тексеру** опциясын белсендіріңіз.
1. **Қосылымды тексеру** түймесін басыңыз.
1. Операцияның аяқталуын күтіңіз.

Тексеру нәтижесінің толық мәліметін көру үшін **Толығырақ** түймесін басыңыз.
