# {heading(MLflow инстансын құру)[id=mlplatform-mlflow-instructions-create]}

{include(/kz/_includes/_translated_by_ai.md)}

Мақалада {var(cloud)} жеке кабинеті арқылы MLflow инстансын қалай құру керектігі сипатталған.

Инстансты келесі құралдардың көмегімен де құруға болады:

- [Terraform](/kz/tools-for-using-services/terraform/how-to-guides/mlplatform/mlflow);
- Cloud ML Platform кітапханасы, [create_mlflow](../../../mlplatform-lib/lib-reference#mllib-lib-reference-create_mlflow) әдісі.

MLflow инстансы екі режимде жұмыс істей алады: [JupyterHub-пен бірге](../../../concepts/mlflow-modes#with_jh) немесе бөлек, [Standalone режимінде](../../../concepts/mlflow-modes#mlplatform-standalone).

Егер сізге JupyterHub-пен бірге жұмыс істейтін MLflow инстансы қажет болса және JupyterHub инстансы әлі құрылмаса, оны [құрыңыз](../../../jupyterhub/instructions/create).

MLflow инстансын құру үшін:

1. {var(cloud)} жеке кабинетіне [өтіңіз](https://kz.cloud.vk.com/app).
1. **ML Platform** бөліміне өтіңіз.
1. **MLflow** карточкасындағы **Создать инстанс** батырмасын басыңыз.
1. **Конфигурация** қадамында инстанс параметрлерін көрсетіңіз:

    - **Имя инстанса**: инстанс атын беріңіз, ол сондай-ақ ОС жүйесіндегі хост атауы (hostname) болады. Атау тек латын әріптерін, цифрларды және `-`, `_`, `.` арнайы таңбаларын қамтуы мүмкін.
    - **Категория виртуальной машины**: ВМ алдын ала орнатылған конфигурацияларының [санатын](/kz/computing/iaas/concepts/vm/flavor) таңдаңыз.
    - **Тип виртуальной машины**: ВМ алдын ала орнатылған конфигурациясын таңдаңыз.
    - **Зона доступности**: инстанс іске қосылатын [деректер орталығын](/kz/start/concepts/architecture#architecture-az) таңдаңыз.
    - **Включить демо-версию MLflow**: егер опция қосулы болса, барлық деректер инстанс ВМ-інде сақталады, ал өшірулі болса — деректерді сақтау үшін DBaaS PostgreSQL дерекқоры бар {var(s3)} бакеті қосылады. Жұмыс деректерін сақтау үшін демо режимін функционалмен танысу үшін ғана пайдалану ұсынылады.
    - **Размер диска**: ВМ дискінің өлшемін ГБ-пен көрсетіңіз. Ең аз мәні — 20 ГБ. Демо режимінде тек бір диск пайдаланылады. Қалыпты режимде инстансты құру кезінде оған root-диск және қосымша деректер дискісі (data-диск) қосылады. Root-диск өлшемі тіркелген — 25 ГБ. Сіз тек data-диск өлшемін өзгерте аласыз.
    - **Тип диска**: құрылатын дискінің [түрін](/kz/computing/iaas/concepts/data-storage/disk-types) таңдаңыз.
    - **Подключить инстанс JupyterHub для совместной работы**: әдепкі бойынша әрбір MLflow инстансы [JupyterHub инстансымен бірге](../../../concepts/mlflow-modes#mlplatform-with_jh) жұмыс істейді. MLflow инстансы {var(cloud)} ішіндегі JupyterHub инстансынан [бөлек](../../../concepts/mlflow-modes#mlplatform-standalone) жұмыс істеуі үшін опцияны өшіріңіз.
    - **Инстанс JupyterHub**: бірлесіп жұмыс істеу үшін JupyterHub инстансын таңдаңыз. **Подключить инстанс JupyterHub для совместной работы** опциясы қосулы болса, қолжетімді.
    - **Выбор доменного имени**: инстанстың DNS атауын таңдаңыз. **Подключить инстанс JupyterHub для совместной работы** опциясы өшірулі болса, қолжетімді. Өз доменіңізді қосу үшін тізімнен **Подключение своего домена** тармағын таңдап, төмендегі өрісте атауды беріңіз.
    - **Имя**: инстанстың DNS атауын көрсетіңіз. **Подключение своего домена** таңдалса, қолжетімді. DNS атауына қойылатын талаптар:

      - Ең көп ұзындығы — 63 таңба.
      - Тек цифрларға, кіші латын әріптеріне, `.` және `-` арнайы таңбаларына рұқсат етіледі.
      - `-` таңбасынан басталып, сонымен аяқталмауы керек.
    - **Имя пользователя**: инстанс ОС жүйесінде авторизациядан өту үшін пайдаланушы атын беріңіз. **Подключить инстанс JupyterHub для совместной работы** опциясы өшірулі болса, қолжетімді.
    - **Пароль пользователя**: инстанс ОС жүйесінде авторизациядан өту үшін пайдаланушы паролін беріңіз. **Подключить инстанс JupyterHub для совместной работы** опциясы өшірулі болса, қолжетімді. Парольде мыналар болуы керек:

        - 8 немесе одан көп таңба;
        - латын әліпбиінің бір немесе бірнеше бас әрпі;
        - латын әліпбиінің бір немесе бірнеше кіші әрпі;
        - бір немесе бірнеше цифр;
        - келесі ауқымнан бір немесе бірнеше арнайы таңба: `!`, `#`, `$`, `%`, `&`, `(`, `)`, `*`, `+`, `,`, `.`, `:`, `;`, `<`, `=`, `>`, `?`, `@`, `[`, `]`, `^`,`_`, `{`,`|`, `}`,`~`,`-`.

    {note:err}

    Парольді сақтап қойыңыз. Парольді қалпына келтіру мүмкін емес.

    {/note}

1. **Следующий шаг** батырмасын басыңыз.

1. **Выбор сети** қадамында инстанс орналастырылатын желі параметрлерін көрсетіңіз:

    - **Сеть**: желіні таңдаңыз немесе жаңасын құрыңыз.
    - **Адрес подсети**: жаңа желі құру таңдалса, ішкі желі мекенжайын көрсетіңіз.

   {note:info}

   Егер MLflow инстансы JupyterHub-пен бірге құрылса, оларды бір желіде орналастырыңыз.

   {/note}

1. **Создать инстанс** батырмасын басыңыз.

Инстансты құру әдетте 10–15 минутты алады. Осы уақыт ішінде инстанс дискінде операциялық жүйе жайылып, инстанс ВМ-і таңдалған параметрлерге сәйкес бапталады.

Инстанс құрылғаннан кейін оның сипаттамалары бар бет ашылады.
