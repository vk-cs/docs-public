{cut(Қолдау қызметіне жүгіну)}

Қолдау қызметіне жүгінген кезде Cloud ML Platform инстансыңыздың ID-н көрсетуді ұмытпаңыз. Инстанс ID-н «ML Platform» -> «Инстансы» -> Инстанс атауы бөлімінен табуға болады.

Инстанс атауын басқаннан кейін толық ақпарат бар бет ашылады. «ID» жолында «Көшіру» белгішесін басу арқылы инстанс ID-н көшіре аласыз.

{/cut}

{cut(JupyterHub инстансы қолжетімсіз болғандағы әрекеттер)}

Егер JupyterHub интерфейсі жауап беруді тоқтатса, JupyterHub бар ВМ инстансын қайта жүктеуіңіз керек.

ВМ-ді қайта жүктеу үшін:

1. **Бұлттық есептеулер** -> **Виртуалды машиналар** бөліміне өтіңіз.
2. JupyterHub бар ВМ инстансы үшін ![ ](/kz/assets/more-icon.svg "inline") белгішесін басып, **Қайта жүктеу** тармағын таңдаңыз.

Виртуалды машиналарды басқару туралы толығырақ — [«ВМ басқару»](/kz/computing/iaas/instructions/vm/vm-manage) мақаласында.

{/cut}

{cut(Jupyter Kernel қателерін жою)}

Кітапханаларды орнатқаннан кейін немесе Jupyter Kernel тұрып қалған кезде оны қайта іске қосу қажет. Kernel-ді қайта іске қосу үшін JupyterHub интерфейсінде «Kernel» -> «Restart Kernel» мәзір тармағын таңдаңыз.

JupyterHub интерфейсіне қосылу туралы толығырақ [«JupyterHub-қа қосылу»](../jupyterhub/instructions/connect) мақаласынан біле аласыз.

{/cut}
