Бұл нұсқаулық жергілікті жұмыс орныңызда орнатылған VS Code пайдаланып, бұлттағы JupyterHub серверіне қосылуға көмектеседі. Осылайша, ресурсты көп қажет ететін есептеулерді іске қосу үшін жергілікті ноутбугыңыздан қуаттырақ бұлт серверлерін пайдалана аласыз.

Бұлтта бірнеше GPU, ондаған ядро және жүздеген гигабайт жедел жадысы бар ВМ-де JupyterHub-ты өрістете аласыз.

Жергілікті компьютеріңізде Visual Studio Code арқылы JupyterHub-пен жұмыс істеуді бастау үшін келесі қадамдарды орындаңыз:

1. JupyterHub access token-ын алыңыз.

    {note:info}

    JupyterHub үшінші тарап қолданбаларымен пайдалану үшін өз токендерін жасауға мүмкіндік береді. Бұл токендерді Jupyter Notebook қол жеткізу токені қажет болатын барлық жерде пайдалануға болады. Бұл токен Visual Studio Code үшін қажет.

    {/note}

    1. JupyterHub-қа логиніңіз бен құпиясөзіңізді енгізіп, авторизациядан өтіңіз.

    1. Hub Control Panel бөліміне өтіңіз. Бұл тармақты **File -> Hub Control Panel** таңдау арқылы табуға болады. Содан кейін мәзірдің жоғарғы бөлігінен осы тармақты таңдап, **Token** қойындысына өтіңіз.

    1. Жаңа токеннің сипаттамасын енгізіңіз, токеннің жарамдылық мерзімін таңдаңыз және «Request new API token» түймесін басыңыз.

    1. Жасалған токенді көшіріп, оны қауіпсіз жерде сақтаңыз. Оны JupyterHub үшін құпиясөз ретінде қарастыруыңыз керек. Оны пайдалануды тоқтатқан кезде, дәл осы Hub Control Panel бөлімі арқылы токенді кері қайтарып алуға болады және қайтарып алуыңыз керек.

1. VS Code-ты бұлттағы JupyterHub инстансына қосыңыз.

    1. Visual Studio Code-та командалар палитрасын (command palette) ашыңыз (MacOS-та CMD+SHIFT+P, басқа ОС-та CTRL+SHIFT+P).

    1. «Python: Specify local or remote Jupyter server for connections» тармағын таңдаңыз.

    1. `https://<your-hub-url>/jh/user/<your-hub-user-name>/?token=<your-token>` түріндегі сервер URL-ін жасаңыз.

        Мысал:

        ```http
        https://ml-platform-3c000000000000.ml.msk.vkcs.cloud/jh/user/admin/?token=000f00bdf7d00fbd0000000c0006ebfd
        ```

        {note:warn}

        JupyterHub пайдаланушы атыңызда арнайы таңбалар болса, ол экрандалып көрсетілуі мүмкін екеніне назар аударыңыз. Сервер URL-ін құрастыруға арналған дұрыс пайдаланушы атын браузер жолындағы `/user/` бөлігінен кейін көруге болады.

        Біздің мысалда бұл `admin`:

        ```http
        https://ml-platform-3c000000000000.ml.msk.vkcs.cloud/jh/user/admin/lab
        ```

        {/note}

1. (Қалауыңызша) VS Code үшін [Jupyter Extension](https://devblogs.microsoft.com/python/introducing-the-jupyter-extension-for-vs-code/) кеңейтімін орнатыңыз.
1. Жасалған қосылымды тексеріңіз.

    VS Code-та жаңа Notebook жасаңыз немесе барын ашыңыз. Енді оған арналған kernel бұлттағы JupyterHub-та іске қосылады.

    Мұны `!hostname` командасын орындау арқылы тексере аласыз, ол жергілікті хост атауының орнына қашықтағы JupyterHub серверіңіздің хост атауын қайтаруы тиіс.

    Сондай-ақ, жергілікті ортаңызда жоқ, бірақ қашықтағы JupyterHub серверінде бар кітапханаларды импорттап көруге болады.

    {note:info}

    Пайдаланушыға арналған қашықтағы JupyterHub сервері Notebook ашуға тырысқан сәтте іске қосылып тұруы керек — Visual Studio Code оны автоматты түрде іске қоспайды. Егер ол тоқтатылған болса, JupyterHub-қа кіріп, оны қайта іске қосуыңыз қажет. Бұл үшін жаңа токен керек емес.

    {/note}

    {note:warn}

    Файлдық жүйеге қол жеткізуді қадағалаңыз. `open()` шақырған кезде (немесе түптеп келгенде оқу үшін файлға жүгінетін көмекші функцияны), ол жергілікті жүйеңіздің ағымдағы директориясынан емес, қашықтағы JupyterHub серверіңіздің үй директориясынан оқылады.

    Егер жаңа файлды жергілікті түрде Jupyter notebook жанында қоссаңыз, ол VS Code-тағы Jupyter notebook ішінде оқу үшін автоматты түрде қолжетімді болмайды.

    {/note}

    {note:warn}

    Жергілікті түрде `pip` немесе `conda` пакеттерін орнату ешқандай әсер бермейді, себебі сіздің Python kernel бұлттағы қашықтағы JupyterHub серверінде жұмыс істеп тұр. Қажетті пакеттерді бұлттағы JupyterHub ортасына `pip` немесе `conda` арқылы орнатуға болады.

    {/note}

{note:info}

JupyterHub серверіне қосылу туралы қосымша ақпарат [VS Code ресми құжаттамасында](https://code.visualstudio.com/docs/datascience/jupyter-notebooks#_connect-to-a-remote-jupyter-server) берілген.

{/note}
