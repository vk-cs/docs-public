# {heading(MLflow Deploy)[id=mlflowdeploy]}

{include(/kz/_includes/_translated_by_ai.md)}

MLflow Deploy — MLflow және JupyterHub-пен интеграцияланған, ML-модельдерді Docker-контейнерлерге автоматты түрде орап, оларды REST API арқылы қолжетімді етуге мүмкіндік беретін сервис.

Бұл бөлімде MLflow Deploy сервисімен жұмыс істеу қағидаттары, MLflow Deploy инстанстарын жасау және олармен жұмыс істеу сипатталған.
