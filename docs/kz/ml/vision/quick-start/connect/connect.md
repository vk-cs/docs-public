Машиналық оқыту сервисі әдепкі бойынша VK Cloud жобаларында қосылған. Сервиске төменде сипатталған тәсілдермен қосылуға болады.

## API Endpoint

API Endpoint — бұл басқару үшін сервиске қосылу жүзеге асырылатын URL болатын «кіру нүктелері». Машиналық оқыту үшін олардың екеуі бар:

- Vision кескіндерді тануға арналған `https://smarty.mail.ru/`
- Vision бейнені тануға арналған `https://smarty.mail.ru/`

## Клиент идентификаторы және құпия кілт арқылы қол жеткізу

Клиент идентификаторы мен құпия кілт арқылы қолжетімділікті басқару үшін [Vision API бетінде](https://kz.cloud.vk.com/app/services/machinelearning/vision/access/) қажетті ақпарат көрсетіледі. Клиент идентификаторы әлдеқашан жасалған және ауыстыруға жатпайды, ал кілтті тиісті батырма арқылы қайта шығаруға болады.

## Сервистік токен арқылы қол жеткізу

Сервистік токенді жасау үшін батырманы басып, қажетті скоупты таңдау керек:

1. «Сервистік токен қосу» батырмасын басыңыз.
2. Қажетті скоупты таңдаңыз.
3. «Жасау» түймесін басыңыз.

Токен жасалып, бетте көрсетіледі.
