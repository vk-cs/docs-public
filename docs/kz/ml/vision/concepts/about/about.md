Vision — бұл машиналық оқыту мен жасанды нейрондық желілерге негізделген тұлғаларды, объектілерді, процестерді тану технологиясы. Vision әртүрлі күрделіліктегі кешенді визуалды тексеруді автоматтандыруға және оның дәлдігін арттыруға мүмкіндік береді.

- Миллионның ішінде тұлғаны анықтау дәлдігі 98%.
- Инференсте TensorRT, GPU-да фотосуретке <10ms.
- Сахналарды танудың 314 класы, 25 000 класс — объектілер.

Vision технологиялары үнемі кеңейтіліп отыратын API бағдарламалық интерфейстері арқылы қолжетімді. Олардың тізімі осы анықтамалық орталықта қолжетімді. API көмегімен сіз келесі кейстерді шеше аласыз:
