# {heading(Сервис туралы)[id=event-log-about]}

{include(/kz/_includes/_translated_by_ai.md)}

Cloud Audit сервисі ақпараттық қауіпсіздік пен қауіпсіздік оқиғаларын басқару жүйесін (SIEM) {var(cloud)} платформасымен біріктіруге арналған. Интеграцияның мақсаты — бұлттық сервистер мен {var(cloud)} виртуалды машиналарынан келіп түсетін қауіпсіздік оқиғалары туралы хабарламаларды SIEM жүйесіне орталықтандырылған түрде жинауды, сақтауды және беруді ұйымдастыру. Сервис басқару қабатының да, деректер қабатының да оқиғаларын жинауға мүмкіндік береді.

Cloud Audit сервисі мыналарға көмектеседі:

- бұлттық сервистерді пайдаланудың ашықтығын арттыруға;
- құпия деректермен жасалған әрекеттер тарихын бұлтта сенімді сақтауға;
- пайдаланушылардың деректермен әрекеттерін талдауға және осы негізде сезімтал ақпаратқа қолжетімділікті шектеу жүйесін жетілдіруге;
- инциденттерді жедел тергеуге.

{ifdef(public)}
Оқиғалар туралы жиналған ақпарат {var(cloud)} жеке кабинетінде тек [рөлдері](/kz/access/iam/concepts/rolesandpermissions) бар пайдаланушыларға қолжетімді:

- жоба иесі;
- суперадминистратор;
- жоба администраторы;
- бақылаушы.
{/ifdef}

Оқиғалар журналымен жұмыс істеу үшін Cloud Audit оқиғалар туралы ақпаратты іздеу, сүзгілеу және экспорттау {linkto(../../instructions/view-event-log#event-log-manage)[text=құралдарын]} ұсынады.
