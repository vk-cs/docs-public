Cloud Audit — ақпараттық қауіпсіздік пен қауіпсіздік оқиғаларын басқару жүйеңізді (SIEM) VK Cloud платформасымен интеграциялауға арналған сервис. Интеграцияның мақсаты — VK Cloud бұлттық сервистері мен виртуалды машиналарынан келіп түсетін қауіпсіздік оқиғалары туралы хабарламаларды жинау, сақтау және SIEM жүйесіне жіберу.
