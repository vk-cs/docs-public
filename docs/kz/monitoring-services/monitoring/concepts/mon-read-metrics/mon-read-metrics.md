# {heading(Метрикалардың функциялары мен топтаулары)[id=monitoring-mon-read-metrics]}

{include(/kz/_includes/_translated_by_ai.md)}

Cloud Monitoring метрика мәндерін алу үшін API ұсынады. Сұраулар негізінде метрикалармен жұмыс істеуге арналған PromQL тілінің ішкі жиынын қолдау іске асырылған.

Метрикаларды оқу сұрауын келесі синтаксиспен құрастыруға және орындауға болады.

- Метрика атауы және агрегаттау функциясы:

  - `cpu:Minimum`

- `Label` мәні бойынша сүзу:

  - `\=` — көрсетілген жолға дәл сәйкес келетін белгілер;
  - `!=` — көрсетілген жолға сәйкес келмейтін белгілер;
  - `\=~` — тұрақты өрнегі көрсетілген жолға сәйкес келетін белгілер;
  - `!~` — тұрақты өрнегі көрсетілген жолға сәйкес келмейтін белгілер.

- Топтаулар:

  - `SUM` — өлшемдер бойынша қосынды;
  - `MIN` — өлшемдер бойынша минимум;
  - `MAX` — өлшемдер бойынша максимум;
  - `AVG` — өлшемдер бойынша орташа мән;
  - `STDDEV` — өлшемдер бойынша жиынтықтың стандартты ауытқуы;
  - `STDVAR` — өлшемдер бойынша бас жиынтықтың стандартты дисперсиясы;
  - `COUNT` — вектордағы элементтер саны;
  - `QUANTILE` — өлшемдер бойынша φ-квантиль (`0 ≤ φ ≤ 1`).

Сұрау мысалдары:

```promql
SUM BY(host) (cpu:Minimum{instance="server1", app!="system"})
```

```promql
SUM BY(job) (cpu:Average{host="server1", job!="system"}[12h] offset 24h)
```

Сұрау тілінің толық синтаксисі мен мүмкіндіктері [Prometheus ресми құжаттамасында](https://prometheus.io/docs/prometheus/latest/querying/basics/) сипатталған.
