{include(/kz/_includes/_translated_by_ai.md)}

Cloud Monitoring — бірыңғай мониторинг сервисі, оның көмегімен сіз бұлттық қолданбаларыңыздың күйін және өнімділігін бақылайсыз, қолданбалардың тұрақты жұмысы үшін ресурс тұтынуын оңтайландырасыз. Сервис жоғары жүктемелі жүйелерге оңтайландырылған және метрикалар мен журналдарды икемді визуализациялауды ұсынады. Сервисте түсінікті интерфейс пен Prometheus-ке ұқсас сұраулар тілі пайдаланылады.
