Cloud Alerting — {var(cloud)} ресурстарынан жиналатын метрикалардағы өзгерістер туралы хабарландыру сервисі.

Сіз бұлттық ресурстардың күйі туралы хабарландыруларды өзіңіз баптап, ала аласыз. Мысалы, виртуалды машинаңыздың немесе дерекқорыңыздың күйінің өзгеруі туралы хабар алып, мәселеге дер кезінде әрекет ете аласыз. Соның арқасында қосымшаларыңыздың жұмысындағы ақауды кейін біліп қоймай, оның алдын ала аласыз.

Cloud Alerting белгілі бір метрикалардағы өзгерістерді триггер ережелері арқылы қадағалап, хабарландыру арналарымен хабарламалар жібереді.
