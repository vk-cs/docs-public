Cloud Direct Connect сервисін қосу үшін байланыс арнасын ұйымдастыру керек.

Байланыс арнасын ұйымдастырудың екі нұсқасы бар:

- Байланыс арнасын өзіңіз ұйымдастырасыз.
- Байланыс арнасын VK Cloud командасы ұйымдастырады.

{tabs}

{tab(Өздігінен ұйымдастыру)}

1. Техникалық талаптармен және шектеулермен танысыңыз:

    {include(/kz/_includes/_requirements.md)}

1. Байланыс арнасының параметрлерін таңдаңыз:

    {include(/kz/_includes/_prerequisites.md)}

1. Өз telecom-серіктесіңізге хабарласып, қажетті технологияны пайдаланып, VK Cloud дейін L2 байланыс арнасын ұйымдастырыңыз. Арна дата-орталық сөрелеріне дейінгі кроссировкаларды қоса алғанда құрылады.

1. [өтінім жасаңыз](https://cloud.vk.com/direct-connect) Direct Connect қосуға.
1. После подключения сервиса [настройте маршрутизацию](../how-to-guides/advanced-router) Сервис қосылғаннан кейін VK Cloud жобаңызда L3 Static / BGP.

{/tab}

{tab(VK Cloud командасы ұйымдастырады)}

{note:warn}

Байланыс арналарын ұйымдастыру жұмыстары үшін үшінші тараптарды тарту қажет және олар ақылы негізде орындалады.

{/note}

1. Техникалық талаптармен және шектеулермен танысыңыз:

    {include(/kz/_includes/_requirements.md)}

1. Байланыс арнасының параметрлерін таңдаңыз:

    {include(/kz/_includes/_prerequisites.md)}

1. [өтінім жасаңыз](https://cloud.vk.com/direct-connect/) және инфрақұрылымыңыздың қатысу нүктелерінің нақты координаттары (қосылыс қайда тартылады), қосылу түрі мен жылдамдығы туралы ақпаратты беріңіз.
1. VK Cloud өкілі жоба мен қосылу құнын келісу үшін сізбен байланысқанша күтіңіз.
1. VK Cloud өкілі VK Cloud инфрақұрылымындағы байланыс арналары қосылғаны туралы сізге хабарлағанша күтіңіз.
1. [Настройте маршрутизацию](../how-to-guides/advanced-router) VK Cloud жобаңызда L3 Static / BGP.
