Қауіпсіздік тобын порттан ажырату интерфейсте оның атын өзгерту мүмкіндігіне әсер етпеуі керек.

### Шешім

Порт атауын OpenStack CLI арқылы өзгертіңіз:
```console
openstack port set --name <НОВОЕ_ИМЯ> <ID_ПОРТА>
```

Егер мәселе сақталса, [техникалық қолдауға хабарласыңыз](https://cloud.vk.com/contacts/).
