Желі топологиясы — виртуалды машиналардың желілік байланысының сызбасы. Онда келесі элементтер, сондай-ақ олардың арасындағы барлық байланыстар көрсетіледі:

- желілер;
- ішкі желілер;
- маршруттауыштар;
- виртуалды машиналар.

## {heading(Топологияны басқару)[id=vnet-topology-manage]}

Желі топологиясымен қолжетімді операциялар:

- *Жылжыту*. Оң жақ жоғарғы бұрыштағы бағдаршаларды пайдаланыңыз немесе тінтуірдің сол жақ батырмасын басып тұрып, сызбаны қажетті бағытқа жылжытыңыз.
- *Масштабтау*. Сызбаның оң жақ жоғарғы бұрышындағы **+** және **-** белгілерін пайдаланыңыз немесе тінтуірдің доңғалағын айналдырыңыз.
- *Жазуларды жасыру немесе көрсету*. Сызбаның сол жақ жоғарғы бұрышындағы **Жазуларды көрсету** ауыстырғышын пайдаланыңыз.
- *Элемент туралы толық ақпаратты қарау*. Сызба элементін тінтуірдің сол жақ батырмасымен басыңыз. Келесі параметрлер көрсетіледі:

  - **Атауы** — таңдалған элементтің атауы.
  - **ID** — жүйедегі элементтің идентификаторы.
  - **Түрі** — таңдалған элементтің түрі:

    - network — желі;
    - subnet — ішкі желі;
    - instance — инстанс;
    - router — маршруттауыш;
    - balancer — жүктеме теңгергіші;
    - port — порт.

  - **Күйі** — таңдалған элементтің күйі. Ішкі желі үшін көрсетілмейді. <!-- todo заполнить возможные статусы-->
  - Элемент түріне байланысты қосымша параметрлер.
  - Жеке кабинеттегі элементке өту сілтемесі. Порттар үшін көрсетілмейді.
