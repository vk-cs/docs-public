# {heading(Жетілдірілген маршрутизаторлар)[id=vnet-advanced-router]}

{include(/kz/_includes/_translated_by_ai.md)}

Виртуалды желілер сервисіндегі жетілдірілген маршрутизаторларды басқару. Жетілдірілген маршрутизаторлар тек {linkto(../../../../networks/vnet/concepts/sdn#vnet-sdn)[text=SDN Sprut]} қосылған жобаларда ғана қолжетімді. Жобаңызда қай SDN қолданылатынын білу немесе басқасын қосу үшін [техникалық қолдау қызметіне](/kz/contacts) хабарласыңыз.
