VPN (виртуалды жеке желі) — басқа желілердің үстінен бір немесе бірнеше қорғалған желілік қосылымды қамтамасыз етуге мүмкіндік беретін технология.

Қолданылатын хаттамаларға байланысты VPN мынадай мүмкіндіктерге ие болуы мүмкін:

- соңғы нүктелердің әртүрлі түрлерін қосу (торап-торап, торап-желі, желі-желі);
- деректердің қорғалу деңгейі әртүрлі болуы;
- әртүрлі мақсаттарда пайдаланылуы (пайдаланушының корпоративтік желілерге қол жеткізуі, қашықтағы желілерді бір инфрақұрылымға біріктіру және т.б.).

VK Cloud ішінде VPN сервисі (VPNaaS) келесі міндеттерді шешу үшін VK Cloud бір немесе бірнеше ішкі желісі мен клиенттің қашықтағы инфрақұрылымының (on-premises) ішкі желілері арасында қорғалған байланыс туннелін құруға мүмкіндік береді:

- Бұлттық инфрақұрылымға қол жеткізу үшін компания желісін VK Cloud желісіне қосу.
- VK Cloud инфрақұрылымын басқаруға арналған қорғалған арнаны ұйымдастыру (мысалы, виртуалды машиналармен бірге telnet пайдалану үшін).

VK Cloud ішіндегі VPN сервисі IPsec және IKEv1, IKEv2 стегін пайдалана отырып [StrongSwan](https://www.strongswan.org) негізінде іске асырылған. Келесі параметрлерге қолдау көрсетіледі:

- Жұмыс режимдері: IKEv1, IKEv2 (ұсынылады).
- Аутентификация: Pre-Shared Key (PSK).
- Диффи-Хеллман топтары: modp1024 (Group 2), modp1536 (Group 5), modp2048 (Group 14).
- Шифрлау алгоритмдері: AES-128, AES-192, AES-256, 3DES.
- Хэштеу: SHA1, SHA256.
- SA өмір сүру уақыты: 3600 секунд (әдепкі бойынша).

VPN-сервис VK Cloud ішінде пайдаланылатын екі [SDN](../sdn) (Software Defined Network) түрінде де қолжетімді:

- SDN Neutron: VPN-ді тек [стандартты маршрутизаторға](../router#standard).
- SDN Sprut: VPN-ді тек [кеңейтілген маршрутизаторға](../router#advanced).

SDN Sprut ішінде бұлт пен қашықтағы инфрақұрылым арасында VPN-ді интернет-қосылым үстінен немесе [Direct Connect](/kz/networks/directconnect). үстінен құруға болады. Direct Connect үстіндегі VPN қашықтағы инфрақұрылымнан интернетке қол жеткізуді талап етпейді және ең тұрақты байланысты қамтамасыз етеді.
