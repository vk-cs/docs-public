# {heading(CDN-ресурс)[id=cdn-resource]}

{include(/kz/_includes/_translated_by_ai.md)}

CDN-ресурс — бұл VK Cloud ұсынатын {linkto(../../../../networks/cdn/concepts/about#cdn-about)[text=CDN сервисінің]} логикалық мәні, ол {linkto(../../../../networks/cdn/concepts/origin-groups#cdn-about)[text=бастапқы көздерден]} контентті CDN арқылы тарату баптауларын қамтиды және осы баптауларды өзгертуге мүмкіндік береді.

Пайдаланушыларға контентті CDN арқылы жеткізуді бастау үшін {linkto(../../../../networks/cdn/instructions/create-resource#cdn-create-resource)[text=ресурс құру]} қажет. Құру кезінде мыналарды баптауға болады:

- Пайдаланушылар үшін контенттің қолжетімділігін және ресурстың бастапқы көзбен өзара әрекеттесу протоколын;
- Бастапқы көздер мен домендер конфигурациясын;
- SSL сертификатының көмегімен шифрлау баптауларын.

Ресурс құрылғаннан кейін, сіз сондай-ақ мыналарды баптай аласыз:

- {linkto(../../../../networks/cdn//manage-cdn/origin-settings#cdn-origin-settings)[text=бастапқы көзбен өзара әрекеттесуді]} — өзара әрекеттесу протоколын, контент бастапқы көздерін қосу мен өзгертуді, Host тақырыптарын баптауды және shielding-ті қосуды;
- {linkto(../../../../networks/cdn//manage-cdn/caching#cdn-caching)[text=кэштеуді]};
- {linkto(../../../../networks/cdn//manage-cdn/http-headers#cdn-http-headers)[text=HTTP тақырыптарын, HTTP сұрау әдістерін және CORS-ты]};
- {linkto(../../../../networks/cdn//manage-cdn/content-settings#cdn-content-settings)[text=контентті басқаруды]} — ресурс кэшін тазарту және толтыру, HTTP жауап кодтары, контентті сығу және үлкен файлдарды жеткізуді оңтайландыру;
- {linkto(../../../../networks/cdn//manage-cdn/security#cdn-security)[text=қауіпсіздік параметрлерін]} — елдер, домендер және IP мекенжайлар бойынша қол жеткізу саясаттарын, сондай-ақ қолданылатын TLS протоколы нұсқаларын;
- Secure token технологиясының көмегімен {linkto(../../../../networks/cdn//manage-cdn/secure-token#cdn-secure-token)[text=контентке қол жеткізуді шектеуді]}.

## {heading(Ресурс күйлері)[id=cdn-resource-statuses]}

CDN-ресурс келесі күйлердің бірінде болуы мүмкін:

- `Белсенді` — контент соңғы пайдаланушыларға жеткізіледі;
- `Тоқтатылған` — контент соңғы пайдаланушыларға қолжетімсіз.

Ресурстың күйі [жеке кабинеттегі](https://mcs.mail.ru/app) CDN ресурстары тізімінде белгіше түрінде көрсетіледі, оның көмегімен сіз CDN-ресурсты өзіңіз {linkto(../../../../networks/cdn//manage-cdn/enable-cdn#cdn-enable-cdn)[text=қосып немесе өшіре]} аласыз.

Егер белгілі бір уақыт ішінде ресурс арқылы елеулі трафик өтпесе, CDN-ресурстар автоматты түрде тоқтатылады:

- Егер ресурс 30 күннен аз уақыт бұрын жасалған болса, құрылған сәттен бастап 1 МБ-тан аз трафик;
- Егер ресурс 30 күннен астам уақыт бұрын жасалған болса, соңғы 14 күнде 1 МБ-тан аз трафик.

Тоқтатылған CDN-ресурс өзінің баптауларын сақтайды және оны қайтадан {linkto(../../../../networks/cdn//manage-cdn/enable-cdn#cdn-enable-cdn)[text=қосуға]} болады.
