CDN-ресурс үшін, тіпті контенттің бір ғана бастауы қолданылса да, әрқашан _бастауыштар тобы_ бапталады. Контент сұратылған кезде CDN сервері бастауды бастауыштар тобы деңгейінде берілген опцияларға байланысты таңдайды. Таңдалған бастаудың жауаптарына қарай басқа бастауыш таңдалуы мүмкін.

Келесі опциялар CDN серверлері контентті сұратқан кезде топтан бастауышты таңдау ретіне әсер етеді:

- **Бастауыш түрі**: белсенді немесе резервтік.

  Топтағы жеке бастауыш деңгейінде беріледі.

- **Бастауышта 4XX және 5XX кодтары болғанда тізімнен келесі бастауышты пайдалану**.

  Бүкіл бастауыштар тобы деңгейінде беріледі. Параметр топта бірден көп бастауыш бапталған болса, қолжетімді болады.

Бастауышты таңдау келесі ережелер бойынша орындалады:

- Егер **Бастауышта 4XX және 5XX кодтары болғанда тізімнен келесі бастауышты пайдалану** опциясы таңдалмаса, онда CDN сервері контентті алдымен белсенді бастауыштардың бірінен, содан кейін резервтік бастауыштардың бірінен сұратады.

  CDN сервері контентті резервтік бастауыштардың бірінен тек белсенді бастауыштан 5XX ауқымындағы HTTP мәртебесімен жауап келген жағдайда ғана сұратады. Белсенді немесе резервтік бастауыштан 4XX ауқымындағы HTTP мәртебесімен жауап келгенде CDN сервері тұтынушыға қатені қайтарады.

  Жүктемені теңгеруді қамтамасыз ету үшін белсенді бастауыш round-robin алгоритмі бойынша таңдалады.

- Егер **Бастауышта 4XX және 5XX кодтары болғанда тізімнен келесі бастауышты пайдалану** опциясы таңдалса, онда CDN сервері контентті бастауыштар тізімі бойынша қозғалып сұратады.

  Егер тізімдегі барлық бастауыштар белсенді болса, онда CDN сервері контентті бірінші белсенді бастауыштан сұратады [404](https://developer.mozilla.org/en-US/docs/Web/HTTP/Status/404), [500](https://developer.mozilla.org/en-US/docs/Web/HTTP/Status/500), [502](https://developer.mozilla.org/en-US/docs/Web/HTTP/Status/502), [503](https://developer.mozilla.org/en-US/docs/Web/HTTP/Status/503) немесе [504](https://developer.mozilla.org/en-US/docs/Web/HTTP/Status/504) HTTP мәртебелерімен жауап келгенде CDN сервері тізім бойынша төмен жылжып, контентті қалған бастауыштардан сұратады.
  
  Егер тізімде резервтік бастауыштар болса, онда CDN сервері контентті бірінші белсенді бастауыштан сұратады. 4XX немесе 5XX ауқымындағы HTTP мәртебесімен жауап келгенде CDN сервері олардың түріне қарамастан тізім бойынша төмен жылжып, контентті қалған бастауыштардан сұратады.

  Егер барлық бастауыштар қолжетімсіз болса, CDN сервері тізімдегі соңғы бастауыштың жауабын қайтарады.
