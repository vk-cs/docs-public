# {heading(Secure token)[id=cdn-secure-token]}

{include(/kz/_includes/_translated_by_ai.md)}

*Secure token* — бұл CDN-ресурс файлдарын қалаусыз жүктеп алудан қорғауға мүмкіндік беретін, {linkto(../../../../networks/cdn/instructions/manage-cdn/secure-token#cdn-secure-token)[text=жеке кабинетте қосылатын]} опция.
Secure token көмегімен контентке уақытша қолжетімділікті ұйымдастырып, деректеріңіздің қауіпсіздігін қамтамасыз етуге болады.

Егер Secure token опциясы қосулы болса, CDN арқылы таратылатын файлдарға қолжетімділік арнайы кодты (secure token) қамтитын қолтаңбаланған сілтемелер арқылы ғана жүзеге асырылады. Пайдаланушы сілтеме бойынша өткен кезде CDN алынған secure token-ді ресурс құпия кілтінен және сұрау параметрлерінен есептелген хэшпен салыстырады да, нәтижесіне қарай контентке қолжетімділікке рұқсат береді немесе тыйым салады.

Контенттің бір бөлігіне кәдімгі сілтемелер арқылы қолжетімділікті ұйымдастыру үшін, ол үшін бөлек {linkto(../../../../networks/cdn/instructions/create-resource#cdn-create-resource)[text=ресурс жасаңыз]} және жеке {linkto(../../../../networks/cdn/concepts/origin-groups#cdn-origin-groups)[text=бастауы]} болсын.

Қолтаңбаланған сілтеме келесі түрде болады:

```plaintext
http://cdn.example.com/files/image.jpg?md5=CUQ1rzAvtQCxwLS&expires=1301605293
```

Мұнда:

- `md5=CUQ1rzAvtQCxwLS` — мына мәндерді қамтитын жолдың md5-хэші түріндегі secure token:

  - құпия кілт;
  - сілтеменің әрекет ету мерзімі;
  - бастаудағы файлға жол;
  - (қалауыңызша) рұқсат етілген IP мекенжайы.

- `expires=1301605293` — [Unix-уақыт](https://www.unixtimestamp.com) пішіміндегі сілтеменің әрекет ету мерзімі.

Қолтаңбаланған сілтемені осы мақсат үшін алдын ала арнайы скрипт қосылуы тиіс сайтыңыз жасайды.

## {heading(Скрипт мысалдары)[id=cdn-secure-token-scripts]}

### {heading(IP мекенжайы бойынша қолжетімділікті шектеусіз қолтаңбаланған сілтеме жасауға арналған скрипттер)[id=cdn-secure-token-standardscripts]}

{tabs}
{tab(PHP)}
```php
<?php

$secret = '<СЕКРЕТНЫЙ_КЛЮЧ>';
$path = '<ПУТЬ_К_ФАЙЛУ>';
$expires = time() + <СРОК_ДЕЙСТВИЯ_ССЫЛКИ>;
$hostname = '<ДОМЕННОЕ_ИМЯ>';
$link = "$expires$path $secret";
$md5 = md5($link, true);
$md5 = base64_encode($md5);
$md5 = strtr($md5, '+/', '-_');
$md5 = str_replace('=', '', $md5);
$url = "{$hostname}{$path}?md5={$md5}&expires={$expires}";

?>
```

Мұнда:

- `<СЕКРЕТНЫЙ_КЛЮЧ>` — ұзындығы 6-дан 32 таңбаға дейінгі символдардың еркін жиыны.
- `<ПУТЬ_К_ФАЙЛУ>` — сілтеме арқылы қолжетімділік қажет болатын файлға жол. Мысал: `/files/photo.jpg`.
- `<СРОК_ДЕЙСТВИЯ_ССЫЛКИ>` — сілтеме жасалған сәттен бастап секундпен көрсетілген оның әрекет ету мерзімі.
- `<ДОМЕННОЕ_ИМЯ>` — `HTTP` немесе `HTTPS` протоколы көрсетілген CDN-ресурстың домендік атауы. Мысал: `https://cdn.example.com`.

`$url` жолы файлға арналған дайын қолтаңбаланған сілтемені қамтиды. Сілтемені экранға шығару үшін осы жолдан кейін келесі кодты қосыңыз:

```php
echo $url;
echo "
";
```

{tab(Python)}
```python
import base64 
from hashlib import md5 
from time import time 
secret = '<СЕКРЕТНЫЙ_КЛЮЧ>'
path = f'<ПУТЬ_К_ФАЙЛУ>'
```

- `<ДОМЕННОЕ_ИМЯ>` — `HTTP` немесе `HTTPS` протоколы көрсетілген CDN-ресурстың домендік атауы. Мысал: `https://cdn.example.com`.

`secured_url` жолы файлға арналған дайын қолтаңбаланған сілтемені қамтиды. Сілтемені экранға шығару үшін осы жолдан кейін келесі кодты қосыңыз:


{/tab}
{tab(OpenSSL)}
```bash
#!/bin/bash
# Generate a pre-signed link without IP address restriction
let "EXPIRES=$(date +%s) + <СРОК_ДЕЙСТВИЯ_ССЫЛКИ>"
HOSTNAME="<ДОМЕННОЕ_ИМЯ>"
FILEPATH="<ПУТЬ_К_ФАЙЛУ>"
SECRET="<СЕКРЕТНЫЙ_КЛЮЧ>"
TOKEN=$(echo -n $EXPIRES$FILEPATH' '$SECRET | openssl md5 -binary | openssl base64 | tr +/ -_ | tr -d = )
echo $HOSTNAME$FILEPATH'?md5='$TOKEN'&expires='$EXPIRES
```

- `<СЕКРЕТНЫЙ_КЛЮЧ>` — ұзындығы 6-дан 32 таңбаға дейінгі символдардың еркін жиыны.
- `<ПУТЬ_К_ФАЙЛУ>` — сілтеме арқылы қолжетімділік қажет болатын файлға жол. Мысал: `/files/photo.jpg`.
- `<СРОК_ДЕЙСТВИЯ_ССЫЛКИ>` — сілтеме жасалған сәттен бастап секундпен көрсетілген оның әрекет ету мерзімі.
- `<ДОМЕННОЕ_ИМЯ>` — `HTTP` немесе `HTTPS` протоколы көрсетілген CDN-ресурстың домендік атауы. Мысал: `https://cdn.example.com`.
