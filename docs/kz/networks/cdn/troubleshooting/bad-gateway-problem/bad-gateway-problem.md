CDN ресурсына жүгінген кезде `502 Bad Gateway`.

Бұл қате CDN сервері сіздің көзіңізден дұрыс жауап ала алмағанын білдіреді.

### Шешім

1. Көздің интернеттен қолжетімділігін тексеріңіз. Көз тарапынан VK Cloud CDN серверлерінің IP мекенжайларына қолжетімділік шектелмегеніне көз жеткізіңіз. IP мекенжайларының өзекті тізімін [техникалық қолдаудан](/kz/contacts).
1. CDN баптауларында дереккөздің дұрыс IP мекенжайы немесе домені көрсетілгеніне [көз жеткізіңіз](/kz/networks/cdn/instructions/manage-cdn/origin-settings#derekter_kozin_ozgertu).
1. [Тексеріңіз](/kz/networks/cdn/instructions/manage-cdn/http-headers#zhauap_takyryptaryn_kosu) ол сіздің веб-серверіңіз күтетін мәнге сәйкес келуі керек). 
1. Көздегі брандмауэрді тексеріңіз.

Егер мәселе сақталса, [техникалық қолдауға хабарласыңыз](/kz/contacts).
