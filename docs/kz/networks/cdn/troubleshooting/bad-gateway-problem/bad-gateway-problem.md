CDN ресурсына жүгінген кезде `502 Bad Gateway` қатесі пайда болады.

Бұл қате CDN сервері сіздің көзіңізден дұрыс жауап ала алмағанын білдіреді.

### Шешім

1. Көздің интернеттен қолжетімділігін тексеріңіз. Көз тарапынан {var(cloud)} CDN серверлерінің IP мекенжайларына қолжетімділік шектелмегеніне көз жеткізіңіз. IP мекенжайларының өзекті тізімін [техникалық қолдаудан](/kz/contacts) сұратуға болады.
1. CDN баптауларында көздің дұрыс IP мекенжайы немесе домені көрсетілгеніне [көз жеткізіңіз](/kz/networks/cdn/instructions/manage-cdn/origin-settings#cdn-origin-settings-data-source). 
1. Host тақырыбының баптауларын [тексеріңіз](/kz/networks/cdn/instructions/manage-cdn/http-headers#cdn-http-headers-add-request) (ол сіздің веб-серверіңіз күтетін мәнге сәйкес келуі керек). 
1. Көздегі брандмауэрді тексеріңіз.

Егер мәселе сақталса, [техникалық қолдауға хабарласыңыз](/kz/contacts).
