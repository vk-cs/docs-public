1. [Өтіңіз](https://kz.cloud.vk.com/app) VK Cloud жеке кабинетіне.
1. **Қолданбалар дүкені** бөліміне өтіңіз.
1. Қажетті сервистің карточкасында **Толығырақ** түймесін басыңыз.
1. Сервистің сипаттамасымен танысыңыз. [image-based](../../concepts/about#servis_turleri) сервисін орнату кезінде жобада квоталар жеткілікті екеніне көз жеткізіңіз.
1. **Қосу** түймесін басыңыз.
1. Тарифтік жоспарды конфигурациялау шеберінің қадамдарын орындаңыз.

   {note:info}

   Қадамдар [әртүрлі сервистер](../../initial-configuration) үшін өзгеше болуы мүмкін.

   {/note}

1. Сервисті қосуды растаңыз.

   Орнатудың аяқталуын күтіңіз — поштаға сервистімен жұмыс істеуге арналған деректемелері бар бір реттік сілтеме келеді. Жұмысты бастамас бұрын оларды сақтап қойыңыз.

{note:info}

Сервистермен жұмысты бастау жөніндегі нұсқаулықтар [Қолданбалар дүкені сервистерін баптау](../../initial-configuration) бөлімінде келтірілген.

{/note}
