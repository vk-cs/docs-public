Marketplace — клиенттерге әртүрлі үшінші тарап сервистеріне қолжетімділік беретін VK Cloud ішкі жүйесі:

- вендор ұсынатын;
- opensource-шешімдер;
- меншікті әзірлеме сервистері.

Инфрақұрылым мен БҚ-ны өрістету немесе тенантты (аккаунтты) жасау жөніндегі барлық егжей-тегжейлер VK Cloud жағында қалады. Пайдаланушы сервисінің дайындалған данасын (сервис инстансын) алады, оны қажет болған жағдайда өз қажеттіліктеріне қарай қосымша баптай алады. Өрістетілген барлық сервис инстанстары VK Cloud жобасы аясында [тарификацияланады](../../tariffication) және [басқарылады](../../instructions/pr-instance-manage). Қолданбалар дүкені сервистерді жылдамырақ іске қосуға көмектеседі және оларды қолдауды жеңілдетеді.

{note:warn}

Қолданбалар дүкеніндегі сервистерді пайдалана отырып, сіз [пайдаланушы келісімімен](/kz/start/legal/digital-cloud/marketplace) келісесіз.

{/note}

## {heading(Сервис түрлері)[id=marketplace-about-types]}

VK Cloud сервистердің екі түрін қолдайды:

- SaaS сервистері — орталықтандырылған орнатылған [мультитенантты](https://habr.com/kz/companies/microsoft/articles/145027) өнімдер. Вендор сервисті не меншікті инфрақұрылымында, не VK Cloud-тағы өз жобасының инфрақұрылымында өрістетеді. Пайдаланушыға сервиске жеке аккаунт (тенант) арқылы қолжетімділік беріледі. Сервисті және оның инстанстарын басқару VK Cloud жағында орындалады.
- Image-based сервистері — VK Cloud жобасында виртуалды машиналар образдарының негізінде өрістетілетін өнім. Өнімді қолдау үшін қосымша инфрақұрылым пайдаланылуы мүмкін: виртуалды желілер, жүктеме теңгергіштері, DBaaS кластерлері, S3 объектілік қоймасы, резервтік көшіру. Сервисті, оның инстанстарын, сондай-ақ инфрақұрылымды басқару VK Cloud жағында орындалады.

Әр сервис түрі VK Cloud-та әртүрлі жолмен инициализацияланады:

{tabs}

{tab(SaaS)}

1. Пайдаланушы сервисті жобаға [қосады](../../instructions/pr-instance-add).
1. VK Cloud жеткізушіге оның өнімінде аккаунт-тенант жасауға сұрау жібереді.
1. Жеткізуші жаңа аккаунтты тіркейді, реквизиттерді VK Cloud-қа жібереді.
1. VK Cloud [басқару](../../instructions/pr-instance-manage) үшін сервис инстансын жасайды және қолжетімділік реквизиттерін пайдаланушыға жібереді.

{/tab}

{tab(Image-based)}

1. Пайдаланушы сервисті жобаға [қосады](../../instructions/pr-instance-add).
1. VK Cloud пайдаланушы жобасында қажетті инфрақұрылымды жасайды.
1. VK Cloud құрылған инфрақұрылымға сервисті орнатады.
1. VK Cloud [басқару](../../instructions/pr-instance-manage) үшін сервис инстансын жасайды және қолжетімділік реквизиттерін пайдаланушыға жібереді.

{/tab}

{/tabs}
