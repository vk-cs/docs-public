Milvus — құрылымдалмаған деректерді талдауға арналған икемді платформа. Оның көмегімен үлкен дерекқордағы ұқсас векторларды жылдам таба аласыз. Бұл әсіресе жасанды интеллект, компьютерлік көру, табиғи тілді өңдеу және ұсынымдық жүйелер салаларында өзекті.

Milvus мүмкіндіктері:

- векторлық деректердің үлкен көлемдерін сақтау және басқару;
- жоғары өлшемді кеңістіктерде ең жақын көршілерді (ANN) жылдам салыстыру және іздеу;
- TensorFlow, PyTorch және ONNX сияқты танымал машиналық оқыту фреймворктарымен интеграция;
- векторлық іздеуге қосымша атрибуттар бойынша сүзгілеу мен сұрыптаудың ыңғайлы тетіктері;
- икемді көлденең масштабтауды және таратылған сақтауды қолдау.

Milvus түрлі тілдерге арналған RESTful API мен клиенттік кітапханаларды ұсынады, бұл қолданбаларыңызбен интеграцияны жеңілдетеді, сондай-ақ Milvus-пен өзара әрекеттесуге арналған веб-интерфейс — Attu құрамына кіреді.

Milvus пайдалану арқылы сіз [Marketplace](/kz/start/legal/vk/marketplace) және [Milvus](https://github.com/milvus-io/milvus/blob/master/LICENSE) лицензиялық келісімдерімен келісесіз.

VK Cloud жобасында Milvus сервисін өрістету үшін:

1. [Тіркеліңіз](/kz/intro/onboarding/account) және VK Cloud [жеке кабинетіне](https://kz.cloud.vk.com/app) өтіңіз.
1. Өрістетілген сервисі бар ВМ-ге және интернеттен Milvus веб-интерфейсіне қол жеткізу үшін:

    1. Егер желі бұрын жасалмаған болса, интернетке қолжетімді [желіні жасаңыз](/kz/networks/vnet/instructions/net#zhelini_zhasau).
    1. Өрістетілген сервисі бар ВМ орналастырылатын [ішкі желі баптауларында](/kz/networks/vnet/instructions/net#zhelini_redakciyalau) **Жекеменшік DNS** опциясын өшіріңіз.

1. **Қолданбалар дүкені** бөліміне өтіңіз.
1. **Milvus** сервисі карточкасында **Толығырақ** түймесін басыңыз.
1. Сервис сипаттамасы бар бетте **Қосу** түймесін басыңыз.
1. **Сервер параметрлері** бетінде деректерді енгізіңіз.
    1. Milvus параметрлерін көрсетіңіз:

        - **Milvus қалай орналастырылады**: сервисті орналастыру түрін таңдаңыз.

            - `external` — VK Cloud ұсынған сыртқы IP-мекенжайымен және домендік атаумен. Интернеттен өрістетілген Milvus сервисі бар ВМ-ге қол жеткізу үшін осы орналастыру түрін таңдаңыз.
            - `internal` — тек ішкі IP-мекенжай арқылы қолжетімділікпен.

        - **S3-ке сақтық көшірмелеуді қосу**: [VK Cloud объектілік қоймасына](/kz/storage/s3/concepts/about) Milvus деректерінің күн сайынғы сақтық көшірмелерін жасау үшін опцияны қосыңыз. Қойма автоматты түрде жасалады. Соңғы 7 көшірме сақталады.

        - **Мониторингті қосу**: Milvus өрістетілген виртуалды машинаның метрикаларын [Cloud Monitoring](/kz/monitoring-services/monitoring) сервисіне автоматты түрде жіберу үшін опцияны қосыңыз. Бұл метрикалар жеке кабинеттің **Мониторинг** бөлімінде қолжетімді болады.

    1. Сервис инстансы өрістетілетін виртуалды машинаның параметрлерін көрсетіңіз:

        - **Желі**: өрістетілген сервисі бар ВМ орналастырылатын желіні таңдаңыз. Егер `external` орналастыру түрін көрсетсеңіз, интернетке қолжетімді желіні және **Жекеменшік DNS** опциясы өшірілген ішкі желіні таңдаңыз.
        - **Қолжетімділік аймағы**: ВМ іске қосылатын [ДОД](/kz/start/concepts/architecture#az) таңдаңыз.
        - **Виртуалды машина түрі**: [алдын ала орнатылған ВМ конфигурациясын](/kz/computing/iaas/concepts/vm/flavor) таңдаңыз.

    1. Жүйелік диск пен деректер дискісінің параметрлерін көрсетіңіз.

        - **Диск өлшемі**: ВМ дискісінің қажетті көлемін гигабайтпен көрсетіңіз.
        - **Диск түрі**: [диск түрлерінің бірін](/kz/computing/iaas/concepts/data-storage/disk-types#disk_types) таңдаңыз — `HDD`, `SSD` немесе `High-IOPS SSD`.

1. **Келесі қадам** түймесін басыңыз.
1. Сервисті өрістетуге арналған инфрақұрылым құнымен танысып, **Тарифті қосу** түймесін басыңыз.

    Бұлттық инфрақұрылымды өрістету басталып, сервис қасиеттері беті ашылады. Процесс мәртебесі сондай-ақ **Қолданбалар дүкені → Менің сервистерім** бөлімінде көрсетіледі.

    Сәтті өрістетуден кейін VK Cloud-тағы аккаунтыңызға байланған поштаға келесі хабарламалар келеді:

    - VK Cloud жеке кабинетіндегі сервис инстансына сілтемесі бар хабарлама;
    - сервиске қолжетімділік деректері бар бір реттік сілтемесі бар хат.

1. Хаттағы бір реттік сілтеме бойынша өтіп, сервиске қол жеткізу деректерін сақтаңыз:

    - `milvus_url` — Milvus веб-интерфейсіне сілтеме;
    - `milvus_default_user` — қосылуға арналған логин;
    - `milvus_root_pass` — қосылуға арналған құпиясөз;
    - `keypair` — сервис өрістетілген виртуалды машинаға SSH арқылы қосылуға арналған жабық кілт.

   {note:info}

   Егер қолжетімділік деректері жоғалса, жаңаларын [генерациялаңыз](../../instructions/pr-instance-manage#update_access).

   {/note}

1. `milvus_url` сілтемесі арқылы Milvus сервисінің веб-интерфейсіне өтіп, өрістерді толтырыңыз:

   * **Milvus Address** — әдепкі бойынша өрісте `<АДРЕС_MILVUS>:19515` түріндегі мән көрсетіледі, мұнда `<АДРЕС_MILVUS>` — өрістетілген Milvus инстансы бар ВМ-нің домені немесе мекенжайы, `19515` — қосылу порты.

     `<АДРЕС_MILVUS>` орнына таңдау бойынша келесі мәндердің бірін көрсетіңіз:

      * `milvus_url` сілтемесіндегі домендік атау ( `/attu` постфиксінсіз);
      * сервис өрістетілген ВМ-нің жария мекенжайы;
      * сервис өрістетілген ВМ-нің жеке мекенжайы.

     ВМ-нің жария және жеке мекенжайларын **Бұлттық есептеулер → Виртуалды машиналар** бөліміндегі ВМ бетінде, **Желілер** қойындысынан көруге болады.
     
     {note:warn}

     Порт нөмірін өзгертпеңіз.

     {/note}

   * **Authentication** — логин мен құпиясөз арқылы сервисте аутентификациядан өту үшін осы опцияны қосыңыз.
   * **token** — өрісті бос қалдырыңыз.
   * **username** —  сервис өрістетілген кезде алынған `milvus_default_user` параметрінің мәнін енгізіңіз.
   * **password** —  сервис өрістетілген кезде алынған `milvus_root_pass` параметрінің мәнін енгізіңіз.

   Сервиске қосылу үшін **Connect** түймесін басыңыз.

1. (Қалауыңызша) Сервиспен жұмысты бастау үшін [нұсқаулықты](https://milvus.io/bootcamp) пайдаланыңыз. Қажет болса, [Milvus ресми құжаттамасын](https://milvus.io/docs) оқып шығыңыз.
