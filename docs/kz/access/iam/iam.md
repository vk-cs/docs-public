# {heading(IAM)[id=iam]}

{include(/kz/_includes/_translated_by_ai.md)}

IAM (Identity and Access Management) — бұл VK Cloud пайдаланушыларының қолжетімділік құқықтарын икемді басқаруға мүмкіндік беретін идентификация мен қолжетімділікті басқару сервисі. Сервис пайдаланушылардың платформа сервистеріне қауіпсіз, ыңғайлы және бақыланатын қолжетімділігін қамтамасыз етеді және ресурстарға қатысты барлық операциялардың тек қажетті құқықтары бар пайдаланушылар арқылы орындалуына кепілдік береді.
