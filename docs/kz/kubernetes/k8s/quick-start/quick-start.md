# {heading(Жылдам бастау)[id=k8s-quick-start]}

{include(/kz/_includes/_translated_by_ai.md)}

Жылдам бастау сервиспен жұмысты бастауға және оның мүмкіндіктерімен танысуға көмектеседі.

Жылдам бастаудың барлық қадамдарынан өткен соң, сіз:

1. Шағын Kubernetes кластерін жасайсыз.
1. Оған қосылуды үйренесіз.
1. Kubernetes-пен және {linkto(../concepts/addons-and-settings/addons#k8s-addons)[text=оған арналған аддондармен]} танысасыз:
   1. Басқару мен мониторингке арналған құралдарды қосасыз.
   1. Docker-образдарды Docker тізіліміне жүктейсіз.
   1. Жүктелген образдар негізінде қарапайым қолданбаларды, VK Object Storage қоймасын пайдалану мүмкіндігімен, орналастырасыз.
   1. Ingress-контроллер арқылы орналастырылған қолданбаларға қол жеткізуді ұсынасыз.
   1. Бұл қолданбалардың шынымен жұмыс істейтініне көз жеткізесіз.

{note:warn}

Жұмыс істеп тұрған Kubernetes кластеры есептеу ресурстарын тұтынады.

Жылдам бастаудан өткеннен кейін, егер кластер енді қажет болмаса, оны {linkto(#k8s-quick-start-delete-resources)[text=тоқтатыңыз немесе жойыңыз]}.

{/note}
