# {heading(Cloud Containers)[id=k8s-k8s]}

{include(/kz/_includes/_translated_by_ai.md)}

Сервис CNCF сертификатына ие және стандартты Kubernetes API-мен үйлесімді. Автоматты масштабтаудың (autoscaling) арқасында Cloud Containers жүйесіндегі қолданбалар жүктеме шегіне жеткен сәтте бірден қосымша қуат алады және жүктеме азайған кезде тұтынуды төмендетеді. Автоматты масштабтау есептеулерге кететін шығынды 60%-ға дейін үнемдеуге мүмкіндік береді.
