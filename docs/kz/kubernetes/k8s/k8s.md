Сервис CNCF сертификатына ие және стандартты Kubernetes API-мен үйлесімді. Автоматты масштабтаудың (autoscaling) арқасында Cloud Containers жүйесіндегі қолданбалар жүктеме шарықтаған сәтте қосымша қуатты бірден алады және жүктеме азайғанда тұтынуды төмендетеді. Автомасштабтау есептеу ресурстарынан 60%-ға дейін үнемдеуге мүмкіндік береді.
