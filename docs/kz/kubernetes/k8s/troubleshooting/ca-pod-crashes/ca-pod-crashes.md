OpenStack немесе Kubernetes API жүйесіне қосылу әрекеті кезінде [автоматты масштабтау агентінің (Cluster Autoscaler)](/kz/kubernetes/k8s/concepts/cluster-autoscaler) подтар `CrashLoopBackOff` күйінде болады, соның салдарынан аутентификациядан өтпейді.

Мәселе автоматты масштабтау агенті подының манифестінде аутентификация үшін қажет параметрлер көрсетілмеген жағдайда туындайды. Манифест автоматты түрде жасалады, сондықтан ондағы параметрлердің болмауы әртүрлі себептерден болуы мүмкін. Манифесті өз бетіңізше өзгертпеңіз, пайда болатын қателер туралы егжей-тегжейлі алқпаратты беріп, мәселені техникалық қолдау арқылы түзетіңіз.

### Шешім

1. `kube-system` аттар кеңістігіндегі автоматты масштабтау агенті подының логтарын шығарыңыз:

   ```console
   kubectl logs <ИМЯ_ПОДА> -n kube-system -p
   ```
1. Логтарды ықтимал қателердің бар-жоғын тексеріңіз.

   - Егер под манифестінде `cloud-config` файлына жол көрсетілмесе немесе бұл файл жоқ болса, логтарда мынадай қателер болады:

      ```console
      You must provide a password to authenticate
      ```
   - Егер подта аутентификация үшін қате деректер көрсетілсе, логтарда мынадай қателер болады:

      ```console
      Authentication failed
      ```
   - Егер под манифестінде `Service Account: cluster-autoscaler` параметрі көрсетілмесе, логтарда мынадай қателер болады:
   
      ```console
      Failed to get nodes from apiserver: nodes is forbidden: User "system:serviceaccount:kube-system:default" cannot list resource "nodes" in API group "" at the cluster scope
      ```   
     
1. Егер логтар `Service Account: cluster-autoscaler` параметрінің жоқтығына қатысты қатені көрсетсе, қосымша диагностика жүргізіңіз:
   
   1. Автоматты масштабтау агенті подының манифестін шығару үшін пәрменді орындаңыз:

      ```console
      kubectl describe pod <ИМЯ_ПОДА> -n kube-system
      ```
   1. Манифест шығаруында келесі жолдың бар-жоғын тексеріңіз:

      ```console
      Service Account: cluster-autoscaler
      ```
      Егер бұл жол жоқ болса, оны техникалық қолдау қызметіне жолданатын өтінімде көрсетіңіз.

1. [Техникалық қолдау қызметіне](/kz/contacts) анықталған қателерді көрсете отырып хабарласыңыз.
