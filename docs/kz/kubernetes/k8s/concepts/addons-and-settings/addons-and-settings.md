# {heading(Аддондар мен баптаулар)[id=k8s-addons-and-settings]}

{include(/kz/_includes/_translated_by_ai.md)}

Cloud Containers-тің әрбір кластері сізге ыңғайлы болу үшін әлдеқашан алдын ала бапталған. Бұл бөлімде кластерде қолданылатын баптаулар, сондай-ақ кластерге орнатуға болатын аддондар сипатталған.
