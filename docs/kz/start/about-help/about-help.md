## Бетбелгілер

VK Cloud құжаттамасындағы таңдаулы мақалаларға жылдам қол жеткізу үшін арнайы **Бетбелгілер** бөлімі бар.

{note:info}Бетбелгілер әр браузер үшін бөлек сақталады.{/note}

{tabs}

{tab(Бетбелгілер бөлімін ашу)}
**Бетбелгілер** бөлімін ашу үшін беттің жоғарғы жағындағы ![Бетбелгі](../assets/bookmark-icon.svg "inline") белгішесін басыңыз.
{/tab}

{tab(Бетбелгі қосу)}
Бетті бетбелгілерге қосу үшін қажетті мақала немесе бөлім тақырыбының жанындағы ![Бетбелгі](../assets/bookmark-icon.svg "inline") белгішесін басыңыз. Бөлім беттерінде бетбелгілерге қосу белгішесі тек меңзерді мақала карточкасының үстіне апарғанда ғана көрінеді.
{/tab}

{tab(Бетбелгіні жою)}
Бетті бетбелгілерден жою үшін қажетті мақала немесе бөлім тақырыбының жанындағы ![Бетбелгі](../assets/active-bookmark-icon.svg "inline") белгішесін басыңыз.
{/tab}

{/tabs}

## Кері байланыс

### Мақаланы бағалау

Мақаланың пайдалы екенін бағалау үшін беттің төменгі жағындағы белгішелерді пайдаланыңыз:

* ![ ](../assets/like-icon.svg "inline") — егер мақала сізге пайдалы болса.
* ![ ](../assets/dislike-icon.svg "inline") — егер құжаттамадан сұрағыңызға жауап таппасаңыз. Көмек немесе кеңес алу үшін [техникалық қолдауға](/kz/contacts) хабарласыңыз.

Белгішені басқан кезде:

* мақала бойынша пікіріңіз VK компаниясына жіберіледі;
* бағалауыңызды сақтай отырып, белгішенің түсі өзгереді: ![ ](../assets/color-like-icon.svg "inline");
* тиісті мәтіні бар қалқымалы хабарлама пайда болады.

### Құжаттаманы өзгерту бойынша ұсыныстар

Егер құжаттамада қате терілген сөзді немесе қатені байқасаңыз не қандай да бір бөлімді толықтырғыңыз келсе, өзгерістерді тікелей бастапқы мәтіндерге ұсыныңыз. Ол үшін беттің төменгі жағындағы **GitHub-та өңдеу** батырмасын басыңыз.

Толығырақ — [GitHub репозиторийінде](https://github.com/vk-cs/docs-public/blob/master/README.md).

## Контентті пішімдеу

VK Cloud құжаттамасында әртүрлі ақпарат түрлерін ерекшелеудің бірнеше тәсілі қолданылады.

### Ақпараттық кірістірмелер

Ақпараттық кірістірмелердің үш түрі қолданылады:

{note:info}
Пайдалы болуы мүмкін қосымша ақпараты бар түсіндірме.
{/note}

{note:warn}
Жұмысты жалғастырмас бұрын білу маңызды ақпарат.
{/note}

{note:err}
Қауіпсіздікпен немесе деректерді жоғалту қаупімен байланысты аса маңызды ақпарат.
{/note}

### Код блоктары

Орындалатын командалар, конфигурация файлдары және бағдарламалық код синтаксисі бөлектелген арнайы блоктарда көрсетіледі.

Мұндай блоктың мазмұнын көшіру үшін оның тақырыбындағы ![Көшіру](../assets/copy-icon.svg "inline") белгішесін басыңыз.

```javascript
// пример кода на JavaScript
console.log('Hello World');
```

### Жасырын мәтін

Үлкен код блоктары мен ұзын мысалдар үшін жасырын мәтін блоктары қолданылады:

{cut(Жабық блок осылай көрінеді)}
Жасырын мәтіннің мысалы.
{/cut}

Оның мазмұнын көру үшін блокты басыңыз. Мәтіндік блокты жасырғыңыз келсе, қайта басыңыз.

### Қойындылар

Балама нұсқаулықтар үшін қойындылар қолданылады. Мысалы, әртүрлі операциялық жүйелердің терминалдарына арналған командалар.

{tabs}

{tab(Бірінші қойынды)}
Бірінші қойындыдағы мәтін осылай көрінеді.

Оған ауысу үшін екінші қойындыны басыңыз.
{/tab}

{tab(Екінші қойынды)}
Екінші қойындыдағы мәтін осылай көрінеді.

Оған ауысу үшін бірінші қойындыны басыңыз.
{/tab}

{/tabs}

Егер қойындыларда ретімен орындалатын нұсқаулық қадамдары орналастырылса, қойындылар нөмірленеді.
