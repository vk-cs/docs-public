VK Cloud серіктестік бағдарламасы ынтымақтастықтың екі моделін көздейді:

- Серіктес,
- Агент.

## Серіктес

Клиенттерге VK Cloud бұлттық сервистерін өзіңіз сатыңыз және серіктестік жеңілдік есебінен табыс табыңыз.

VK Cloud шешімдерін ілгерілетуге және қайта сатуға дайын компания (заңды тұлға немесе ЖК) серіктес бола алады.

### Модель қалай жұмыс істейді

1. Сіз жаңа клиентті тартасыз.
1. Сіз VK Cloud шешімдеріне [серіктестік жеңілдік](#discount) аласыз.
1. Сіз шешімді сатасыз және баға айырмасы есебінен табыс табасыз.

Сіз клиентпен қарым-қатынастың заңдық және қаржылық бөлігін өзіңіз жүргізесіз.

Осы ынтымақтастық моделін таңдағанда, өз брендіңізбен White Label тұжырымдамасы бойынша VK Cloud ресурстарын қайта сату мүмкіндігі бар. White Label серіктестеріне бірегей функционал қолжетімді:

- Клиент интерфейсін және құжаттаманы өз брендіңізге бейімдеу.
- Клиенттің тарифтерін және оның кабинетінде көретін бағаларын басқару.
- Клиенттің төлем моделін өзгерту, клиенттің VK Cloud сервистерін пайдалануды бастауға дейінгі жолын қысқарту.

White Label серіктесі болу үшін серіктестікке арналған өтініміңізбен жұмыс істейтін менеджерге хабарласыңыз немесе [partner@mcs.mail.ru](mailto:partner@mcs.mail.ru) поштасына жазыңыз.

### {heading(Серіктестік жеңілдік неге байланысты)[id=discount]}

Серіктестік жеңілдік екі бөліктен тұрады:

- Жеңілдіктің базалық бөлігі — 10%.
- Мәмілені тіркеуге арналған жеңілдік — 20%-ға дейін.

Жалпы жеңілдік VK Cloud [прайс-парағында](https://cloud.vk.com/pricelist) көрсетілген бағадан 30%-ға дейін жетуі мүмкін.

## Агент

Жаңа клиенттерді VK Cloud бұлттық сервистерімен жұмыс істеуге тартыңыз және клиенттердің тұтынуы үшін [сыйақы](#reward) алыңыз.

Агент бола алады:

- Заңды тұлға.
- Тіркелген мерзімі 1 жылдан асқан жеке кәсіпкер.

### Модель қалай жұмыс істейді

1. Сіз клиенттерге VK Cloud шешімдерін ұсынасыз.
1. VK Cloud шешімді клиентке сатады.
1. Сіз клиенттің платформадағы ресурстарды тұтынғаны үшін сыйақы аласыз.

Клиентпен қарым-қатынастың бүкіл заңдық және қаржылық бөлігі VK Cloud жағында жүргізіледі.

### {heading(Сыйақы неге байланысты)[id=reward]}

Агенттің сыйақысы екі бөліктен тұрады:

- Сыйақының базалық бөлігі — тартылған клиенттер төлейтін VK Cloud қызметтерінің төлемінен алынатын пайыз.
- Техникалық пресейл жүргізгені үшін сыйақы.

Сіздің сыйақыңыз тіркелген.

## Сыйақыны не үшін алу мүмкін болмайды

Таңдалған ынтымақтастық моделіне қарамастан, клиенттердің келесі сатып алулары үшін сыйақы берілмейді:

-	Балансындағы бонустық рубльдерге жасалған сатып алулар.
-	Қосымшалар дүкеніндегі өнімдерді сатып алу.
-	Үшінші тарап вендорларының лицензияларын сатып алу.
-	Инфрақұрылымды әкімшілендірудің ақылы қызметтері.
