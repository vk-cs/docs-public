VK Cloud платформасы бұлттық қосымшаларды әзірлеу және пайдалану үшін инфрақұрылымдық және платформалық сервистерді, сараптамалық қолдауды, бапталатын инсталляцияларды (соның ішінде On-premise), консалтингтік қызметтерді ұсынады. Платформада виртуализация, мониторинг, контейнерлер, деректер қоймалары сервистері орналастырылуы мүмкін. [Платформа архитектурасы](/kz/start/concepts/architecture) туралы толығырақ.<br>Сервистерді енгізу мен пайдаланудың барлық кезеңдерінде [техникалық қолдау](/kz/start/support) қарастырылған. Сервистердің төлемі туралы [Биллинг](/kz/intro/billing) бөлімінен оқыңыз.
