## Қолдау құзыреттері

Техникалық қолдау:

- VK Cloud сервистерін қолдануға қатысты архитектура бойынша консультациялар береді.
- Жобадағы ресурстар жұмысындағы ақауларды диагностикалайды.
- [Есептік жазбаға](/kz/tools-for-using-services/account/faq) қолжетімділікті қалпына келтіру мәселелерімен айналысады.
- Биллингтің жұмысы туралы хабарлайды.
- Платформадағы проблемаларды жою бойынша кеңес береді.
- VK Cloud сервистерін баптауға көмектеседі.

<warn>

VK Cloud техникалық қолдау командасы операциялық жүйелерді және Платформаны пайдалана отырып жасалған өзге сервистерді әкімшілендірумен және қолдаумен айналыспайды, егер бұл жеке келісімде келісілмесе.

</warn>

## Техникалық қолдаудан хабарламалар

Қолдау командасы клиенттерді клиенттің есептік жазбасындағы сервиспен немесе ресурспен болған іркілістер туралы хабардар етеді, сондай-ақ клиент жобасындағы ықтимал мәселелерді жою бойынша ұсынымдар береді.

Хабарламалар қатарына жоба иелерін қауіпсіздік қатерлері немесе деректердің жоғалуына не VK Cloud сервистеріне қолжетімділіктің үзілуіне әкелуі мүмкін басқа жағдайлар туралы хабардар ету де кіреді.

## Жеке қолдау

VK Cloud платформасы сервистерінің тізбесіне басымдық берілген қолдауды (Professional Services) ұсыну мүмкіндігі кіреді.

Бұл қызметтің құрамына клиент жобаларында енгізу, сүйемелдеу және іркілістерді жоюдың барлық мәселелері бойынша кеңейтілген қолдау кіреді:

- 24/7 жүйелік әкімшілендіру:

  - IT-инфрақұрылым мен бұлттық сервистер аутсорсингі;
  - техникалық қолдау;
  - мониторинг және қолжетімділікті қолдау;
  - инциденттерге әрекет ету;
  - іркілістерден кейін қалпына келтіру.

- Аудит және консалтинг:

  - архитектура бойынша консалтинг;
  - инфрақұрылым аудиті;
  - өнімділік аудиті.

- Конфигурациялау және енгізу:

  - мониторингті баптау;
  - резервтік көшіруді, жоғары қолжетімділікті (high availability, HA), іркілістерден кейін қалпына келтіру жүйелерін баптау;
  - PaaS-сервистердің БҚ-сын жаңарту;
  - деректер мен ақпараттық жүйелерді көшіру.

[Толығырақ білу](https://cloud.vk.com/professional-services/) және құнын есептеуге өтінім қалдыру.
