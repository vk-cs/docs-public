VK Object Storage сервисі HTML, CSS және JavaScript технологияларында құрылған статикалық сайттарды хостингтеуге арналған ресурстарды пайдалануға мүмкіндік береді. Сондай-ақ сақтау қоймасын сұрауларды қайта бағыттау үшін пайдалануға болады.

{note:warn}

Хостинг үшін пайдаланылатын бакетті веб-сервер жағында іске қосуды талап ететін скрипттерді орналастыру үшін пайдалану мүмкін емес.

{/note}

## Доменді байланыстыру

Бакет объектілеріне бірегей сайтқа апаратын сілтемелер арқылы қолжеткізуді ұсыну үшін **Доменді байланыстыру** қойындысында өз доменіңізді байланыстырыңыз.

Алдын ала мыналар қажет болады:

1.  Доменді кез келген хостинг-провайдерде тіркеу.
1. Жасалған бакет үшін пайдаланылатын ішкі доменді жасау.
1. VK Cloud панелінде ішкі доменді байланыстыруды орындау.

Байланыстырғаннан кейін бакет мына түрдегі сілтеме бойынша қолжетімді болады:

```console
http://<ИМЯ_БАКЕТА>.<АДРЕС_САЙТА>.<ДОМЕННАЯ_ЗОНА>
```

{note:info}

Атауында `.` символы пайдаланылатын бакет тек HTTP протоколы арқылы қолжетімді болады. HTTPS протоколын пайдалану үшін SSL сертификатын орнату қажет.

{/note}

## SSL сертификатын орнату

Жеке SSL сертификатын орнату үшін жоба мен бакет деректерін көрсетіп, орнату үшін SSL сертификатын ұсына отырып, [техникалық қолдауға хабарласыңыз](/kz/contacts).

Сертификат орнатылғаннан кейін бакет мына түрдегі сілтеме бойынша қолжетімді болады:

```console
https://<ИМЯ_БАКЕТА>.<АДРЕС_САЙТА>.<ДОМЕННАЯ_ЗОНА>
```
