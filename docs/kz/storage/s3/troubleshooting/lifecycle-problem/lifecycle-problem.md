# {heading(Өмірлік цикл ережелері іске қосылмайды)[id=s3-lifecycle-problem]}

{include(/kz/_includes/_translated_by_ai.md)}

Объектілерді автоматты түрде жою үшін {linkto(../../concepts/lifecycle#s3-concepts-lifecycle)[text=өмірлік цикл]} ережелері бапталған, бірақ берілген мерзім өткеннен кейін объектілер жойылмайды.

Мәселе ережелерде көрсетілген объект кілтінде қате жазылу болған жағдайда туындауы мүмкін.

### {heading(Шешімі)[id=s3-lifecycle-problem-resolve]}

1. Ережеде көрсетілген {linkto(../../concepts/about#s3-concepts-about-object-key)[text=объект кілті]} бакеттегі объект кілтімен дәл (әріп регистрін ескере отырып) сәйкес келетініне көз жеткізіңіз.
1. Егер объект кілті дұрыс көрсетілсе, ереже қолданылғанша 24–48 сағат күтіңіз. Мұндай кідіріс — қате емес, сервистің қалыпты мінез-құлқы.
1. Егер мәселе сақталса, [техникалық қолдауға](/kz/contacts) жүгініңіз.
