Объектілік хранилищемен жұмыс істеу кезінде `x-amz-content-sha256 must be UNSIGNED-PAYLOAD` хабарламасы пайда болады. 

Бұл AWS CLI 2-нұсқасы қолданылып жатқандықтан туындауы мүмкін.

### Шешім

Командаңызға `--no-sign-request` жалаушасын қосыңыз немесе AWS CLI профилін 2-нұсқадағы қолтаңбаны (SigV2) пайдалану үшін баптаңыз.
