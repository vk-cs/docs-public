## Тарифтеу туралы

Сервис жекелеген компоненттерінің құны [прайс-парақта](https://cloud.vk.com/pricelist) келтірілген. Сервистің жалпы құнын есептеу үшін [калькуляторды](https://cloud.vk.com/pricing) пайдаланыңыз.

Тарифтеу «pay as you go» қағидаты бойынша жүргізіледі: төлем тек тұтынылған ресурстар үшін секундқа дейінгі дәлдікпен алынады.

Сервисті қалай төлеу керектігі туралы [Биллинг](/kz/intro/billing) бөлімінен оқыңыз.

## Тарифтелмейді

VK Object Storage жүйесіндегі қызметтер тарификациясы деректердің [сақтау класына](../concepts/about#storage_class) байланысты.

Hotbox және Icebox жүйелерінде мыналар тарифтелмейді:

- Сақталатын деректер көлемі (ГБ).
- Шығыс трафик (ГБ).

    Шығыс трафик деп бакеттен ақпаратты кез келген жүктеп алу есептеледі: VK Cloud ішінен, мысалы, ВМ тарапынан да, VK Cloud сыртынан, мысалы, пайдаланушының ДК-сынан да. Екі жағдайда да бакетке жүгіну сыртқы IP мекенжай арқылы жүзеге асырылады.

Backup жүйесінде тек сақталатын деректер көлемі (ГБ) тарифтелмейді.
