|===
|Қысқарту
|Ашып жазылуы

|ACL
|Access Control List, объектіге кімнің немесе ненің қол жеткізе алатынын және субъектінің қандай операцияларды орындауына рұқсат етілетінін немесе тыйым салынатынын анықтайтын қолжетімділікті басқару тізімі

|API
|Application Programming Interface — аппараттық-бағдарламалық интерфейс. Бір компьютерлік бағдарламаның басқа бағдарламамен өзара әрекеттесу тәсілдерінің (кластар, процедуралар, функциялар, құрылымдар немесе константалар жиынтығы) сипаттамасы

|AWS
|Amazon Web Services — Amazon Simple Storage Service (Amazon S3) объектілерді сақтау сервисін ұсынатын бұлттық объектілік қойма

|AWS CLI
|AWS пәрмен жолы интерфейсі. {var(s3)} сервистерін басқару үшін қолданылады

|JavaScript
|Көппарадигмалы бағдарламалау тілі. Объектіге бағытталған, императивті және функционалдық стильдерді қолдайды

|JSON
|JavaScript Object Notation — JavaScript негізіндегі деректер алмасудың мәтіндік форматы

|HashiCorp Vault
|Мысалы, құпиясөздер сияқты әртүрлі құпия деректерді қауіпсіз сақтауды және оларға қолжетімділікті қамтамасыз ететін құрал

|IAM
|Веб-интерфейс (Web UI) арқылы әкімшілендіру құралы.

|KMS
|Key Management Service — шифрлау кілттерін орталықтандырылған басқару қызметі. {var(s3)} үшін KMS ретінде Hashicorp Vault қолданылады

|S3
|Simple Storage Service. Объектілерді сақтау сервисі

|SSE
|Server-side encryption, сервер жағындағы шифрлау. {var(s3)} ішіндегі әр объект KMS көмегімен бірегей кілтпен (SSE-E) немесе пайдаланушы ұсынған кілтпен (SSE-C) шифрланады

|URL
|Uniform Resource Locator, ресурстың унификацияланған көрсеткіші — электрондық ресурстардың унификацияланған адрестер жүйесі немесе ресурстың орналасқан жерінің біріздендірілген анықтағышы. Интернеттегі объектілерге сілтемелерді жазудың стандартты түрі ретінде қолданылады

|ОС
|Операциялық жүйе

|pub/sub
|Хабарландырулардың баспагер-жазылушы (publisher-subscriber) жұмыс моделі

|push-модель
|Конфигурацияларда берілген шарттар орындалған кезде хабарландыруларды лезде жіберу моделі

|SNS
|Simple Notification Service, көрсетілген URL-адрестерге (HTTP-сұраулар арқылы) хабарландыруларды автоматты түрде жіберу механизмі

|EventBridge
|Хабарландыруларды жіберу сервисі
