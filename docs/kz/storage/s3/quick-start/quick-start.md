# {heading(Жылдам бастау)[id=s3-quick-start]}

{include(/kz/_includes/_translated_by_ai.md)}

Жылдам бастау сервиспен жұмысты бастауға және оның мүмкіндіктерімен танысуға көмектеседі.

Жылдам бастауның барлық қадамдарынан өткеннен кейін, сіз бакетке объектілерді жүктеуді және жүктелген объектілерге сілтеме арқылы қол жеткізуді ұсынуды үйренесіз.

{note:warn}
Бакетке жүктелген объектілер [тарифтеледі](../tariffication). Жылдам бастаудан өткеннен кейін, егер олар сізге енді қажет болмаса, [осы объектілер мен бакетті жойыңыз](#delete_unused).
{/note}
