## {heading(Тарифтеу туралы)[id=backup-tariffication]}

Cloud Backup сервисіндегі [резервтік көшіру](/kz/storage/backups/concepts/about) құны дискінің уақытша снимогын сақтау және резервтік көшірмені кейіннен сақтау құнынан құралады:

1. Резервтік көшірмені жасау үшін жүйе [дискілік қоймада](/kz/computing/iaas/concepts/data-storage/disk-types) сақталатын және минут сайын тарификацияланатын (орта есеппен бір сағат) дискінің уақытша снимогын (snapshot) қалыптастырады.

2. Диск снимогының негізінде [VK Cloud Storage](/kz/storage/s3/concepts/about) объектілік қоймасында `BackupBucket` [сақтау класы](/kz/storage/s3/concepts/about#storage_class) бар бакеттерде сақталатын сығылған резервтік көшірме жасалады.

   {cut(Резервтік көшірмелердегі деректер қалай сығылады)}

   Дискідегі файл түріне байланысты оларға әртүрлі сығу дәрежесі қолданылады:

    - жоғары: мәтіндік құжаттар, дерекқорлар;
    - төмен: бұрыннан сығылған форматтар (кескіндер, бейнежазбалар, архивтер).

   Деректерді сығудың нақты коэффициенті алдын ала белгісіз, сондықтан олардың қорытынды көлемін резервтік көшірмені жасау аяқталғаннан кейін ғана анықтауға болады.
   {/cut}

Резервтік көшірмені сақтаудың қорытынды құны деректер көлеміне, жиілігіне және таңдалған резервтік көшіру стратегиясына байланысты. Сервисті пайдалану шығындарын азайту үшін [резервтік көшіруді баптау бойынша үздік тәжірибелерді](/kz/storage/backups/concepts/best-practices#cost-manage) пайдаланыңыз.

Бағалар [прайс-парақта](https://cloud.vk.com/pricelist) келтірілген. VK Cloud сервистерінің ақысын төлеуге байланысты құралдардың қалай жұмыс істейтіні туралы [Биллинг](/kz/intro/billing) бөлімінен оқыңыз.

Сервистердің таңдалған конфигурациясының құнын есептеу үшін [калькуляторды](https://cloud.vk.com/pricing/) пайдаланыңыз.

## {heading(Тарифтелмейді)[id=backup-tariffication-yes]}

- Әрбір 1 ГБ дискілік кеңістік үшін диск снимоктары (snapshot). Бағасы барлық [диск түрлері](/kz/computing/iaas/concepts/data-storage/disk-types) үшін бірдей және [прайс-парақтың](https://cloud.vk.com/pricelist) **Виртуалды серверлер** бөлімінде келтірілген.
- `BackupBucket` сақтау класы үшін VK Cloud Storage [тарифтері](/kz/storage/s3/tariffication) бойынша әрбір 1 ГБ сақталатын деректер көлемі.

## {heading(Тарификацияланбайды)[id=backup-tariffication-no]}

- VK Cloud Storage ішіндегі сұраулар.
- Резервтік көшірмелерді жасау операциялары.
- Кіріс және шығыс трафик.
- Мониторинг.
